一、函数速查表

函数数据源鉴权用途首次引入
eastmoney_reports(code, max_pages=5) 东财 reportapi Referer 头 个股研报列表（评级 + 三年EPS预测） V1.0
download_pdf(record, target_dir) 东财 PDF (pdf.dfcfw.com) Referer 头 单份研报 PDF 下载（文件名清洗 + 去重 + 头验证） V1.0
eastmoney_industry_reports(industry_code, max_pages) 东财 reportapi Referer 头 行业研报列表（qType=1，通过全行业反查行业码） V3.2.3
ths_eps_forecast(code) 同花顺 basic.10jqka.com.cn UA 头 机构一致预期 EPS DataFrame（均值/最小/最大/机构数） V1.0
iwencai_search(query, channel, size) iwencai OpenAPI API Key + X-Claw NL 语义搜索研报/公告/新闻 V2.0
iwencai_query(query, page, limit) iwencai OpenAPI API Key + X-Claw NL 结构化数据查询（如 "贵州茅台 ROE"） V2.0
dedup_articles(articles) —（工具函数） — iwencai 段落级结果去重（同 uid 保留最高 score） V2.0
_claw_headers(call_type) —（工具函数） — 生成 iwencai X-Claw 鉴权头（5 个自定义头 + 随机 trace_id） V2.0

二、端点速查

端点协议封 IP 风险关键参数
reportapi.eastmoney.com/report/list HTTP GET 有风控 code (个股/行业码)、qType (0=个股 1=行业)、beginTime/endTime、pageNo/pageSize
pdf.dfcfw.com/pdf/H3_{info_code}_1.pdf HTTP GET 同东财 info_code 来自 reportapi 返回的 infoCode
basic.10jqka.com.cn/new/{code}/worth.html HTTP GET 低 6 位股票代码、编码 GBK
openapi.iwencai.com/v1/comprehensive/search HTTP POST 低 query、channels、size
openapi.iwencai.com/v1/query2data HTTP POST 低 query、page、limit

五、鉴权机制对比

数据源鉴权方式获取难度示例头

六、典型调研流程（C 路线——主题研报）

C 路线：主题研报检索（~2 分钟） 1. iwencai_search("人形机器人 行星滚柱丝杠 2026", channel="report") → dedup_articles() 去重 → 获得相关研报列表（段落级 → uid 去重后按日期倒序） 2. 交叉补充：东财 reportapi → eastmoney_reports("标的代码") → 发现 iwencai 未覆盖的研报 3. PDF 下载 → download_pdf(record) 逐一下载 → Referer 鉴权 + 文件名清洗 + PDF 头验证
