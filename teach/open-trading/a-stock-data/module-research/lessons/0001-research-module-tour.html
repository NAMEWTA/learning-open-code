东财研报 — 研报列表与 PDF 下载

L1 研报层 第 1 节 / 共 2 节 估计时长：12 分钟

1. 研报层的定位

研报层是 10 层架构中的 第 2 层（L2），回答一个问题："机构怎么看待这个标的？"它提供了机构视角的三类信息——评级与 EPS 预测、PDF 全文、以及自然语言主题搜索。东财 reportapi 是主力（免费、无 Key、覆盖全），同花顺提供一致预期 EPS 交叉验证，iwencai 的 NL 搜索是唯一能做跨主题概念检索的数据源。

研报层数据流 ┌─────────────┐ ┌─────────────────┐ ┌──────────────┐ │ eastmoney │ │ eastmoney_ │ │ download_pdf │ │ _reports() │───▶│ industry_ │───▶│ () │ │ 个股研报列表 │ │ reports() │ │ Referer鉴权 │ │ qType=0 │ │ 行业研报 qType=1│ │ PDF头验证 │ └─────────────┘ └─────────────────┘ └──────────────┘ │ └── em_get() 统一限流 ── 串行 1s+ 间隔 + 会话复用

核心原则：本层所有东财端点（reportapi + PDF）均通过 em_get() 发出请求，内置串行限流（最小 1s 间隔 + 随机抖动）、Keep-Alive 会话复用和连接级自动重试。忽略此点会触发 IP 封禁。

2. eastmoney_reports() — 个股研报列表

端点 https://reportapi.eastmoney.com/report/list 提供指定股票的所有历史研报 JSON 数据。核心参数：code 为 6 位代码，qType=0 锁定个股研报，max_pages 控制翻页深度。

翻页逻辑：逐页请求 → 检查返回行数是否为空 → 检查是否到达 TotalPage → 任一条件满足即停止。每页最多返回 100 条，以 beginTime/endTime 控制时间范围。

字段含义用于
title研报标题列表展示
orgSName机构简称文件名、筛选
infoCodePDF 标识码拼 PDF URL
predictThisYearEps今年 EPS 预测估值计算
predictNextYearEps明年 EPS 预测前向 PE
predictNextTwoYearEps后年 EPS 预测远期估值
emRatingName评级（买入/增持/…）机构态度
注意事项：三年 EPS 预测来自研报本身，不同机构的预测模型差异大，不建议直接跨机构比较。同一只股票的预测取中位数或加权平均才有意义。

3. download_pdf() — PDF 下载的鉴权链

东财 PDF 的 URL 模板为 https://pdf.dfcfw.com/pdf/H3_{info_code}_1.pdf，直接从 eastmoney_reports 返回的 infoCode 拼接即可。但直接 wget 会得到空响应——必须带 Referer 头。

下载函数做了三件事：

  • 文件名清洗：机构名可能含 /（如"中信建投/研究所"），用正则 re.sub(r'[\\/:*?"<>|]', '_', ...) 替换为下划线，防止路径拼坏。同时截断机构名 ≤40 字符、标题 ≤80 字符。
  • 去重：若 target.exists() 已存在则跳过下载，直接返回路径。
  • PDF 头验证：检查 r.status_code == 200 且 len(r.content) >= 1024，排除空响应和错误页面。

下载同样走 em_get()，timeout 设为 60s（PDF 可能较大）。

4. eastmoney_industry_reports() — 行业研报（V3.2.3）

与个股研报同一端点，唯一区别是将 qType 从 0 改为 1。行业研报按东财行业代码过滤，industry_code="*" 表示全行业。返回的 record 可直接喂给 download_pdf()。

行业码的发现方法：东财行业码无公开码表（旧端点 bxpa 已 404）。实践中先用 industry_code="*" 拉一批，从返回的 industryName/industryCode 字段提取目标行业码，再用该码精确过滤。全行业约 47928 条记录。

字段含义
industryName行业名称（如 IT服务Ⅱ、风电设备）
industryCode东财行业代码（用于精确过滤）
emRatingName行业评级（买入/增持/中性/…）
attachPagesPDF 页数

检索练习

不看屏幕，尝试回答（每题 30 秒）：

  1. eastmoney_reports("688017") 和 eastmoney_industry_reports("*") 的端点 URL 是否相同？哪个参数区分它们？
  2. download_pdf() 中文件名清洗正则 [\\/:*?"<>|] 覆盖了哪些情况？为什么机构名需要清洗？
  3. 东财行业码怎么获取？为什么不能用 bxpa 端点？
参考答案：
1. 相同端点 reportapi.eastmoney.com/report/list，通过 qType 区分：0 = 个股研报，1 = 行业研报。
2. 清洗 Windows/Unix 非法文件名字符：反斜杠、正斜杠、冒号、星号、问号、双引号、尖括号、竖线。机构名可能含 /（如"中信建投/研究所"），不替换会被操作系统理解为子目录，导致写文件失败。
3. 先用 industry_code="*" 拉全行业列表，从结果中提取 industryCode 字段。bxpa 等旧端点已 404，无可用公开码表。
