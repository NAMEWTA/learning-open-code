# 使命：行情层 — mootdx K线+五档盘口、腾讯PE/PB/市值、百度K线均线

## 为什么
学习者需要掌握 A 股行情数据获取的第一入口——不封IP的数据源（mootdx TCP / 腾讯 HTTP / 百度 HTTP），能在 Python 中快速拉取实时价格、K线、PE/PB估值、均线指标，为后续的量化分析、估值判断、策略回测提供可靠的行情数据基础。

## 成功的样子
- 能用 mootdx 拉取任意 A 股的日线/分钟K线，理解 frequency 参数的正确用法
- 能区分 mootdx（K线+盘口）、腾讯（PE/PB/市值）、百度（K线+MA均线）三源的能力边界，选对数据源
- 能用 tencent_quote() 批量获取多只股票的实时估值数据（PE/PB/市值/换手率）
- 能将三源数据拼装成一支股票的完整行情快照（实时价+K线+均线+估值）

## 约束条件
- 学习时间：15 分钟内完成本课
- 基础要求：理解 L0 项目总览中的 10 层架构，知道 tdx_client() 和 get_prefix() 的作用
- 环境：已安装 mootdx、requests（或待学完基础设施模块后实操）

## 不在范围内
- 不涉及东财数据源（研报、龙虎榜、资金流等归后续模块）
- 不涉及技术指标计算（MACD/RSI/BOLL 等归后续模块）
- 不涉及复权算法实现细节
