参数名警告：参数名是 frequency，不是 category。传错参数名会被 **kwargs 静默吞掉，退化为默认日线（frequency=9）。V3.2.5 前文档曾误写为此名称，现在已修正。

五、百度 baidu_kline_with_ma() 接口速查

项目说明
端点https://finance.pae.baidu.com/selfselect/getstockquotation
核心参数group=quotation_kline_ab, ktype=1（日线）, all=1
返回结构Result.newMarketData.keys (字段名列表) + Result.newMarketData.marketData (; 分隔的多行数据)
独有字段ma5avgprice, ma10avgprice, ma20avgprice
踩坑点ResultCode 在不同场景下可能是 int 或 str 类型，做判断建议 str(d.get("ResultCode")) != "0"
更多参数ktype 可调整周期（1=日线），start_time 可指定起始时间

七、典型场景组合示例

场景 A：快速估值看一眼

q = tencent_quote(["600519"])["600519"]
print(f"{q['name']}: {q['price']}元 PE={q['pe_ttm']} PB={q['pb']} 市值={q['mcap_yi']}亿 换手={q['turnover_pct']}%")
# → 贵州茅台: 1680.50元 PE=30.45 PB=9.81 市值=21100.32亿 换手=0.32%

场景 B：盘中盯盘（实时盘口 + 价格）

# 盘口深度
dq = tdx_client().quotes(symbol=["600519"])
# 买一价 = dq[0]['bid1'], 买一量 = dq[0]['bid_vol1']
# 卖一价 = dq[0]['ask1'], 卖一量 = dq[0]['ask_vol1']

# 实时价比对
tq = tencent_quote(["600519"])["600519"]
if abs(dq[0]['price'] - tq['price']) > 0.05:
    print("mootdx与腾讯价差>0.05，注意延迟")
