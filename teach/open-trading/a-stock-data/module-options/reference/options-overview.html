关键踩坑：raw 返回列表中，raw[1:4] 是 3 个空字符串。代码必须执行 v = [raw[0]] + raw[4:] 跳过它们，否则字段索引全部错位——Delta 读到空串、IV 读到 Delta 位置，数据完全不可用。这是本模块最容易出 Bug 的地方。

五、希腊字母快速含义卡

指标一句话范围买方视角卖方视角
Delta 标的涨 1 元，期权涨多少 认购 0~1，认沽 -1~0 Delta=0.6 → 标的涨 1 元赚 0.6 Delta=-0.6 → 标的涨 1 元亏 0.6
Gamma Delta 变化的速度 平值最大，实值/虚值递减 Gamma 大 = 需要频繁调仓 Gamma 大 = 风险暴露变化快
Theta 每天贬值多少 通常为负（买方） 每天在"烧钱"，到期前加速 每天在"收租"，时间的朋友
Vega 波动率涨 1%，期权涨多少 通常为正 恐慌时 Vega 带来额外收益 波动率飙升时承担风险
IV 市场预期的未来波动 无固定范围 高 IV = 期权贵，低 IV = 便宜 高 IV = 权利金丰厚但风险大

六、sina_option_codes() 合约获取流程

1. 映射 underlying → 新浪分类名
   "510050" → "50ETF"
   "510300" → "300ETF"
   "588000" → "科创50ETF"
   "510500" → "500ETF"

2. GET https://stock.finance.sina.com.cn/futures/api/openapi.php/
        StockOptionService.getStockName?exchange=null&cate={分类名}

3. 从返回 JSON 中提取 result.data.contractMonth
   丢弃第一个月份，剩余月份 YYYY-MM → YYMM 转换

4. 逐月调用 _sina_opt_list(f"OP_UP_{underlying}{YYMM}")
   或 _sina_opt_list(f"OP_DOWN_{underlying}{YYMM}")
   过滤出 CON_OP_ 开头的代码，去掉前缀

5. 返回 {YYMM: [code1, code2, ...], ...}
   第一个 key = 近月 → 交易最活跃

七、典型场景代码示例

场景 A：获取 50ETF 所有月份的所有认购合约

codes = sina_option_codes("510050", call=True)
for month, contract_list in codes.items():
    print(f"{month}月: {len(contract_list)} 个合约")
# → 06月: 24 个合约
# → 07月: 18 个合约
# → 09月: 15 个合约
# → 12月: 12 个合约

场景 B：构建近月期权链（T型报价 + 希腊字母）

codes = sina_option_codes("510050", call=True)
near = list(codes)[0]  # 近月
chain = []
for code in codes[near]:
    tq = sina_option_tquote(code)
    gk = sina_option_greeks(code)
    if tq and gk:
        chain.append({**tq, **gk})
# 按行权价排序
chain.sort(key=lambda x: x.get('strike', 0))
# chain 现在 = 近月完整期权链，每行含 tquote + greeks 全部字段

场景 C：IV 偏度快速判断

# 拉取近月所有合约的 IV，按行权价排列看偏斜
for row in chain:
    if 'strike' in row and 'iv' in row:
        print(f"行权价{row['strike']:.3f} IV={row['iv']:.2%}")
# 如果低行权价 IV 显著高于高行权价 → IV 左偏（市场担忧下跌）
# 如果高行权价 IV 显著高于低行权价 → IV 右偏（市场预期上涨）

场景 D：300ETF 科创50ETF 同样用法

# 换 underlying 参数即可，其余逻辑完全一致
codes_300 = sina_option_codes("510300", call=False)  # 300ETF 认沽
codes_kcb = sina_option_codes("588000", call=True)   # 科创50ETF 认购
codes_500 = sina_option_codes("510500", call=True)   # 500ETF 认购

八、已知踩坑点汇总

#坑点严重程度现象修复方式
1 GBK 编码未设置 高 中文乱码，数据不可读 r.encoding = "gbk"
2 缺少 Referer 头 高 403 Forbidden，零数据 加 "Referer": "https://stock.finance.sina.com.cn/"
3 greeks raw[1:4] 空串未跳过 高 Delta/IV 全部错位 v = [raw[0]] + raw[4:]
4 iv 当成百分比而非小数 中 IV 显示 1735% 而非 17.35% f"{iv:.2%}" 格式化
5 tquote 返回 <43 字段未检查 中 取到空字典后的 KeyError 检查 len(v) >= 43
6 非交易时间请求 低 部分字段为空或超时 加 timeout=10，判空处理
