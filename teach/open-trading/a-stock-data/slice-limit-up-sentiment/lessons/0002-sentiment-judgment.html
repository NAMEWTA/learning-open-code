情绪数据解读 — 从数字到判断

L2 垂直切片 打板情绪 第 2/2 节 估计时长：12 分钟

1. 四个核心指标的含义

limit_up_sentiment() 返回的 dict 中有 5 个数字字段。每个都不是孤立的——它们构成一个相互验证的信号体系：

指标计算方式信号含义
涨停数量len(zt_pool)市场参与热度。>80 = 过热，<30 = 冰点
炸板率zb/(zt+zb)*100%涨停质量。>40% = 封板不稳，情绪脆弱
最高连板max(limit_days)空间板高度。≥5板 = 赚钱效应扩散，≤2板 = 行情压缩
连板梯队{板数: 家数}结构健康度。梯队断档 = 行情不可持续
跌停数量len(dt_pool)恐慌程度。>20 = 恐慌蔓延，做多需谨慎
关键原则：单指标会骗人。炸板率低 + 涨停数少 = 无人参与（非健康）。涨停数多 + 炸板率高 = 游资互割（风险高）。必须三个以上指标共振才有判断价值。

2. 连板梯队 — 市场结构的骨架

连板梯队 ladder 是最被忽视却最重要的指标。它反映的不是"有多少涨停"，而是"涨停股的层次是否健康"。 

# 健康梯队示例
{1: 28, 2: 15, 3: 8, 4: 3, 5: 2, 7: 1}
# 解读：逐板递减，近似金字塔结构 → 行情可持续

# 不健康梯队示例
{1: 35, 2: 2, 3: 1, 7: 1}
# 解读：2板/3板断层严重 → 首板次日大面积被淘汰，行情一日游

层次感原则：每一个连板数上的家数应大致为上一级的 40%-60%。如果从 1 板到 2 板的存活率低于 20%，说明市场接力意愿极弱，首板多为套利性质。

健康信号：梯队完整、逐板递减、最高板 ≥4 板
警惕信号：梯队某档断档（如 3 板 0 只）、炸板率 30%-40%
危险信号：多档断档、最高板 ≤2 板、炸板率 >50%、跌停数 >涨停数

3. 晋级率 — 赚钱效应的量化

用 em_yzt_pool()（昨日涨停池）可以计算晋级率：昨日涨停的股票中，今天继续涨停的比例。这是比涨停数更真实的赚钱效应指标。 

yzt = em_yzt_pool(date)  # 昨涨停今表现
advance = sum(1 for s in yzt if s["pct"] >= 9.8)
advance_rate = advance / len(yzt) * 100  # 晋级率%

# 经验阈值
# >30%：强赚钱效应，可积极做多
# 15-30%：正常轮动
# <15%：弱市，追高风险大
注意：晋级率的精度受涨跌幅限制影响（北交所 30%、科创板 20%、主板 10%），统一用 9.8% 作为"涨停"阈值可覆盖主板，但科创/创业板需调整为 19.8%。

4. 综合判断框架 — 进攻日 vs 防守日

将所有指标组合成一个三层判断框架：

层级检查项达标条件
1. 温度涨停数 + 跌停数涨停 >40 且 跌停 <10 → 参与意愿正常
2. 质量炸板率 + 梯队完整度炸板率 <30% 且 2板存活率 >30% → 封板有效
3. 持续性晋级率 + 题材集中度晋级率 >20% 且 涨停揭秘前3题材占比 >40% → 有主线
使用方式：三个条件全部满足 → 进攻日（可以出手）。满足 1-2 个 → 观察日（只做研究不做交易）。全部不满足 → 防守日（只看不动）。这套规则不是预测涨跌，而是控制自己的参与节奏。

练习：情绪场景判断

某日数据：涨停 95 只、炸板 55 只、跌停 3 只、最高 6 连板、梯队 {1:60,2:18,3:7,4:2,5:1,6:1}、2板存活率 30%、晋级率 22%。请判断：

  1. 炸板率是多少？
  2. 梯队是否健康？
  3. 综合判断：进攻/观察/防守？
1. 炸板率 = 55/(95+55) = 36.7%，偏高（>30%）
2. 梯队有递减但 2板存活率仅 30%（18/60），偏低。1板→2板淘汰严重 → 偏弱
3. 观察日。温度达标（涨停多），但质量和持续性存疑（炸板率偏高、2板存活率偏低）。涨停多可能是假繁荣——大资金首板套利，次日无人接力。
