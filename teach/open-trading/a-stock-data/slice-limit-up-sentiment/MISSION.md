# 使命：打板情绪分析全链路

## 为什么
用户在复盘每日市场情绪时，需要快速判断当天涨停/炸板/跌停的整体格局——连板梯队是否健康？炸板率高不高？赚钱效应如何？哪个题材在领涨？本主题训练用户拉通东财四池+同花顺涨停揭秘+情绪速算的完整链路，从原始API数据到结构化情绪判断。

## 成功的样子
- 输入一个交易日日期，能拉取涨停池、炸板池、跌停池、昨涨停池，并计算炸板率、连板梯队、最高连板
- 能从同花顺涨停揭秘中提取涨停原因题材和封板成功率，与东财数据互补验证
- 能综合判断当日打板情绪：是进攻日还是防守日？题材是共振还是分散？

## 约束条件
- 使用 a-stock-data V3.3.0 内置端点，走 push2ex/push2 同源限流
- date 必须传交易日，非交易日 data 返回 null
- 价格字段已 ÷1000，时间字段需特殊处理（东财是HHMMSS整数，同花顺是Unix秒时间戳）

## 不在范围内
- 盘中实时打板监控与推送（属于实时系统设计）
- 历史打板数据的回测框架搭建
- 个股打板策略的编写（本主题聚焦市场情绪分析，非交易策略）
