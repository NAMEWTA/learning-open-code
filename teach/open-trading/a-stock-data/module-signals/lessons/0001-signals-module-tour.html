信号层导览：热门题材归因与北向资金

L1 信号层 第 1/3 节 估计时长：12 分钟

1. 信号层在架构中的位置

信号层是 a-stock-data 的第三层，也是端点最多的数据层（8 个端点）。它位于行情层（K线/盘口）和研报层（EPS/研报）之上，负责把原始行情数据转化为可交易的信号。

交易信号链路（信号层覆盖范围） │ ├── 发现机会 ──── ths_hot_reason() 当日强势股 + 题材归因（同花顺编辑部人工标注） │ industry_comparison() 全行业涨跌排名，识别轮动方向 │ ├── 归属归因 ──── eastmoney_concept_blocks() 个股所属全部板块/概念（一次拿全） │ ├── 资金验证 ──── hsgt_realtime() 北向资金分钟级流向（262个时间点） │ eastmoney_fund_flow_minute() 个股分钟级资金流（主力/大单/中单/小单） │ ├── 席位跟踪 ──── dragon_tiger_board() 个股龙虎榜 + 买卖TOP5席位 + 机构动向 │ daily_dragon_tiger() 全市场龙虎榜汇总 + 净买额排名 │ └── 风险预警 ──── lockup_expiry() 限售解禁日历（历史 + 未来90天）

数据源覆盖 同花顺（热点归因、北向资金）和东方财富（板块归属、资金流向、龙虎榜、解禁、行业排名）。同花顺端零鉴权极速，东财端全部走 em_get() 串行限流。

2. 同花顺热点 — 强势股 + 题材归因

ths_hot_reason(date) 是信号层最独特的端点：它不仅告诉你哪些股票走强，还告诉你为什么走强。reason 字段由同花顺编辑部人工标注，是目前 A 股唯一的"官方级"题材归因标签。

关键特征： 盘后 15:30 更新 · 零鉴权（仅需 UA）· 响应约 73ms · 约 125 只股票 · ST 股无标注 · 数据源：zx.10jqka.com.cn 
from datetime import date
df = ths_hot_reason()  # 默认今天
# df.columns → ['代码','名称','题材归因','涨幅%','换手率%','成交额','大单净量','市场']
# reason 示例："算力租赁+Token工厂+AI政务"（+ 分隔多标签）

# 快速统计当日热门题材
from collections import Counter
tags = []
for r in df['题材归因'].dropna():
    tags.extend(t.strip() for t in str(r).split('+') if t.strip())
cnt = Counter(tags)
print(cnt.most_common(10))  # 当日 TOP10 题材热度

核心字段： 题材归因（reason）是灵魂字段——人工运营的题材标签，如"光刻机""机器人""AI Agent"。大单净量（ddejingliang）反映主力资金偏好。两个字段配合即可完成"什么题材热 + 主力在买什么"的快速扫描。

3. 北向资金 — 分钟级实时流向 + 本地自缓存

hsgt_realtime() 从同花顺 hexin API 拉取沪股通 + 深股通当日分钟级累计净买入（09:10-15:00，最多 262 个时间点），单位为亿元。

已知问题： 东方财富全系北向数据自 2024-08 后净买额返回 NaN/0，属上游断供。因此项目改走同花顺 hexin 源，并设计了本地 CSV 自缓存机制——每次拉取实时数据后自动追加到 ~/.tradingagents/cache/northbound_daily.csv，历史数据越跑越丰富。首次使用时只有当天数据，持续运行即可积累。 
# 实时分钟流向
df_north = hsgt_realtime()
print(f"分钟点数: {len(df_north)}")  # 通常 242-262 个
print(df_north.tail(3))
# → time | hgt_yi（沪股通累计净买入，亿元）| sgt_yi（深股通）

# 收盘快照写入本地缓存
last = df_north.dropna().iloc[-1]
_save_northbound_snapshot("2026-07-08", last["hgt_yi"], last["sgt_yi"])

# 读取最近 20 天历史
hist = _load_northbound_history(20)
print(hist)

缓存路径： ~/.tradingagents/cache/northbound_daily.csv，格式为 date,hgt,sgt。辅助函数 _save_northbound_snapshot() 支持同一天多次写入（覆盖更新），_load_northbound_history(n) 读取最近 N 天。

4. 信号组合：热点题材 + 北向方向

单独看热点或北向都只能得到片面信号。组合使用才能验证"资金是否认可题材"： 

# 1. 当日题材热度词频
df_hot = ths_hot_reason()
# ... 统计 cnt.most_common(10)

# 2. 北向当日净流向
df_north = hsgt_realtime()
hgt_close = df_north["hgt_yi"].dropna().iloc[-1]
sgt_close = df_north["sgt_yi"].dropna().iloc[-1]
print(f"北向收盘: 沪股通 {hgt_close:.1f}亿 / 深股通 {sgt_close:.1f}亿")
