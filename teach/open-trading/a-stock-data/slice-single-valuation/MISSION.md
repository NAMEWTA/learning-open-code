# 使命：单票完整估值全链路

## 为什么
用户希望掌握 A 股个股估值的实战技能——输入一个股票代码，30 秒内拿到实时价、机构一致预期 EPS、前向 PE、PEG 和 PE 消化年数，形成可复用的估值判断框架，用于实际投资决策。

## 成功的样子
- 能独立运行 `full_valuation(code)` 函数，看懂返回的每个估值指标
- 能判断一家公司当前估值是否合理（PE 消化时间、PEG 区间）
- 能识别 EPS 预测数据异常时该如何处理，不盲信返回值

## 约束条件
- 需要先掌握行情层 `tencent_quote`、研报层 `ths_eps_forecast`、基础数据层（已在前置 L1 模块中讲解）
- 代码可直接复制运行，依赖腾讯财经和同花顺接口，无需 API Key

## 不在范围内
- 财务三表深度分析（利润表/资产负债表/现金流量表）
- DCF 绝对估值模型
- 行业横向对比的批量估值（另有 `slice-batch-comparison` 主题）
