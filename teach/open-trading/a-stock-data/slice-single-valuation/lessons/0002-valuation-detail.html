估值公式详解与异常处理

L2 垂直切片 单票估值 第 2/2 节 估计时长：14 分钟

1. 三个公式的防御式设计

估值计算不是纯数学——在真实市场中，EPS 可能为负、CAGR 可能为零、PE 可能已经合理。forward_pe()、calc_peg()、pe_digestion() 三个函数都内建了边界保护。

1.1 forward_pe — 前向市盈率

forward_pe = price / eps_forecast
def forward_pe(price: float, eps_forecast: float) -> float:
    if eps_forecast <= 0:
        return float("inf")
    return price / eps_forecast

只有一个守卫条件：eps_forecast <= 0。亏损公司没有有意义的 PE——10 元股价除以 -0.5 元 EPS 得到 -20 倍 PE，这在投资上毫无意义。返回 inf 是一个清晰信号：PE 不可计算。

设计原则：宁可返回 inf（无法判断）也不要返回一个误导性的负数或极大正值。inf 在后续 PEG 计算中会被自然阻断。

1.2 calc_peg — 成长估值性价比

PEG = pe_fwd / (CAGR * 100)
def calc_peg(pe: float, cagr: float) -> float:
    if cagr <= 0:
        return float("inf")
    return pe / (cagr * 100)

关键细节：为什么是 CAGR * 100？CAGR 是小数（如 0.73 表示 73%），PEG 的定义是 PE 除以增长率百分比数值。所以 78.6 / 73 = 1.08。

PEG 判断标准：

PEG 区间判断含义
< 1便宜成长性完全覆盖估值溢价，有安全边际
1 ~ 1.5合理估值与成长匹配，没有明显高估或低估
> 1.5贵市场对成长性定价过高，需要超预期增长来消化
PEG 不适用场景：壁垒极深但盈利尚未释放的公司，本质是看涨期权而非成长股，PEG 可能 > 3。此时不适用 PEG 框架判断。

1.3 pe_digestion — PE 消化时间

digestion_years = ln(pe_fwd / 30) / ln(1 + CAGR)
def pe_digestion(current_pe: float, cagr: float, target_pe: float = 30) -> float:
    if current_pe <= target_pe:
        return 0.0          # 已经合理，不需要消化
    if cagr <= 0:
        return float("inf")  # 不增长就无法消化
    return math.log(current_pe / target_pe) / math.log(1 + cagr)

数学直觉：假设盈利每年增长 CAGR，PE 每年自然下降（价格不变时）。问：从 current_pe 降到 target_pe 需要多少年？

PE_t = current_pe / (1 + CAGR)^t = target_pe
=> (1 + CAGR)^t = current_pe / target_pe
=> t = ln(current_pe / target_pe) / ln(1 + CAGR)

两个守卫条件各自对应一种情况：

  • current_pe <= 30 → 已经合理，消化年数 = 0
  • cagr <= 0 → 盈利不增长，PE 不会自动下降，返回 inf
消化年数判断
0 ~ 2 年合理 — 估值溢价可接受
2 ~ 4 年偏贵 — 需要较强增长支撑
> 4 年太贵 — 消化时间过长，风险收益比不佳

3. 投资框架速查

a-stock-data SKILL.md 内置的估值判断框架——记住这个顺序：

壁垒 → 增速 → PE消化 → PEG校验 1. 有壁垒吗？(tech_moat / capacity_moat) └── 没有则排除，不要继续 2. 增速多少？ └── CAGR > 30% 才有意义 3. PE 多久消化到 30x？ ├── < 2 年 → 合理 ├── 2~4 年 → 偏贵但可接受 └── > 4 年 → 太贵 4. PEG 多少？ ├── < 1 → 便宜 ├── 1~1.5 → 合理 └── > 1.5 → 贵
使用顺序很重要：先判断壁垒（没壁垒排除），再看增速（< 30% 不追），然后消化时间（量化估值消化速度），最后 PEG 校验（最后的性价比检查）。四步缺一不可。

判断练习：估值速算

某股票当前价 150 元，eps_cur = 3.0 元，eps_next = 4.5 元。不翻看上文，口算以下指标并给出判断：

  1. 前向 PE = ____
  2. CAGR = ____%
  3. PEG = ____（便宜/合理/贵？）
  4. 消化年数约 ____ 年（用近似：ln(1.67)/ln(1.5) ≈ ?）
1. 前向 PE = 150 / 3.0 = 50
2. CAGR = 4.5 / 3.0 - 1 = 50%
3. PEG = 50 / 50 = 1.0 → 合理
4. 消化年数 = ln(50/30) / ln(1.5) = ln(1.667) / ln(1.5) ≈ 0.511 / 0.405 = 约 1.26 年 → 合理

综合判断：50 倍 PE 看起来贵，但 50% 增速下 PEG=1.0、消化只需 1.3 年——估值合理。
