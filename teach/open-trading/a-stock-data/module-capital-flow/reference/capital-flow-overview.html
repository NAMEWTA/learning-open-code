四、资金面组合诊断矩阵

诊断维度端点看多信号看空信号
杠杆方向 margin_trading() 融资余额持续上升 + 买入 > 偿还 融资余额快速下降 + 偿还 > 买入
机构动向 block_trade() 溢价成交 + 买方为"机构专用" 大幅折价 + 卖方为原始股东
筹码结构 holder_num_change() 股东户数持续减少 + 户均持股上升 股东户数持续增加 + 户均持股下降
回报质量 dividend_history() 每股派息稳定增长 + 进度完成 派息逐年下降 + 预案被否
主力意图 stock_fund_flow_120d() 20日/60日主力累计净流入 连续多日主力净流出

五、数据源依赖关系

资金面/筹码层 数据源依赖 │ ├── eastmoney_datacenter() ← 被 4 个端点调用（内含 em_get 限流） │ ├── margin_trading() → RPTA_WEB_RZRQ_GGMX │ ├── block_trade() → RPT_DATA_BLOCKTRADE │ ├── holder_num_change() → RPT_HOLDERNUMLATEST │ └── dividend_history() → RPT_SHAREBONUS_DET │ └── em_get() 直连 ← push2his 不走 datacenter └── stock_fund_flow_120d() → push2his.eastmoney.com (+ UA + Referer + Origin 三件套)

六、已知限制

限制项影响端点说明
融券数据趋零 margin_trading() 2024 融券新规后，大部分中小盘股融券余额已无参考价值
大宗交易非每日发生 block_trade() 没有大宗交易的日期返回空列表，不等于数据缺失
股东户数为季度数据 holder_num_change() 每年仅 4 个数据点，不能用于短期交易判断
push2his 间歇风控 stock_fund_flow_120d() 大陆住宅 IP 偶发 HTTP 000/空，需重试或换网络
分红进度非实时 dividend_history() ASSIGN_PROGRESS 可能滞后于实际公告
