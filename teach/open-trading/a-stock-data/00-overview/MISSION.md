# 使命：a-stock-data 项目总览

## 为什么
学习者希望掌握一个完整的 A 股数据获取工具链，能够不依赖任何第三方封装库（如 akshare），直连底层 HTTP API 获取行情、研报、资金流向、龙虎榜、公告等全量数据，用于搭建自己的量化投研工作流。a-stock-data 是目前唯一做到"自包含单文件 + 10 层数据架构 + 40 端点"的 A 股数据工具包，掌握它就能解锁一个完整的投研数据基础设施。

## 成功的样子
- 能画出 a-stock-data 的 10 层数据架构图，说出每层包含的数据源和典型端点
- 能解释「数据源优先级体系」——为什么 mootdx/腾讯优先、东财仅用于独有数据
- 能理解 em_get() 防封机制的工作原理，知道什么时候需要调大 EM_MIN_INTERVAL
- 能规划自己的学习路线：先学基础设施 → 各数据层 → 调研流程 → 估值公式

## 约束条件
- 学习时间：每天 30-60 分钟，利用碎片时间
- 基础要求：会 Python 基础（requests/pandas），了解 A 股基本概念
- 环境：需要能 pip install mootdx requests pandas stockstats

## 不在范围内
- 不涉及具体 A 股投资策略或选股方法论
- 不涉及量化回测框架（backtrader/zipline 等）
- 不涉及实时交易执行（量化交易接口）
