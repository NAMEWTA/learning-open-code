一、四源能力矩阵

数据源协议封IP风险依赖函数核心能力限制
mootdx finance() TCP 7709 不封 tdx_client() 37字段季报快照（EPS/ROE/净利润/主营收入/流通股本等） 仅最新单期，无历史序列
mootdx F10() TCP 7709 不封 tdx_client() 九大类公司文本资料（概况/股东/财务分析/资本运作等） 原始文本需截断优化，无结构化字段
东财 push2 HTTP 有风控 em_get()（单次可免） 8字段个股基本信息（行业/总股本/流通股/市值/上市日期） 只供元信息，无财务数据
新浪 quotes HTTP 低 直连 requests 财报三表（资产负债表/利润表/现金流量表），支持多期同比 科目名为中文字段，无标准化 key；report_type 需精确匹配

注：mootdx finance() 和 F10() 共享同一 TCP 连接（由 tdx_client() 管理），东财 push2 走 HTTP（在基础设施层中属于优先级 4——仅用于东财独有数据）。
