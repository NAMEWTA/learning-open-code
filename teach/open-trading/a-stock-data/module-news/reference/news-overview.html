三、字段对照表

字段名原始字段含义备注

四、JSONP 解析流程

eastmoney_stock_news() 数据流 │ ├── 1. 构造 inner_params JSON │ keyword=code, type=["cmsArticleWebOld"], pageIndex/pageSize │ ├── 2. 发送 GET 请求（走 em_get 限流） │ URL: search-api-web.eastmoney.com/search/jsonp │ Headers: UA + Referer (so.eastmoney.com) │ ├── 3. 解析 JSONP 响应 │ "jQuery_news({...})" → text.index("(") → text.rindex(")") │ ├── 4. 提取文章列表 │ result.cmsArticleWebOld → 直接是 list (V3.2.1 修复) │ └── 5. 清理 HTML 标签 re.sub(r'<[^>]+>', '', title/content)

五、风控与已知限制

问题影响端点表现应对
间歇 IP 风控 eastmoney_stock_news() 返回仅含 passportWeb，无 cmsArticleWebOld 隔几分钟重试 / 换网络 / 降频
间歇连接风控 两者均可能 HTTP 000 / 连接超时 重试 / 换网络 / 调大 EM_MIN_INTERVAL
财联社 API 下线 cls_telegraph() HTTP 404 永久迁移到 eastmoney_global_news()
新闻无去重 两者 同一条新闻可能出现在两个源 应用层自行去重
