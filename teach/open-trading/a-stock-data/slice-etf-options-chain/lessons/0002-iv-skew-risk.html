ETF期权链分析 — IV偏度与风险分析

L2 垂直切片 第 2 节 / 共 2 节 估计时长：14 分钟

1. IV偏度 — 市场情绪的量化信号

IV偏度（Implied Volatility Skew）是指同一到期月份、不同行权价的期权 IV 呈现倾斜曲线。偏度方向反映市场对尾部风险的定价：

偏度形态IV 特征市场解读常见场景
左偏 低行权 IV > 高行权 IV 担忧下跌，愿意为下行保护付更高权利金 财报前、政策不确定性
右偏 高行权 IV > 低行权 IV 预期上涨，追涨情绪浓厚 利好事件驱动
微笑 两端高、中间低 同时定价上涨和下跌的尾部风险 重大事件前夕
# 拉取近月认购+认沽 IV 曲线 codes_call = sina_option_codes("510050", call=True) codes_put = sina_option_codes("510050", call=False) near = list(codes_call)[0] chain = [] for c in codes_call[near]: tq, gk = sina_option_tquote(c), sina_option_greeks(c) if tq and gk: chain.append({**tq, **gk}) chain.sort(key=lambda x: x['strike']) for r in chain: print(f"K={r['strike']:.3f} IV={r['iv']:.2%}") # 偏度 = 最低行权IV - 最高行权IV skew = chain[0]['iv'] - chain[-1]['iv'] print(f"偏度={skew:+.1%} → {'左偏偏空' if skew>0.03 else '右偏偏多' if skew<-0.03 else '平坦'}")
快速判断：低行权 IV 高出高行权 3%+ → 左偏（偏空）；反之右偏（偏多）。A股 ETF 期权 IV 通常集中在 15~25%，3% 差值已是标准偏差级别信号。

2. 希腊字母风险评估框架

指标含义风险信号安全区间
Delta方向敏感度过小→深度虚值，大涨也难赚钱0.3~0.7
GammaDelta 稳定性平值最大→需频繁盯盘< 3
Theta每日衰减到期前 5 天急剧放大，"烧钱"加速< |0.005|
Vega波动率敞口高 IV 买入后 IV 回落 = IV crushIV 15~25%
IV价格贵贱>30%=贵；=0=数据异常15~25%
到期日警告：距离到期日不足 5 天时，平值期权 Gamma 和 Theta 急剧放大（"Gamma 暴击"），买方每天损耗可达合约价 5~10%，非强方向判断不持仓过夜。

3. 三类异常路径与降级

异常1 — 非交易时段：盘前/周末 tquote 列表长度 < 43 或 greeks 返回空。降级：检查长度后跳过，标记"非交易时段"，不用 0 填充。
异常2 — IV 为零：深度虚值冷门合约 IV=0，与同月其他合约显著偏离。降级：IV < 0.01 的合约从偏度分析排除，保留 tquote 供流动性评估。
异常3 — 到期日：到期当天 Delta 二元化（趋 0 或 1），希腊字母失去指导意义。降级：跳过当月合约（YYMM=当前月），只分析次近月。

4. 综合实战摘要

def analyze_chain(underlying="510050", call=True, n=11): codes = sina_option_codes(underlying, call=call) if not codes: return near = list(codes)[0] all_codes = codes[near] start = max(0, len(all_codes)//2 - n//2) target = all_codes[start:start+n] chain = [] for c in target: tq, gk = sina_option_tquote(c), sina_option_greeks(c) if tq and gk and gk.get('iv',0) > 0.01: chain.append({"code":c, **tq, **gk}) chain.sort(key=lambda x: x['strike']) if len(chain) < 2: return low_iv, high_iv = chain[0]['iv'], chain[-1]['iv'] skew = low_iv - high_iv iv_list = [r['iv'] for r in chain] avg_iv = sum(iv_list)/len(iv_list) print(f"近月={near} | IV偏度={skew:+.1%} | 均IV={avg_iv:.1%}")
