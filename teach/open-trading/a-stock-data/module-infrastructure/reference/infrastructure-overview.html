一、核心函数总表

函数定位功能关键设计
em_get() 东财统一入口 所有 eastmoney.com 请求的节流代理 串行限流(≥1s+随机抖动) + Keep-Alive会话复用 + 连接级重试(total=3, 指数退避, 403不重试)
tdx_client() mootdx 连接工厂 创建通达信客户端，规避 0.11.x BESTIP bug TCP探测(_probe) → BESTIP测速 → 裸factory → RuntimeError明确报错。支持 0.10/0.11 双版本
get_prefix() 市场前缀判断 6位代码 → sh/sz/bj 6/9→sh, 8→bj, 其余→sz。所有层通用
eastmoney_datacenter() 数据中心查询 龙虎榜/解禁/两融/大宗/股东/分红 统一查询 统一 reportName 路由 + columns/filter/sort 参数体系，已内置 em_get 限流
EM_SESSION 全局 HTTP 会话 Keep-Alive 连接复用 挂载 HTTPAdapter(max_retries=3, backoff_factor=0.6), 默认 UA=Mozilla/5.0...
EM_MIN_INTERVAL 限流参数 两次东财请求最小间隔（秒） 默认 1.0s。批量建议 1.5-2s。配合 0.1-0.5s 随机抖动
_probe() TCP 探测 判断通达信服务器可达性 socket.create_connection, timeout=2s。3次握手即返回，不等应用层
_TDX_SERVERS 服务器列表 10 个通达信IP:7709 按延迟排序，2026-06 验证。需自行维护更新
