1. 项目定位与技术栈

TradingAgents-Astock 是基于 TauricResearch/TradingAgents（65K Stars）的 A 股深度特化 fork——零外部服务依赖、pip install 即跑的多 Agent LLM 投研框架。用 7 个 AI Analyst 分析一只 A 股，经 Bull vs Bear 辩论和三方风险辩论后，由 Portfolio Manager 输出 Buy/Hold/Sell 决策，全链路约 30-50 次 LLM 调用。

核心差异 原版面向美股，本 Fork 从数据层（mootdx/东财/新浪/同花顺直连）、Agent 角色（+3 个 A 股特化）、交易规则（T+1/涨跌停/手数）三个维度深度改造。

3. 核心架构决策

  1. Vendor 路由模式：interface.py 的 VENDOR_METHODS 字典为每个数据方法注册 a_stock / yfinance / alpha_vantage 实现，切换数据源只需改 config，新增 vendor 不改调用方代码。
  2. 7 Analyst 角色模型：原版 4 个 + A 股特化 3 个——政策分析师（政策市）、游资追踪师（短线定价核心）、解禁监控师（供给冲击），每个携带专属 ToolNode。
  3. A 股交易约束：Trader 内置 T+1、涨跌停、100 股最小手数、交易时段检查。
  4. 输出语言分层：用户报告中文输出，内部辩论保持英文（当前 LLM 金融推理英文质量更优）。
  5. 零外部服务依赖：7 个数据源全部免费直连，不依赖 Tushare 或任何付费 API。
