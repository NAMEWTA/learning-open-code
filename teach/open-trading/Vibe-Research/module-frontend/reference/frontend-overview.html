组件文件Props用途

AskAiButton components/ui/AskAiButton.tsx context, suggestions?, label? 嵌入式 AI 对话面板：打开后展示对话框，把当前页面上下文喂给用户 AI，支持流式吐字、工具调用可视化（查行情/查估值/查研报/查新闻）、中止请求、未配置时引导跳转设置页。对话结果可通过 SaveNoteButton 存入沉淀。
SaveNoteButton components/ui/SaveNoteButton.tsx kind, title, content 「存入沉淀」按钮：点击后将内容写入 localStorage（调 addNote()），成功后变灰显示「已存入沉淀」。
EarningsSnapshot components/ui/EarningsSnapshot.tsx val: Valuation, fin: Financials?, pctl: ValPercentile? 财报速览卡：结论先行结构，展示估值（PE-TTM/PB/前向PE/PEG）+ 财报（营收/利润同比/ROE/毛利率/净利率）+ 估值分位（PE-TTM 历史分位值）。A股红涨绿跌配色。
Disclaimer components/ui/Disclaimer.tsx compact? 中立免责声明：声明产品只呈现客观公开数据，不推荐、不预测、不构成投资建议。紧凑模式（11px 单行）和完整模式（卡片样式）。
ErrorBoundary components/common/ErrorBoundary.tsx children, fallback? React 类组件错误边界：捕获子组件渲染异常，展示 AlertTriangle 图标 + 错误信息。可在 main.tsx 中看到它包裹整个应用。

""; for (const line of lines) { const ev = JSON.parse(line.trim()); if (ev.type === "delta") onDelta?.(ev.text); // 逐字吐字 else if (ev.type === "tool") onTool?.(ev.tool, ev.args); // 工具调用 else if (ev.type === "done") { trace = ev.trace; rounds = ev.rounds; } else if (ev.type === "error") errMsg = ev.message; } }
