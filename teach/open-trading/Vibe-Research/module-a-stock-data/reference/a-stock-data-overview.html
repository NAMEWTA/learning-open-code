A股数据工具箱速查 a-stock-data v3.3.0

40 个端点、13 个数据源、10 层架构。本页是快速查阅参考，详细代码和使用说明见 SKILL.md。

1. 十层架构总图

行情层（①）→ 研报层（②）→ 信号层（③）→ 资金面（④）→ 新闻层（⑤） → 基础数据（⑥）→ 公告层（⑦）→ 打板层（⑧）→ 期权层（⑨）→ 舆情互动（⑩） 不封 IP 层（①⑥ 部分）：mootdx(TCP 7709) + 腾讯财经(HTTP) 东财独有层（②③④⑤⑦⑧⑩ 部分）：必须走 em_get() 限流，串行 ≥1s + 随机抖动

2. 完整端点清单（按层）

4. 东财防封速查

行为触发封禁的阈值风险
每秒请求数> 5 次/秒高
单 IP 并发连接≥ 10高
1 分钟请求总数≥ 200 次中高
5 分钟请求总数≥ 300 次触发封禁
User-Agent空 UA / 无浏览器特征中

被封表现：连续请求后 403 / 429 / 连接超时 / 返回空数据。临时封禁通常几分钟到几小时。

5. 全局 Helper 速查

Helper用途适用场景
get_prefix(code)6位代码 → 市场前缀 (sh/sz/bj)所有需要市场前缀的接口
tdx_client(market='std')创建 mootdx 客户端，规避 0.11.x BESTIP bug所有 mootdx 调用
em_get(url, params, headers)东财统一请求入口：自动节流 + 会话复用 + 默认 UA所有 eastmoney.com 接口
eastmoney_datacenter(report_name, columns, filter_str)东财数据中心统一查询龙虎榜/解禁/两融/大宗/股东/分红
EM_MIN_INTERVAL两次东财请求最小间隔（默认 1.0s，批量建议 1.5~2s）批量调用时调大降速
