gstock.py 源码结构

backend/gstock.py ← 美股/港股数据层 (183行) │ ├── _INDICES ← 全球指数列表（道指/标普/纳指/恒生/恒生科技） ├── _MKT ← MktNum→(后缀,市场名) 映射 ├── _QUOTE_FIELDS ← 行情字段选择（11字段） │ ├── _push2_stock_get(secid, fields) ← 东财push2行情（push2优先→push2delay降级） ├── _price(d, key) ← 价格字段还原（÷10^f59） ├── _quote_from(d) ← 行情dict标准化 │ ├── global_indices() ← 全球指数快照 ├── _search(q) ← 东财搜索+精确代码匹配 ├── resolve_symbol(query) ← 代码/名称→市场信息解析 ├── _key_metrics(secucode) ← 东财GMAININDICATOR最新一期 └── us_hk_stock(query) ← 主入口：代码→行情+财务指标聚合
