2. 请求体与校验

端点请求体字段校验规则
POST /holding code: string, shares: number, cost: number code = 6位纯数字；shares > 0；cost 不限正负（允许融券/返息负成本）
DELETE /holding ?code=XXXXXX（Query 参数） 无额外校验，按 code 精确匹配删除
POST /close code, date, price, shares, cost code = 6位数字；date = "YYYY-MM-DD" 可解析；price > 0；shares > 0；cost 不限正负
DELETE /close ?index=N（Query 参数） index 范围 [0, len(closed))，否则无操作
POST /refresh 无请求体 无校验
