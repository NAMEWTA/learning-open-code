# 教学笔记：MCP 工具注册与调用全链路

## 用户偏好
- 偏好从代码入口点出发理解系统架构
- 关注异常处理路径（错误恢复、安全过滤）
- 需要清晰的函数调用链和数据转换过程

## 教学注意事项
- 本节是 L2 垂直切片，依赖于 L1 的模块概览课程
- 5 层链路较长，建议分段学习，每层确认理解后再进入下一层
- 异常路径（第 7 节）是本节的重点价值之一，不要跳过
- 交叉链接中的同级 L2 课程有助于对比理解不同的垂直切片
