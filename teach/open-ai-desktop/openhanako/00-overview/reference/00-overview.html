1. 项目定位

HanaAgent 是一个有记忆、有灵魂的私人 AI 助理桌面应用。与传统的 Coding Agent（如 Claude Code、Codex）不同，HanaAgent 面向所有电脑用户——它强化了 Agent「像人」的属性（人格、记忆、主动行动），同时针对日常办公场景做了工具性和流程性优化，并提供完备的图形界面。

核心理念：弥合绝大多数人和 AI Agent 之间的缝隙，让强大的 Agent 能力不再只局限于命令行里。

一句话描述：基于 Electron + React 的跨平台桌面 AI Agent，具备记忆系统、多 Agent 协作、插件架构、安全沙盒、多平台消息接入能力。

数据流概要

  1. 用户输入 → React 渲染进程 → HTTP/WS → Hono Server → Engine → SessionCoordinator
  2. Agent 响应 → Pi SDK 流式生成 → EventBus → WebSocket → 渲染进程实时渲染
  3. 工具调用 → Agent → Tool Registry → 对应工具实现（文件读写/命令执行/浏览器操作）→ 结果回传 Agent
  4. 外部消息 → Telegram/飞书/QQ Bridge → BridgeSessionManager → EventBus → Agent 处理
  5. 定时任务 → CronScheduler (lib/desk) → Agent Executor (hub) → Agent 自动执行

5. 核心设计哲学 / 架构决策

5.1 双层隔离安全模型

应用层 PathGuard + 操作系统级沙盒：

  • 应用层 lib/sandbox/path-guard.ts — 四级访问控制（只读/写入/删除/网络），限制在工作目录与受控数据目录
  • 系统层 — macOS Seatbelt、Linux Bubblewrap (lib/sandbox/bwrap.ts)、Windows Restricted Token
  • 用户可在设置 → 安全页面调整沙盒级别；外部网络可配置系统代理/手动代理/直连

5.2 约定优先的插件架构

插件系统是 HanaAgent 的核心扩展机制：

  • 两级权限：restricted（默认，仅 tools/skills/commands）vs full-access（routes/extensions/providers/生命周期）
  • 拖拽安装：文件夹或 zip 拖入即安装，即时生效无需重启
  • PluginContext 注入：路由可直接访问核心服务（Session Bus、Agent 对话、历史管理）
  • SDK 四包体系：protocol（协议）→ sdk（前端）→ runtime（Node）→ components（React UI）
  • 开发闭环：Agent 辅助开发 → EventBus 安装/刷新/测试 → 场景自动化测试

5.3 Agent = 文件夹

每个 Agent 是独立的文件夹，包含人格文件、记忆数据、定时任务配置：

  • 记忆隔离：每个 Agent 拥有独立记忆，通过编译式快照实现近期记忆牢固
  • 人格模板：通过 lib/identity-templates/ 和 lib/yuan/ 定义角色，Agent 之间风格分离清晰
  • 角色卡导出：按白名单携带人格、头像、可选记忆和 Skills，导出为 zip
  • 频道群聊协作：Agent 之间通过 hub/channel-router.ts 频道通信，可互相委派任务

5.4 Server-First 架构

桌面端与 Server 分离为独立进程：

  • Server 作为独立 Node.js 子进程运行（child_process.spawn，非 fork），避免阻塞 Electron 主进程
  • 同一 Server 支持桌面端、LAN 接入、移动端 PWA（/mobile/）同时连接
  • 设备访问密钥 + 本地账号双重认证

5.5 书桌（Desk）异步协作模型

每个 Agent 拥有独立书桌（lib/desk/）：

  • 文件拖拽操作 + 文件预览 + 文件树变更监听 (chokidar)
  • 笺（类似便签）：Agent 主动读取并执行
  • Cron 定时任务：调度"触发条件"与"执行动作"分离
  • Agent 定期巡检书桌变化，自动化执行器支持复杂任务/轻量通知/插件动作

5.6 其他关键设计

  • 流式会话与 JSONL 持久化：core/session-jsonl-file.ts 以 JSONL 格式记录完整会话历史，支持搜索、归档与恢复
  • 会话压缩：core/session-compactor.ts 自动压缩长对话上下文，保持 Agent 记忆窗口高效
  • 技能市场兼容：兼容庞大 Skills 社区生态，Agent 可主动从 GitHub 安装技能并自学新技能
  • 国际化：桌面端支持 5 种语言（中/英/日/韩/繁），通过 desktop/src/locales/ 实现
  • 多模型支持：OpenAI 兼容、Anthropic 风格、OAuth Provider、Ollama 本地模型，通过 lib/providers/ 统一适配
