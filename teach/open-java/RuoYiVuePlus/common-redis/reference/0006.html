关键方法备忘

  • isSaveTtl：setCacheObject(key, value, true) 保留原 TTL（验证码场景）
  • setIfAbsent：不存在才写入，防重提交底层（返回 boolean）
  • delMultiCacheMapValue：用 RBatch 批量异步删 Map field，一次网络往返
  • keys：底层 SCAN（非 KEYS），chunkSize=1000
  • deleteObject(Collection)：用 RBatch 批量异步删除
  • rateLimiter：RRateLimiter 令牌桶，返回 -1=失败
  • publish/subscribe：Redis Pub/Sub，消息不持久化

使用方式

  • 全 static 方法，构造器私有，直接用 RedisUtils.xxx()
  • CLIENT = SpringUtils.getBean(RedissonClient.class)
  • ⚠ 必须在 Spring 容器就绪后才能用
