三个自定义指令

// 1. v-hasPermi — 权限按钮控制（DOM 移除）
<el-button v-hasPermi="['system:user:add']">新增</el-button>

// 2. v-hasRoles — 角色按钮控制（DOM 移除）
<el-button v-hasRoles="['admin', 'common']">管理</el-button>

// 3. v-copyText — 一键复制
<span v-copyText="'要复制的文本'">点击复制</span>
// 实现：创建隐藏 textarea → select() → execCommand('copy') → 移除

复制指令核心实现

function copyTextToClipboard(input) {
  const el = document.createElement('textarea');
  el.value = input;
  el.setAttribute('readonly', '');
  el.style.cssText = 'position:absolute;left:-9999px';
  document.body.append(el);
  el.select();
  document.execCommand('copy');  // 经典 API，navigator.clipboard 需要 HTTPS
  el.remove();
}

CRUD 页面组合模式

// 标准 CRUD 页面 = 以下组合
<right-toolbar v-model:showSearch="showSearch" :columns="columns"
              @queryTable="getList" @refresh="getList" />
<el-table @selection-change="handleSelectionChange">...</el-table>
<pagination v-show="total>0" :total="total"
            v-model:page="queryParams.pageNum"
            v-model:limit="queryParams.pageSize"
            @pagination="getList" />
注意：execCommand('copy') 已废弃但作为降级方案可用。SvgIcon 依赖 vite-plugin-svg-icons-ng 构建时注册。工作流组件仅在 workflow 模块中使用。
