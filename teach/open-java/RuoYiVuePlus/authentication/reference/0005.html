失败次数限制

SysLoginService.checkLogin(type, name, supplier)：错 N 次锁 M 分钟，计数存 Redis，成功清零。带凭证校验的方式都复用。

social 绑定线

已登录 → /auth/binding/{source} 拿授权URL → 三方回调 → /auth/social/callback → socialRegister() 落 sys_social。登录时 source+uuid 反查 userId。

注册（SysRegisterService:43）

开关 selectRegisterEnabled → 图形码 → 重名校验 → BCrypt.hashpw 存。密码永远单向哈希，无明文。
