# NOTES -- 教学偏好暂存

> 记录用户在课程中表达的教学偏好，供后续课程设计参考。

- 用户要求「真实代码引用、漂亮HTML+练习、简体中文」——课程大量引用真实 POM 文件路径和 XML 代码行，每课均设提取练习。
- 偏好「鸟瞰图 + 分步拆解 + 提取练习」的结构（沿用 teach-ruoyi-sms 和 teach-ruoyi-redis 的 HTML 风格）。
- BOM 模块虽无 Java 代码，但作为 Maven 版本管理核心概念，讲解需兼顾「理论原理」与「本仓库具体实践」两个层次。
- 每课的「想一想」折叠区用于引导深度思考，不应直接给答案（留给学习者先想）。
