ruoyi-common-mail 速查表

Maven: org.dromara:ruoyi-common-mail | ruoyi-common-mail/src/main/java/org/dromara/common/mail/

定位

对 Jakarta Mail + Hutool 的项目化封装模块。仅 3 个 Java 类，核心产物是 MailBuilder —— 一个 Builder 模式的流式邮件构建器，让业务代码用一行链式调用即可发送邮件（含附件、内嵌图片、抄送密送）。

三层架构

层类职责
配置映射MailProperties@ConfigurationProperties(prefix="mail") 绑定 yml 配置，toMailAccount() 转 Hutool MailAccount
装配控制MailConfig@AutoConfiguration + @ConditionalOnProperty(mail.enabled=true) 创建 MailAccount Bean
流式发送MailBuilder私有构造 + 静态工厂 of() + 链式设值 + send() 终止，唯一对外 API

核心文件

类名路径职责
MailBuildercore/MailBuilder.java唯一对外 API，Builder 模式：of() → 链式设值 → validate() → resolveMailAccount() → JakartaMail.create() → send()
MailConfigconfig/MailConfig.java自动配置类，@ConditionalOnProperty(mail.enabled=true) 控制 MailAccount Bean 创建
MailPropertiesconfig/properties/MailProperties.java绑定 yml 中 mail: 段 10 个属性，toMailAccount() 转换为 Hutool MailAccount

术语速查

术语解释
Jakarta MailJakarta EE 邮件 API 标准（javax.mail 的品牌迁移），本模块通过 Hutool 间接使用
MailAccountHutool 邮件账户配置对象，封装 SMTP 地址/端口/用户名密码/SSL/超时
MailPropertiesSpring Boot 配置属性 POJO，@ConfigurationProperties(prefix="mail") 自动绑定 yml
MailConfig模块总开关，@ConditionalOnProperty 控制是否创建 MailAccount Bean
MailBuilder流式构建器，of() 创建 → 链式设值返回 this → send() 终止并发送
JakartaMailHutool 对 Jakarta Mail 的封装类，MailBuilder.send() 底层最终调用它
useGlobalSession内部控制标志，当调用 account()/from()/user()/pass() 覆盖时自动置 false，触发账户克隆
Builder 模式每个设值方法返回 this，分离构建过程与最终产出，支持链式语法

常用代码

配置文件

# application.yml
mail:
  enabled: true
  host: smtp.example.com
  port: 465
  user: noreply@example.com
  pass: your-password
  from: noreply@example.com
  sslEnable: true
  timeout: 10000
  connectionTimeout: 5000

基础发送

MailBuilder.of()
    .to("user@example.com")
    .subject("测试邮件")
    .text("这是一封测试邮件")
    .send();

HTML 正文 + 附件 + 抄送

MailBuilder.of()
    .to("user@example.com")
    .cc("manager@example.com")           // 抄送
    .bcc("archive@example.com")          // 密送
    .subject("项目周报")
    .html("<h1>项目周报</h1><p>本周完成...</p>")
    .attachment("report.pdf", pdfBytes)  // 附件
    .send();

内嵌图片（在 HTML 中引用 cid:xxx）

MailBuilder.of()
    .to("user@example.com")
    .subject("带图片的邮件")
    .html("<img src='cid:logo'><h2>欢迎</h2>")
    .image("logo", imageInputStream)     // cid 在 HTML 中引用
    .send();

临时切换发件人

// account()/from()/user()/pass() 任一覆盖都会克隆全局 MailAccount
MailBuilder.of()
    .from("another@example.com")
    .user("another@example.com")
    .pass("another-password")
    .to("user@example.com")
    .subject("使用临时账户发送")
    .text("hello")
    .send();
