一句话本质

ruoyi-snailai-server 是 Snail AI 平台的独立运行时（部署载体）——只写启动入口和安全配置，AI 能力由 snail-ai-starter 通过 Spring Boot 依赖注入与配置声明装配合成。

模块对比

维度ruoyi-ai（客户端适配器）ruoyi-snailai-server（平台本体）
代码量1 个 Java 文件3 个 Java 文件

部署方式嵌入 ruoyi-admin 进程独立进程，HTTP 8900 + gRPC 18888
依赖 ruoyi-common是否——完全独立

三个核心设计决策

决策含义
不依赖任何 ruoyi-common完全独立的应用——不共享 Sa-Token、Web 基础设施、异常处理框架。任何系统都可通过 gRPC 调用
仅依赖 snail-ai-starter所有 AI 能力（模型、RAG、Agent、gRPC、记忆）由 starter 自动配置引入，server 不需要知道内部细节
配置文件行数远超代码行数"装配式平台"特征——代码定义骨架，配置注入血肉
