认证流程速览

请求 → SaTokenFilter 拦截 ├─ 读取 Header: Authorization (Bearer xxx) ├─ SaTokenDao.get("Authorization:login:token-session:" + token) │ ├─ Caffeine 命中? → 返回 │ └─ Caffeine 未命中? → Redis 读取 → 写入 Caffeine → 返回 ├─ 校验 Session 存活 & Token 有效期 ├─ 注入 LoginUser 到 ThreadLocal ├─ 校验 @SaCheckPermission 等注解 │ └─ 调用 StpInterface.getPermissionList() │ └─ SaPermissionImpl → LoginUser.permissions └─ Controller 执行

关键配置速查

配置项 默认值 作用
sa-token.token-name Authorization Token 在请求头中的键名
sa-token.token-prefix Bearer Token 值的前缀标识
sa-token.timeout 86400 Token 默认有效期（秒）
sa-token.activity-timeout 1800 无操作过期时间（秒）
sa-token.is-concurrent true 是否允许并发登录
sa-token.is-share true 多端共享同一 Token
sa-token.is-log false 是否输出 Sa-Token 操作日志
sa-token.jwt-secret-key （空） JWT 签名密钥
