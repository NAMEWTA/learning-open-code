login() 写入的数据

以下为 LoginHelper.login(loginUser) 执行时写入 TokenSession 的全部数据：

  • TokenSession extra data（常用字段快速访问）：
    • userId → 用户ID
    • userName → 用户名称
    • deptId → 部门ID
    • deptName → 部门名称
    • deptCategory → 部门类别
  • TokenSession object（完整对象存储）：
    • LOGIN_USER_KEY ("loginUser") → LoginUser（完整用户对象，包含权限、角色等所有信息）
