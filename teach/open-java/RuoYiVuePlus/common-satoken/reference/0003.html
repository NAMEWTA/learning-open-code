resolveUserId 解析逻辑

/**
 * 从 Sa-Token 的 loginId 中解析出真实的用户 ID
 *
 * Sa-Token 的 loginId 格式: "loginType:userId:deviceType"
 * 例如: "login:1:web" 表示登录类型=login, userId=1, 设备=web
 *
 * 此处取第二段作为 userId，若格式不符则直接将整个 loginId 作为 userId
 */
private Long resolveUserId(Object loginId) {
    if (loginId == null) return null;
    String str = loginId.toString();
    String[] parts = str.split(":");
    if (parts.length >= 2) {
        return Long.valueOf(parts[1]);
    }
    return Long.valueOf(str);
}

loginId 格式说明

格式示例 用户类型前缀 解析出的userId 使用场景
login:1:web login 1 后台用户 Web 端登录
login:5:app login 5 后台用户 App 端登录
app:100:app app 100 App 用户登录
1 — 1 纯数字格式（兼容旧数据）
