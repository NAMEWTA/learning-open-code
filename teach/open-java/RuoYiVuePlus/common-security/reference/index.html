ruoyi-common-security 速查表

Maven: org.dromara:ruoyi-common-security | 源码: ruoyi-common-security/src/main/java/org/dromara/common/security/

定位

在 ruoyi-common-satoken 提供的基础 Sa-Token 能力之上，用 3 个 Java 类搭建应用级安全横切配置——上下文过滤器重注册、路由鉴权链、URL 自动收集。

核心文件

类名路径职责
SecurityConfigconfig/SecurityConfig.java一站式安全装配点：重注册 Sa-Token 上下文过滤器（覆盖 ASYNC/ERROR）、注册 SaInterceptor 路由鉴权链、注册 actuator Basic Auth 过滤器
SecurityPropertiesconfig/properties/SecurityProperties.java@ConfigurationProperties(prefix="security")，仅含 String[] excludes，定义无需登录鉴权的路径
AllUrlHandlerhandler/AllUrlHandler.java实现 InitializingBean，从 RequestMappingHandlerMapping 收集全系统 URL，{xxx} 替换为 * 通配符，供 SaRouter.match() 使用

术语速查

术语解释
Sa-TokenDromara 轻量级 Java 权限认证框架，核心 API 为 StpUtil，提供登录认证、权限认证、Session 会话等能力
SecurityConfig本模块核心配置类，注册三条安全防线：上下文过滤器 → 路由拦截器 → actuator Basic Auth 过滤器
AllUrlHandlerURL 扫描收集器，在 Spring 容器初始化后自动收集所有 @RequestMapping 路径
FilterRegistrationBeanSpring Boot Servlet Filter 注册 API，用于重注册 Sa-Token 上下文过滤器，关键设置 setDispatcherTypes(REQUEST, ASYNC, ERROR)
SaInterceptor + SaRouter路由拦截器与链式路由匹配 API：SaRouter.match(urls).check(() -> { ... }) 对所有匹配 URL 执行鉴权
LoginHelper登录用户工具类（ruoyi-common-satoken），提供 CLIENT_KEY、CLIENT_ACCESS_PATH_KEY、CLIENT_IP_WHITELIST_KEY 等 token 扩展字段 key
InitializingBeanSpring 生命周期接口，AllUrlHandler 在 afterPropertiesSet() 中执行 URL 收集——是唯一正确时机（RequestMapping 已注册完、拦截器未装配）

三层安全防线

防线Bean/组件时机职责
第1层saTokenContextFilterRegistration()Filter 链最前端重注册 Sa-Token 上下文过滤器，覆盖 ASYNC/ERROR 分发，保证 SSE/WebSocket 场景下 StpUtil 可用
第2层addInterceptors() 中的 SaInterceptorSpring Interceptor登录校验 → clientId 一致性校验 → 客户端访问路径/IP 白名单校验
第3层actuatorBasicAuthFilter()Filter（仅 actuator 路径）对 /actuator/** 做 HTTP Basic Auth 保护

常用代码

// yml 中排除鉴权路径
security:
  excludes:
    - /auth/login
    - /auth/register
    - /public/**

// AllUrlHandler 收集逻辑（关键行）
Map<RequestMappingInfo, HandlerMethod> handlerMethods =
    requestMappingHandlerMapping.getHandlerMethods();
handlerMethods.forEach((info, handlerMethod) -> {
    info.getDirectPaths().stream()
        .map(path -> path.replaceAll("\\{(.*?)\\}", "*"))
        .forEach(urls::add);
});

// SaInterceptor 鉴权链核心
SaRouter.match(allUrlHandler.getUrls())
    .check(r -> {
        StpUtil.checkLogin();  // 1. 登录校验
        // 2. clientId 一致性校验 (header/param vs token)
        // 3. validateClientAccessRules() 路径+IP白名单
    });
