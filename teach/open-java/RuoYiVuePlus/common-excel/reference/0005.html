ExcelBuilder 四个入口

方法用途返回
of(data, headType)标准列表导出ExcelBuilder
writer(headType)自定义写出（多Sheet等）ExcelBuilder
template(templatePath)基于模板填充TemplateBuilder
read(is, clazz)导入读取ReadBuilder

导出常用链式配置

ExcelBuilder.of(list, SysUser.class) .sheetName("用户数据") // 工作表名 .sheetNo(0) // 工作表编号 .merge() // 开启单元格合并 .options(dropDownOptions) // 设置下拉选项 .password("123") // 文件密码 .needHead(true) // 是否写出表头 .automaticMergeHead(true) // 自动合并多级表头 .includeFields(...) / .excludeFields(...) // 字段过滤 .includeIndexes(...) / .excludeIndexes(...) // 列索引过滤 .columnWidth(20) // 固定列宽 .rowHeight((short)20, (short)16) // 表头行高, 内容行高 .registerWriteHandler(handler) // 自定义写处理器 .registerConverter(converter) // 自定义转换器 .zip(1000) // ZIP分页导出 (每1000条一个xlsx) .toResponse(response) // 写到HTTP响应 .toStream(outputStream) // 写到输出流 .toStream(os, wrapper -> {}) // 自定义写出逻辑

导入常用链式配置

ExcelBuilder.read(is, SysUser.class) .validate(true) // 是否JSR303校验 .failFast(false) // 是否快速失败 .listener(customListener) // 自定义监听器 .sheetNo(0) // 工作表编号 .sheetName("sheet1") // 工作表名称 .headRowNumber(2) // 表头行数 .ignoreEmptyRow(true) // 忽略空行 .password("123") // 读取密码 .autoTrim(true) // 自动trim .autoStrip(true) // 自动清理不可见字符 .numRows(500) // 最多读取行数 .registerConverter(converter) // 自定义转换器 .doRead() // 返回 ExcelResult .doReadSync() // 返回 List<T> .doReadAll() // 读取所有Sheet .doReadAllSync() // 读取所有Sheet(同步)

模板导出 TemplateBuilder

ExcelBuilder.template("templates/用户报表.xlsx") .filename("导出文件名") .data(list) // LIST: 单表, 模板 {.属性} .multiList(map) // MULTI_LIST: 多表, 模板 {key.属性} .multiSheet(list) // MULTI_SHEET: 多Sheet .toResponse(resp) / .toStream(os)

ExcelBigNumberConvert 规则

Long → Excel: 值长度 > 15 → 按字符串写入（防精度丢失） 值长度 ≤ 15 → 按数值 BIGDECIMAL 写入 Excel → Long: 一律用 Convert.toLong(cellData.getStringValue())

ExcelWriterWrapper 暴露的方法

write(data, writeSheet) / write(supplier, writeSheet) write(data, writeSheet, writeTable) / write(supplier, writeSheet, writeTable) fill(data, writeSheet) / fill(data, fillConfig, writeSheet) writeContext() // 获取写出上下文 静态: sheetBuilder() / tableBuilder() // 创建 Sheet/Table 未暴露: finish() — 由调用方统一管理

记忆锚点

  • ExcelBuilder 是模块的唯一对外 API，业务代码不应直接使用 Fesod Sheet 的 API。
  • 导出默认自动注册 3 个处理器：ExcelBigNumberConvert + DataWriteHandler + ExcelDownHandler。
  • 导入默认使用 DefaultExcelListener，可通过 .listener() 注入自定义实现。
  • 大数据量导出用 .zip(pageSize) 自动分页打包，避免单个 Excel 过大。
  • 模板导出基于 Fesod Sheet 的 Fill 机制，支持三种模式：单表/多表/多Sheet。
