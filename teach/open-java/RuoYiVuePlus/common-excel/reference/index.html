ruoyi-common-excel 速查表

Maven: org.dromara:ruoyi-common-excel | ruoyi-common-excel/src/main/java/org/dromara/common/excel/

定位

基于 Fesod Sheet（Apache POI 上层封装）的注解驱动 Excel 导入导出模块，提供字典/枚举双向转换、单元格合并、下拉选项、大数字精度保护等企业级高频能力。21 个 Java 文件，对外唯一 API 是 ExcelBuilder。

四层架构

层职责代表文件
注解层声明式标注字段行为@CellMerge, @ExcelDictFormat, @ExcelEnumFormat, @ExcelDynamicOptions, @ExcelRequired, @ExcelNotation
转换器层类型双向转换（Java ↔ Excel）ExcelDictConvert, ExcelEnumConvert, ExcelBigNumberConvert
处理器层生命周期钩子（合并、下拉、批注）CellMergeStrategy, ExcelDownHandler, DataWriteHandler, DefaultExcelListener
工具层对外 DSL 入口ExcelBuilder, ExcelWriterWrapper, CellMergeHandler

术语速查

术语解释
Fesod SheetApache POI 高层封装，提供注解驱动读写、Converter、WriteHandler 等扩展点
ConverterFesod Sheet 类型转换接口，声明 supportJavaTypeKey() + convertToJavaData() + convertToExcelData()
WriteHandler写入生命周期钩子：SheetWriteHandler（Sheet 创建后）、CellWriteHandler（单元格写入后）
AnalysisEventListenerFesod Sheet 读生命周期：invoke() 逐行回调、doAfterAllAnalysed() 全解析完、onException() 异常
@CellMerge标注列合并连续相同值，index 指定列，mergeBy 声明依赖字段
@ExcelDictFormat字典转换：dictType 走字典服务，readConverterExp 走内联表达式（如 0=男,1=女）
@ExcelEnumFormat枚举转换：指定 enumClass + codeField/textField，ConcurrentHashMap 缓存映射
@ExcelDynamicOptions运行时下拉数据源，providerClass 指定 ExcelOptionsProvider 实现类
@ExcelRequired必填列标注，导出模板时将表头字体标红（仅视觉效果）
@ExcelNotation列头批注，在表头单元格添加 POI Comment
DropDownOptions下拉选项 POJO，支持一级/二级级联，buildLinkedOptions() 自动构建

常用代码

列表导出到浏览器

// Controller 中一行搞定：数据列表 + class + response
ExcelBuilder.of(dataList, UserExportVO.class).toResponse(response);

模板导出

// 导出带下拉、必填标注、批注的导入模板
ExcelBuilder.template(UserImportVO.class).toResponse(response);

导入读取

// 读取上传的 Excel 文件，返回 ExcelResult 包含成功/错误回执
ExcelResult<UserImportVO> result = ExcelBuilder
    .read(inputStream, UserImportVO.class)
    .doRead();
List<UserImportVO> successList = result.getAnalysis();

开启合并单元格导出

ExcelBuilder.of(dataList, UserExportVO.class)
    .merge()  // 自动注册 CellMergeStrategy
    .toResponse(response);

实体类标注示例

public class UserExportVO {
    @ExcelProperty("用户编号")
    @ExcelBigNumber  // 避免科学计数法
    private Long id;

    @ExcelProperty("性别")
    @ExcelDictFormat(dictType = "sys_user_gender")  // 导出:编码→标签
    private String sex;

    @ExcelProperty("状态")
    @ExcelEnumFormat(enumClass = UserStatus.class)
    private String status;

    @ExcelProperty("部门")
    @CellMerge(index = 3, mergeBy = {"deptCode"})  // 同部门合并
    private String deptName;
}
