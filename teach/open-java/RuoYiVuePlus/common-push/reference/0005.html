PushHelper API 速查

方法作用跨节点？
sendMessage(userId, message)单发文本消息否（仅本节点）

publishMessage(dto)发布完整 PushDTO是

publishAll(payload)广播 JSON是

Send vs Publish 选择

  • 单节点：send 和 publish 都能用，publish 多一次 Redis 往返。
  • 集群：必须用 publish——不确定目标用户连接在哪个节点上。
  • 经验法则：集群环境无脑用 publish 系列；单节点用 send 稍快。

MessageTopicListener
