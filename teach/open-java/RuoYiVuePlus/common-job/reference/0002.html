速查 02 · SnailJobConfig 自动配置详解

配套课程：lessons/0002.html · 使命：读懂 ruoyi-common-job 定时任务模块

源码结构（43 行）

@AutoConfiguration                           // ① 自动配置
@ConditionalOnProperty(prefix = "snail-job",
    name = "enabled", havingValue = "true")  // ② 开关控制
@EnableScheduling                           // ③ 启用 Spring 调度
@EnableSnailJob                             // ④ 启用 SnailJob 客户端
public class SnailJobConfig {

    @EventListener(SnailClientStartingEvent.class)
    public void onStarting(SnailClientStartingEvent event) {
        LoggerContext lc = (LoggerContext) LoggerFactory.getILoggerFactory();
        SnailLogbackAppender<ILoggingEvent> ca = new SnailLogbackAppender<>();
        ca.setName("snail_log_appender");
        ca.start();
        Logger rootLogger = lc.getLogger(Logger.ROOT_LOGGER_NAME);
        rootLogger.addAppender(ca);             // ⑤ 挂载远程日志 Appender
    }
}

四注解一方法速查

元素源码位置作用不写会怎样
@AutoConfiguration第 21 行声明自动配置类，Spring Boot 自动加载类不被加载，其余注解不生效
@ConditionalOnProperty(...)第 22 行yml 开关控制：snail-job.enabled=true 才生效永远生效，无法按环境开关
@EnableScheduling第 23 行允许使用 Spring @Scheduled不影响 SnailJob，仅影响原生调度
@EnableSnailJob第 24 行触发 SnailJob 客户端全部初始化（连接、扫描、注册）客户端不启动，任务无法被调度
onStarting()第 32-39 行客户端启动时将 SnailLogbackAppender 挂载到 Logback 根 Logger日志不回传 Server，运维看不到任务日志

yml 关键配置

配置项示例值说明
snail-job.enabledtrue / false客户端总开关（对应 @ConditionalOnProperty）
snail-job.group"ruoyi_group"任务分组，需在 Server 后台创建
snail-job.token"SJ_..."接入令牌，Server 验证客户端身份
snail-job.server.host127.0.0.1SnailJob Server IP
snail-job.server.port17888SnailJob Server 端口
snail-job.namespacedev命名空间，隔离不同环境
snail-job.port2${server.port}客户端端口漂移（8080→28080）

启动链路

① Spring 加载 SnailJobConfig → ② 检查 snail-job.enabled → ③ @EnableSnailJob 初始化客户端（读配置、连 Server、注册心跳、扫描 @JobExecutor） → ④ 发布 SnailClientStartingEvent → ⑤ onStarting() 挂载远程 Appender → ⑥ 客户端就绪，等待调度

记忆锚点

  • @EnableSnailJob 是核心——没有它就没有 SnailJob 客户端。
  • @ConditionalOnProperty 是一键开关——开发环境 enabled: false 什么都不会启动。
  • onStarting 挂载 Appender 到根 Logger——所有 log.info() 自动上报 Server。
  • 端口漂移 2${server.port} 是巧妙设计——同一机器多实例不冲突。
