15 · 调试技巧与从零搭建项目骨架

胜利条件：掌握 5 个调试 React 应用的必备技巧；能使用 create-umi 从零搭建一个 UmiJS Max + Ant Design + Zustand 的项目骨架。

Part A：React 调试实战技巧

技巧 1：Chrome DevTools 断点法（最常用）

在 return 之前加 debugger; 是最快的方式：

const handleSubmit = async (values) => {
  debugger;  // ← 执行到这里会暂停，所有变量都可在 Scope 面板查看
  const res = await login(values);
  setToken(res.data.access_token);
};

更优雅的方式：打开 Chrome DevTools → Sources → 找到对应文件 → 点击行号打断点。

推荐：在 request.ts 的请求拦截器中打一个断点。然后做任何操作——你能看到所有 API 请求的参数、token、加密过程。

技巧 2：React DevTools（浏览器扩展）

安装 React Developer Tools 扩展后，F12 会多两个面板：

面板作用追踪什么
⚛️ Components组件树 + props + state选中任意组件 → 右边看它的 props、state、hooks 值——甚至可以实时修改
⚛️ Profiler渲染性能分析录制一次操作，看哪些组件重渲染了几次、花了多少时间
定位 bug 的黄金流程：
1. 在 Components 面板选中可疑组件
2. 在右侧查看 props 和 hooks → 值符合预期吗？
3. 如果 state 是错的 → 找到设置这个 state 的地方（setter），往前追溯
4. 如果 props 是错的 → 往父组件找，谁传的这个 props？

技巧 3：排查"数据不刷新"

React 中最常见的 bug——数据更新了但页面没变。排查清单：

// 检查 1：是不是直接改了 state？（必须用 setter）
user.name = '新名字';  // ❌ React 不知道
setUser({ ...user, name: '新名字' });  // ✅

// 检查 2：是不是直接 push 了数组？（必须返回新数组）
list.push(newItem);      // ❌ 引用没变
setList([...list, newItem]);  // ✅ 新引用

// 检查 3：useEffect 的依赖数组写对了吗？
useEffect(() => {
  fetchData(userId);
}, []);  // ❌ userId 变了不会重新请求
// }, [userId]);  // ✅

// 检查 4：Zustand selector 写对了吗？
useUserStore(state => state.userInfo.nickName);  // 这样写是可以的
useUserStore();  // ← ❌ 任何字段变化都会重渲染（不推荐）

Part B：从零搭建项目骨架

假设你离开 ruoyi-react，要从零开始一个新项目。以下是最小化但可用的企业级 React 项目骨架。

步骤 1：创建 UmiJS Max 项目

# 使用 pnpm 创建项目
pnpm create umi my-admin

# 选择模板：@umijs/max（全功能版）
# 选择包管理器：pnpm

步骤 2：安装核心依赖

# 状态管理
pnpm add zustand

# UI 组件库
pnpm add antd @ant-design/pro-components @ant-design/icons

步骤 4：编写最小 app.tsx

import type { RuntimeConfig } from 'umi';
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';

const queryClient = new QueryClient({
  defaultOptions: {
    queries: {
      staleTime: 10 * 60 * 1000,   // 10 分钟内不重新请求
      retry: 1,
      refetchOnWindowFocus: false,
    },
  },
});

// 应用启动时获取的全局初始数据
export async function getInitialState() {
  const token = localStorage.getItem('token');
  return { token };
}

// 包裹根容器（注入 react-query 等 Provider）
export function rootContainer(container: any) {
  return (
    <QueryClientProvider client={queryClient}>
      {container}
    </QueryClientProvider>
  );
}

步骤 5：编写最小 Zustand store

// src/stores/userStore.ts
import { create } from 'zustand';

interface UserStore {
  userInfo: { name: string; avatar: string } | null;
  setUserInfo: (info: UserStore['userInfo']) => void;
  logout: () => void;
}

export const useUserStore = create<UserStore>((set) => ({
  userInfo: null,
  setUserInfo: (userInfo) => set({ userInfo }),
  logout: () => set({ userInfo: null }),
}));

// src/stores/appStore.ts
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

interface AppStore {
  darkMode: boolean;
  toggleDarkMode: () => void;
}

export const useAppStore = create<AppStore>()(
  persist(
    (set) => ({
      darkMode: false,
      toggleDarkMode: () => set((s) => ({ darkMode: !s.darkMode })),
    }),
    { name: 'app-settings' }
  )
);

步骤 6：编写最小 Axios 封装

// src/services/request.ts
import axios from 'axios';
import { message } from 'antd';
import { useUserStore } from '@/stores/userStore';

const service = axios.create({
  baseURL: '/api',
  timeout: 30000,
});

// 请求拦截器：附加 token
service.interceptors.request.use((config) => {
  const token = localStorage.getItem('token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// 响应拦截器：处理错误
service.interceptors.response.use(
  (response) => response.data,
  (error) => {
    if (error.response?.status === 401) {
      useUserStore.getState().logout();
      localStorage.removeItem('token');
      window.location.href = '/login';
    } else {
      message.error(error.message || '请求失败');
    }
    return Promise.reject(error);
  }
);

export default service;

步骤 7：编写最小登录页

// src/pages/login.tsx
import { ProForm, ProFormText } from '@ant-design/pro-components';
import { message } from 'antd';
import { history } from 'umi';
import axios from 'axios';

const LoginPage = () => {
  const handleLogin = async (values: { username: string; password: string }) => {
    const res = await axios.post('/auth/login', values);
    localStorage.setItem('token', res.data.access_token);
    message.success('登录成功');
    history.push('/');
  };

  return (
    <div style={{ height: '100vh', display: 'flex', alignItems: 'center', justifyContent: 'center' }}>
      <ProForm onFinish={handleLogin}>
        <ProFormText name="username" label="用户名" rules={[{ required: true }]} />
        <ProFormText.Password name="password" label="密码" rules={[{ required: true }]} />
      </ProForm>
    </div>
  );
};
export default LoginPage;
