Hooks 速查

Hook签名触发重渲染典型场景
useStateconst [v, setV] = useState(init)✅ setter 调用时表单输入、弹窗显隐、加载状态
useEffectuseEffect(fn, [deps])❌数据请求、DOM 操作、订阅/清理
useRefconst ref = useRef(init)❌DOM 引用、保存不需要渲染的值
useMemoconst v = useMemo(fn, [deps])❌缓存昂贵计算结果
useCallbackconst fn = useCallback(fn, [deps])❌缓存函数引用（传给 memo 子组件）

useEffect 依赖数组三种形态

形态语法执行时机
无useEffect(fn)每次渲染后
空数组useEffect(fn, [])仅挂载一次（+清理在卸载时）
有依赖useEffect(fn, [a, b])挂载 + a 或 b 变化时
