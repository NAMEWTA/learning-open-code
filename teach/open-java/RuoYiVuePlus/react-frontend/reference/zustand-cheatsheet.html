Zustand 状态管理速查

基础模板

import { create } from 'zustand';

interface XxxStore {
  data: DataType | null;
  setData: (data: DataType) => void;
  reset: () => void;
}

export const useXxxStore = create<XxxStore>((set, get) => ({
  data: null,
  setData: (data) => set({ data }),
  reset: () => set({ data: null }),
}));

带持久化的模板

import { create } from 'zustand';
import { persist } from 'zustand/middleware';

export const useAppStore = create(
  persist(
    (set) => ({
      theme: 'light',
      setTheme: (t) => set({ theme: t }),
    }),
    { name: 'my-app-settings' }  // localStorage key
  )
);

在组件中使用

// ✅ 精准订阅（推荐）
const userInfo = useUserStore(s => s.userInfo);

// ✅ 组件外读写
useUserStore.getState().setUserInfo(data);
const token = useUserStore.getState().token;

// ⚠️ 订阅整个 store（任何字段变都重渲染）
const store = useUserStore();

ruoyi-react 6 个 Store 一览

Store核心字段持久化数据来源
userStoreuserInfo否API getInfo()
permissionStorebackendRoutes, menuData, permissions, reloadMenus()否API getRouters()
appStorelayoutSettings, darkMode, locale✅ persist用户操作
tagsViewStorevisitedViews, cachedViews✅ persist用户操作
dictStoredictCache否（react-query 缓）API getDicts()
noticeStorenotices, unreadCount否SSE/WebSocket
