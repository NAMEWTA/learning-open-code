速查 01 · ruoyi-common-doc 模块全景

配套课程：lessons/0001.html · 使命：读懂 ruoyi-common-doc 接口文档模块

一句话定位

ruoyi-common-doc 不是文档生成框架，而是对 SpringDoc 的增强层——利用扩展点注入中文 JavaDoc 描述、权限注解说明、上下文路径修正三样增强。

模块 8 文件 + 1 清单（4 层架构）

层次文件作用
装配层config/SpringDocConfig.java5 个 @Bean 组装整个文档系统（第 2 课）
装配层...AutoConfiguration.imports声明自动配置类（一行）
配置层config/properties/SpringDocProperties.java映射 springdoc.* yml 段
定制层core/customizer/JavadocOperationCustomizer.java方法 JavaDoc → summary/description
定制层core/customizer/ClassTagOperationCustomizer.java类 JavaDoc → Tag 分组（双接口）
扩展层core/resolver/JavadocResolver.java扩展点接口：supports + resolve
扩展层core/resolver/AbstractMetadataJavadocResolver.java模板方法 + 注解工具
扩展层core/resolver/SaTokenAnnotationMetadataJavadocResolver.javaSaToken 注解 → 权限文档
扩展层core/model/SaTokenSecurityMetadata.java权限模型 + Markdown 输出

5 个 @Bean 速查

Bean 方法返回类型职责
openApi(properties)OpenAPI构建文档根对象（info/tags/components/security）
classTagOperationCustomizer()ClassTagOperationCustomizer双接口定制器：Tag 替换 + 全局清理
javadocOperationCustomizer()JavadocOperationCustomizerJavaDoc 描述 + Resolver 扩展调用

saTokenAnnotationJavadocResolver()JavadocResolver注册 SaToken 权限解析器

核心依赖

  • springdoc-openapi-starter-webmvc-api：文档扫描 + 生成 + UI
  • therapi-runtime-javadoc：JavaDoc 运行时读取
  • jackson-module-kotlin：SpringDoc Kotlin 数据类序列化
  • ruoyi-common-core：StringUtils 等工具
  • sa-token（可选）：有则显示权限文档，无则静默跳过

记忆锚点

  • 模块零行 Controller 扫描代码——文档生成全在 SpringDoc。
  • 不暴露任何端点，是被所有业务模块依赖的增强层。
  • 扩展点体系是精华：JavadocResolver 接口 → List 注入 → supports/resolve 模式，新增权限框架文档只需加一个实现类。
  • PlusPaths 空类解决路径重复——标记类 + instanceof 实现幂等。
  • SaToken 注解用字符串类名加载——可选依赖，不强制引入 sa-token jar。
