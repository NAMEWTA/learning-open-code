ruoyi-common-translation 速查表

Maven: org.dromara:ruoyi-common-translation | 源码: ruoyi-common-translation/src/main/java/org/dromara/common/translation/

定位

注解驱动的字段值翻译引擎（11 个 Java 文件）。在 JSON 序列化管线中把 ID/编码自动换成可读名称——用户 ID 变昵称、部门 ID 变部门名、字典值变标签。核心是 collect → prepare → process 三阶段批量翻译，避免 N+1 查询。

术语速查

术语解释
@Translation字段级注解，标记需要翻译的字段。翻译后的值会替换当前字段的序列化输出
@TranslationType类型级注解，标注在 TranslationInterface 实现类上，声明能处理哪种翻译类型
TranslationInterface翻译 SPI 接口，所有翻译实现必须实现它。推荐覆写 translationBatch() 避免逐条查询
TranslationJsonFieldProcessor核心处理器，实现 JsonFieldProcessor，遵循 collect→prepare→process 三阶段生命周期
JsonFieldProcessorruoyi-common-json 定义的字段处理扩展点：supports() / collect() / prepare() / process()
TransConstant5 个翻译类型字符串常量的集中定义，避免硬编码字符串
mapper 字段映射@Translation 的 mapper 参数机制：不翻译当前字段的值，而是翻译同一对象上 mapper 指定字段的值
批量翻译退化未覆写 translationBatch() 时，default 实现逐条调 translation() 并打警告日志——N+1 问题根源
collectLongIdsTranslationInterface 工具方法，将 Set<Object> 统一转 Set<Long>，支持单值和逗号分隔字符串
SPI 翻译注册策略模式 + Spring 依赖注入：@TranslationType 声明类型 → TranslationConfig 收集实现 → 按 type 建立映射 Map

三阶段生命周期

阶段方法做了什么
collectcollect()递归扫描响应对象树，对每个带 @Translation 的字段收集原始值，按 (type, other) 分组存入 JsonEnhancementContext
prepareprepare()collect 全部完成后执行一次。对每组原始值调用 translationBatch() 批量查询，结果也存入 context
processprocess()渲染 JSON 时对每个字段执行。从 context 取出翻译结果，替换字段值。未匹配到的值保留原值

常用代码

// 1. VO 字段标注翻译
public class UserVO {
    private Long createBy;                          // 原始ID

    @Translation(type = TransConstant.USER_ID_TO_NAME,
                 mapper = "createBy")
    private String createByName;                    // 自动翻译为用户名

    @Translation(type = TransConstant.DICT_TYPE_TO_LABEL,
                 other = "sys_user_sex")
    private String sex;                             // 0→男, 1→女
}

// 2. 自定义翻译实现（标准步骤）
@Component
@TranslationType(type = "order_id_to_summary")
public class OrderSummaryTranslationImpl
        implements TranslationInterface<String> {

    @Override
    public String translation(Object key, String other) {
        // 单值翻译
        return orderService.getSummary((Long) key);
    }

    @Override  // 必须覆写，避免 N+1
    public Map<Object, String> translationBatch(
            Set<Object> keys, String other) {
        Set<Long> ids = collectLongIds(keys);
        return orderService.getSummaryBatch(ids);
    }
}

// 3. 使用自定义翻译
@Translation(type = "order_id_to_summary", mapper = "orderId")
private String orderSummary;
