定位

RuoYi-Vue-Plus 的 Web 层基础设施，覆盖五大能力域：过滤器链（XSS 防护 + 请求体可重复读）、请求包装（Servlet Wrapper）、验证码与国际化、全局异常处理、响应增强。通过 Servlet Filter + Spring Interceptor + RestControllerAdvice 三层武器构建安全、健壮、可观测的 Web 入口。

常用代码

HTTP 请求处理链

RepeatableFilter                     → 缓存请求体 byte[] body，使得后续可重复读
  └─ XssFilter                       → HTML 标签清洗（excludeUrls 跳过，GET/DELETE 放行）
       └─ PlusWebInvokeTimeInterceptor → StopWatch 计时 + 记录 URL/参数（敏感字段脱敏）
            └─ Controller            → 业务处理
                 ├─ GlobalExceptionHandler    → 任何异常兜底，内部 log.error 完整堆栈，对外 R.fail()
                 └─ ResponseEnhancementAdvice → JSON 响应写出前做字段值增强（脱敏/格式化）

XSS 配置

# application.yml
xss:
  enabled: true                # 总开关，false 时 Filter Bean 不存在，零开销
  exclude-urls:
    - /system/notice/*         # 富文本接口跳过过滤

CORS 跨域配置

# application.yml
web:
  cors:
    allow-credentials: true
    allowed-origin-patterns:
      - "https://*.example.com"
    allowed-headers: "*"
    allowed-methods: "*"
    max-age: 3600

BaseController 使用

@RestController
public class UserController extends BaseController {

    @PostMapping("/save")
    public R<Void> save(@RequestBody User user) {
        int rows = userService.save(user);
        return toAjax(rows);   // rows > 0 → R.ok()，否则 R.fail()
    }

    @DeleteMapping("/delete")
    public R<Void> delete(Long id) {
        boolean ok = userService.deleteById(id);
        return toAjax(ok);     // true → R.ok()，false → R.fail()
    }
}

验证码生成流程

GET /captcha → CaptchaConfig 提供 WaveAndCircleCaptcha Bean
  → createImage(): drawString() 彩色文字 + 抗锯齿
                 → shear() X/Y 双向正弦扭曲（防 OCR 切割）
                 → drawInterfere() 圆圈 + 平滑正弦波浪线
  → 输出 PNG 图片流
