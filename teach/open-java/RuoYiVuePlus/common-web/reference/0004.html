WaveAndCircleCaptcha 绘制流程

createImage(code) 1. new BufferedImage(宽, 高, ARGB 或 RGB) 背景 null → TYPE_4BYTE_ABGR(透明) 背景有值 → TYPE_INT_RGB(不透明) 2. drawString(g, code) 抗锯齿 ON → GraphicsUtil.drawStringColourful(每字符随机色) 3. shear(g, w, h, 背景色) shearX: 行级正弦扭曲 + copyArea 平移 + 背景色补空隙 shearY: 列级正弦扭曲 4. drawInterfere(g) 圆圈: N-1 个随机位置/大小/颜色的 drawOval 波浪: 1 条平滑正弦曲线 (逐像素采样 → drawPolyline)

I18nLocaleResolver

方法行为
resolveLocale(req)读 content-language 头 → Locale.forLanguageTag() → 解析失败回退 Locale.getDefault()
setLocale(...)空实现 —— 语言切换由前端控制

I18nConfig 的 before 技巧

@AutoConfiguration(before = WebMvcAutoConfiguration.class) → I18nConfig 先注册 I18nLocaleResolver → WebMvcAutoConfiguration 检测已有 LocaleResolver → 不再注册默认 AcceptHeaderLocaleResolver → "抢占式替换" 完成

记忆锚点

  • 四步绘制：BufferedImage → 彩色文字(抗锯齿) → X/Y正弦扭曲 → 圆圈+波浪线
  • shearX 逐行平移：每行按 sin(y/period) 偏移，copyArea 移动 + drawLine 补空隙
  • drawPolyline 画波浪：逐像素计算 y = baseY + amplitude * sin(x/wavelength * 2PI + phase)
  • 透明背景：null 用 TYPE_4BYTE_ABGR，有背景色用 TYPE_INT_RGB
  • content-language 头：自定义头，避免与浏览器 Accept-Language 冲突
  • before 占位：I18nConfig 在 WebMvcAutoConfiguration 前注册，抢默认 Bean 的位置
