速查 02 · 工具类上篇

配套课程：lessons/0002.html · StringUtils、DateUtils、StreamUtils、MapstructUtils

一句话定位

四个核心数据处理工具：StringUtils(继承Apache+Lombok委托Hutool)、DateUtils(继承Hutool+友好时间)、StreamUtils(消除Stream样板)、MapstructUtils(零Mapper接口的对象转换)。

StringUtils — 继承 Apache + 委托 Hutool

类别方法示例/效果
命名转换toUnderScoreCase()"helloWorld" → "hello_world"
命名转换toCamelCase()"user_name" → "userName"
切分str2Set("A,B", ",")→ Set{"A","B"}
切分+转换splitTo("1,2", Long::parseLong)→ List[1L,2L]
格式化format("用户{}不存在", id)→ "用户123不存在"
URL匹配matches("/admin/**", path)AntPathMatcher 模式匹配

DateUtils — 继承 Hutool DateUtil

formatFriendlyTime(date): 1分钟内 → "刚刚" 1小时内 → "5分钟前" 今天 → "下午 14:30" 昨天 → "昨天 08:45" 本周 → "周三 10:00" 今年内 → "06-15 09:30" 非今年 → "2025-06-15 09:30" formatTimeBetween(start, end, SECOND) → "1天2小时3分4秒" validateDateRange(start, end, 30, DAYS) → 跨度超30天抛异常

StreamUtils — 消除 Stream 样板

方法等效原生 Stream
filter(list, pred)stream().filter().collect(toList)
toList(list, mapper)stream().map().collect(toList)
toIdentityMap(list, keyFn)collect(toMap, identity)
groupByKey(list, keyFn)collect(groupingBy)
groupBy2Key(list, k1, k2)嵌套 groupingBy
join(list, fn, sep)map().collect(joining)

MapstructUtils — MapStruct Plus 封装

底层: private static Converter CONVERTER = SpringUtils.getBean(Converter.class); convert(user, UserVo.class) → 对象 → 新对象 convert(user, existingVo) → 填充已有对象 convert(userList, UserVo.class) → 批量转换 convert(paramMap, User.class) → Map → Bean 对比传统 MapStruct: 不需要写 @Mapper 接口！自动字段名匹配。

记忆锚点

  • StringUtils 三合一：extends Apache StringUtils + 委托 Hutool StrUtil + 自定义业务方法。
  • DateUtils.formatFriendlyTime() — 6 条 if-else 规则链覆盖全部时间跨度。
  • 不要用 Stream.toList() — 返回不可变 List，会有序列化问题。
  • MapstructUtils 不用写 Mapper 接口 — MapStruct Plus 的 Converter 运行时自动匹配。
