定位

RuoYi-Vue-Plus 项目的基石模块。所有其他 ruoyi-common-* 模块都直接或间接依赖它。提供四大支柱：统一响应体（R/PageResult）、分层异常体系（BaseException/ServiceException）、常量与枚举、工具类集合（18+）。

GLOBAL_REDIS_KEY框架级 Redis key 前缀 "global:"，与多租户和业务 key 隔离
CacheNames缓存组名称常量接口，格式编码缓存策略，如 SYS_USER_NAME = "sys_user_name#30d"
HttpStatus本项目的 HTTP 状态码常量接口（非 Spring 的），含自定义 WARN=601
@DictPattern字典值校验注解，底层通过 DictService 查询字典数据做校验
@EnumPattern枚举值校验注解，反射调用枚举 getter 提取合法值集合
@XssXSS 防护注解，基于 Hutool HtmlUtil.RE_HTML_MARK 正则校验
SpringUtilsSpring 容器工具（继承 Hutool SpringUtil），支持虚拟线程检测 isVirtual()
MessageUtils国际化消息工具，找不到 key 时优雅降级返回原 code
MapstructUtilsMapStruct Plus 转换工具，通过静态 CONVERTER 字段获取单例
ValidatorUtils参数校验工具，支持分组校验，校验失败直接抛 ConstraintViolationException
常量接口模式core 中所有常量使用 interface 定义（隐式 public static final），避免冗余修饰符
Fail-Fast 校验ValidatorConfig 中 fail_fast=true，遇到第一个校验失败即返回

常用代码

Controller 返回统一响应

// 成功
return R.ok(data);
return R.ok();          // 无数据
return R.ok("操作成功");

// 失败
return R.fail("参数错误");
return R.fail(HttpStatus.BAD_REQUEST, "请求参数有误");

// 警告（业务层面可接受但需提示）
return R.warn("余额不足，已使用默认额度");

抛出 ServiceException

// 简单消息
throw new ServiceException("用户不存在");

// 带占位符格式化
throw new ServiceException("用户【{}】不存在", userId);

// 带错误码
throw new ServiceException(500, "系统内部错误");

自定义校验注解

// 定义注解
@Target({ElementType.FIELD})
@Retention(RetentionPolicy.RUNTIME)
@Constraint(validatedBy = {XssValidator.class})
public @interface Xss {
    String message() default "不允许包含HTML标签";
    Class<?>[] groups() default {};
    Class<? extends Payload>[] payload() default {};
}

// 使用
public class UserBody {
    @Xss
    private String nickname;
}
