审批弹窗打开流程

  1. openDialog(taskId) - 调 getTask() 查详情+按钮权限
  2. 调 getNextNodeList() 获取下一审批节点
  3. 遍历 buttonList 设置按钮显隐
  4. 用户填 消息类型/抄送人/审批意见/下一步选人
  5. 点提交→completeTask() / backProcess() / taskOperation()

设计器 iframe 嵌入

/warm-flow-ui/index.html?id={definitionId}&onlyDesignShow=true + Authorization=Bearer {token} + clientid={clientId}

业务页面接入模式

  1. 决定 flowCode
  2. 组装 variables（业务数据）
  3. 组装 bizExt（businessTitle/businessCode）
  4. 调 startWorkFlow() → 拿 taskId
  5. 调 submitVerifyRef.openDialog(taskId) 弹审批
