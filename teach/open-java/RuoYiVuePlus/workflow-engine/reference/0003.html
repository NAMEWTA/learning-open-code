StartProcessBo 字段

字段类型必填含义
businessIdString✅业务唯一 ID，与实例一对一
flowCodeString✅流程定义编码（如 "leave1"）
handlerString❌覆盖首节点办理人
variablesMap❌业务变量（影响条件/办理人）
bizExtFlowInstanceBizExt❌实例扩展（标题/编号）

自动注入变量

initiator（当前用户ID）
initiatorDeptId（当前用户部门）
businessId（入参 businessId）
autoPass（来自 definition.ext 的 JSON 字段）

执行步骤

  1. 校验 businessId 不为空
  2. 注入 initiator/deptId/businessId 到 variables
  3. 查是否已有同 businessId 的实例 → 有则 checkStartStatus 并复用
  4. 查流程定义（defService.getPublishByFlowCode）→ 未发布则抛异常
  5. 解析 autoPass 扩展属性
  6. 生成 businessCode（TODO：按业务规则生成）
  7. FlowParams + insService.start(businessId, flowParams) → 实例状态 DRAFT
  8. 保存 bizExt 到 flw_instance_biz_ext
  9. 获取首任务 → 必须恰好一个 → 返回 StartProcessReturnDTO
