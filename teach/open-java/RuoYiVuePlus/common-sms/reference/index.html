ruoyi-common-sms 速查表

Maven: org.dromara:ruoyi-common-sms | 源码: ruoyi-common-sms/src/main/java/org/dromara/common/sms/

定位

对开源短信聚合框架 sms4j 的「项目化适配」——仅 3 个 Java 类，实现 Redis 统一缓存替换、全局异常兜底、Spring Boot 自动配置无侵入接入。

核心文件

类名路径职责
SmsAutoConfigurationconfig/SmsAutoConfiguration.java自动配置入口，@AutoConfiguration(after=DataRedisAutoConfiguration) + @Primary 让 PlusSmsDao 压过 sms4j 默认实现
PlusSmsDaocore/dao/PlusSmsDao.java实现 sms4j 的 SmsDao 接口，所有存取转发给 RedisUtils，key 加 global: 前缀，替代默认 ConcurrentHashMap 实现，使集群下防刷生效
SmsExceptionHandlerhandler/SmsExceptionHandler.java@RestControllerAdvice，捕获 SmsBlendException，内部 log.error 记详情，对外返回统一 R.fail("短信发送失败，请稍后再试...")

术语速查

术语解释
sms4jDromara 开源短信聚合框架（v3.3.5），封装阿里云/腾讯云/华为云等多厂商 SDK，提供统一 API
SmsDaosms4j 缓存接口，声明 set/get/remove/clean 方法，用于短信拦截计数和重试状态存取
PlusSmsDaoRuoYi 对 SmsDao 的 Redis 实现，使短信拦截计数使用统一 Redis 缓存，替代默认单机内存
SmsBlendsms4j 发送器接口，每个厂商实现它。通过 SmsFactory.getSmsBlend("config1") 获取
SmsFactorysms4j 工厂入口，getSmsBlend(configId) 按配置 ID 从 yml blends 段找对应厂商实例
SmsResponse发送返回对象：success（是否成功）、data（厂商原始响应）、configId（配置标识）
SmsBlendExceptionsms4j 统一包装的运行时异常，含 code、message、requestId
SmsExceptionHandler全局异常处理器，精确匹配 SmsBlendException，"内外有别"：对内记完整堆栈，对外返回友好提示
CAPTCHA_CODE_KEY验证码 Redis key 前缀 global:captcha_codes:，完整 key 为 global:captcha_codes:{手机号}，5 分钟过期
SmsAuthStrategy短信登录策略（ruoyi-framework），校验验证码 + 查用户 + 签发 token
@RateLimiterRuoYi 限流注解（ruoyi-common-redis），基于 Redis+Lua，如 @RateLimiter(key="#phoneNumber", time=60, count=1)
config1/config2yml 中 sms.blends 下的自定义配置 ID，可实现多厂商共存或同厂商多账号租户隔离

// 发送短信验证码（CaptchaController.smsCode()）核心链路
@RateLimiter(key = "#phoneNumber", time = 60, count = 1)
public R<Void> smsCode(@NotBlank String phoneNumber) {
    // 1. 生成4位验证码
    String code = RandomUtil.randomNumbers(4);
    // 2. 存入 Redis，5分钟过期
    RedisUtils.setCacheObject(
        GlobalConstants.CAPTCHA_CODE_KEY + phoneNumber,
        code, Duration.ofMinutes(5));
    // 3. 通过 sms4j 发送
    SmsBlend smsBlend = SmsFactory.getSmsBlend("config1");
    SmsResponse response = smsBlend.sendMessage(
        phoneNumber, templateId, Map.of("code", code));
    return R.ok();
}

// 短信登录校验验证码（SmsAuthStrategy.validateSmsCode()）
String cacheCode = RedisUtils.getCacheObject(
    GlobalConstants.CAPTCHA_CODE_KEY + phoneNumber);
if (!code.equals(cacheCode)) {
    throw new ServiceException("验证码错误");
}
