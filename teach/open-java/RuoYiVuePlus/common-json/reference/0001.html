速查 01 · ruoyi-common-json 模块全景

配套课程：lessons/0001.html · 使命：全面读懂 ruoyi-common-json 序列化模块

一句话定位

它是 Jackson 3.x (tools.jackson) 的业务定制层：把 Jackson 的序列化/反序列化能力
定制为 RuoYi 需要的「精度保护 / 多格式日期 / 字段增强 / JSON 校验」等开箱能力。

5 大区域 + 13 个类

区域职责关键类
config 配置启动装配，定全局规则JacksonConfig（注册 7 个处理器）、JsonEnhancementConfig（装配增强器）
handler 处理器自定义序列化/反序列化逻辑BigNumberSerializer（Long 精度）、CustomDateDeserializer、CustomLocalDateTimeDeserializer
enhance 增强框架响应出站前字段加工JsonEnhancementContext、JsonFieldContext、JsonFieldProcessor、JsonValueEnhancer
validate 校验Bean Validation 校验 JSON@JsonPattern、JsonType、JsonPatternValidator
utils 工具业务最常用入口JsonUtils（10 个公开方法）

三驾马车

类一句话
JacksonConfig序列化规则注册中心（SimpleModule + JsonMapperBuilderCustomizer）
JsonEnhancementConfig字段增强装配入口（收集所有 Processor → 创建 JsonValueEnhancer）
JsonUtils业务工具门面（toJsonString / parseObject / isJson 等 10 个方法）

JacksonConfig 注册清单

// 序列化器
Long.class / Long.TYPE / BigInteger.class  → BigNumberSerializer.INSTANCE
BigDecimal.class                            → ToStringSerializer.instance
LocalDateTime.class                         → LocalDateTimeSerializer(yyyy-MM-dd HH:mm:ss)

// 反序列化器
LocalDateTime.class → CustomLocalDateTimeDeserializer
Date.class          → CustomDateDeserializer

模块依赖关系

ruoyi-common-core (底层工具) ↑ ruoyi-common-json ← 当前模块（基础能力层） ↑ ruoyi-common-redis / ruoyi-common-satoken / ruoyi-common-sms / 各业务模块

易混点

  • tools.jackson（re-shade 包名）≠ com.fasterxml.jackson（原始包名）——前者是 Maven Shade 重打包版，避免版本冲突。
  • 模块不直接创建 ObjectMapper，通过 Spring 自动装配的 JsonMapper（Jackson 3.x 不可变构建器）。
  • @AutoConfiguration(before = JacksonAutoConfiguration.class) 保证自定义 Module 在 Spring Boot 创建 Mapper 前注册。

→ 相关术语见 GLOSSARY.md：Jackson (tools.jackson)、JsonMapper、SimpleModule、JsonUtils
