自定义范式（两件套）

文件 1: Auth{Platform}Source.java (enum implements AuthSource) ├── authorize() → OAuth 授权端点 URL ├── accessToken() → Token 端点 URL ├── userInfo() → 用户信息端点 URL └── getTargetClass() → return Auth{Platform}Request.class 文件 2: Auth{Platform}Request.java (extends AuthDefaultRequest) ├── 构造器(带/不带 AuthStateCache) ├── getAccessToken(callback) → 覆写：code 换 token ├── getUserInfo(token) → 覆写：token 换用户信息 └── 可选覆写：doPostAuthorizationCode(), doGetUserInfo(), authorize()

接入新平台四步走

① 创建 AuthNewPlatformSource.java → 定义端点 URL ② 创建 AuthNewPlatformRequest.java → 覆写 getAccessToken / getUserInfo ③ SocialUtils.getAuthRequest() → 加 case "newplatform" ④ yml justauth.type.newplatform → 配 clientId + clientSecret + redirectUri
