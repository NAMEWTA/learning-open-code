ruoyi-common-social 速查表

Maven: org.dromara:ruoyi-common-social | 源码: ruoyi-common-social/src/main/java/org/dromara/common/social/

定位

基于开源 OAuth 聚合框架 JustAuth 的社交登录模块（13 个 Java 文件）。用 Redis 缓存授权状态替代默认内存，通过 SocialUtils 工厂方法统一 27 个平台（含 3 个自研 Gitea/MaxKey/TopIAM）的调用入口。

术语速查

术语解释
JustAuthDromara OAuth 聚合框架，封装 20+ 平台 OAuth 流程，提供统一的 AuthRequest/AuthSource/AuthToken/AuthUser 抽象
AuthRequestJustAuth 标准授权请求接口：authorize(state) 生成授权地址、login(callback) 执行登录回调
AuthDefaultRequestJustAuth 抽象基类，封装 OAuth 标准流程通用逻辑。自定义平台只需覆写 getAccessToken() 和 getUserInfo()
AuthSourceOAuth 平台标识接口，声明三个端点 URL：authorize() / accessToken() / userInfo()
AuthStateCacheJustAuth 授权状态缓存接口，临时存储 state 参数防 CSRF。默认 ConcurrentHashMap 内存实现
AuthRedisStateCacheRuoYi 对 AuthStateCache 的 Redis 实现，key 前缀 social_auth_code:，集群下 state 不丢失
AuthCallbackJustAuth 回调参数对象，含 code（授权码）、state（CSRF 校验）、auth_code
AuthTokenJustAuth 用户令牌对象：accessToken / refreshToken / expireIn 等标准 OAuth 字段
AuthUserJustAuth 用户信息对象：uuid / username / nickname / avatar / rawUserInfo
SOCIAL_AUTH_CODE_KEYRedis key 前缀 "social_auth_code:"，AuthRedisStateCache 所有存取以此为前缀

OAuth 登录四阶段

阶段关键动作关键类
1. 构建授权地址前端跳转到平台授权页AuthRequest.authorize(state)
2. 用户授权回调平台回调带回 code + state，state 通过 AuthRedisStateCache 校验AuthCallback + AuthRedisStateCache
3. code 换 token用 code 向平台换取 accessTokengetAccessToken(callback) → AuthToken
4. token 换用户信息用 accessToken 获取用户资料getUserInfo(token) → AuthUser

// SocialUtils.loginAuth() 核心调用链
public static AuthUser loginAuth(String source, String code,
                                  String state, SocialProperties props) {
    AuthRequest authRequest = getAuthRequest(source, props);
    // switch(source) 匹配平台 → new AuthXxxRequest(config)
    AuthCallback callback = AuthCallback.builder()
        .code(code).state(state).build();
    return authRequest.login(callback);
    //  → login() 内部: getAccessToken() → getUserInfo()
}

// 自定义提供者 3 件套标准范式
// ① AuthSource 枚举
public enum AuthGiteaSource implements AuthSource {
    GITEA {
        public String authorize() { return serverUrl + "/login/oauth/authorize"; }
        public String accessToken() { return serverUrl + "/login/oauth/access_token"; }
        public String userInfo() { return serverUrl + "/api/v1/user"; }
    };
}
