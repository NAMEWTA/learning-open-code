ruoyi-common-mcp 速查表

Maven: org.dromara:ruoyi-common-mcp | ruoyi-common-mcp/src/main/java/org/dromara/common/mcp/

定位

对 Spring AI MCP 2.0.0（Model Context Protocol）的项目化封装模块。仅 4 个 Java 类，核心是 McpClientTemplate —— 封装对 MCP Server 的工具调用（Tools）和资源读取（Resources）操作，支持按 Server 名称精确调用或广播到所有 Server。

核心文件

类名路径职责
McpClientTemplatecore/McpClientTemplate.java核心模板类，封装 List<McpSyncClient> 遍历，提供 listTools()/callTool()/readResource()/findClient()
McpAutoConfigurationconfig/McpAutoConfiguration.java自动配置，@ConditionalOnBean(McpSyncClient.class) 确保有连接时才创建 Bean
McpToolCallResultcore/McpToolCallResult.java工具调用结果 Record：serverName + error + content + structuredContent
McpResourceReadResultcore/McpResourceReadResult.java资源读取结果 Record：serverName + contents (List<ResourceContents>)

术语速查

术语解释
MCPAnthropic 提出已演化为行业标准的开放协议，定义 AI 与外部数据源/工具的 Client-Server 通信规范
Spring AI MCPSpring AI 项目的 MCP Java 实现 2.0.0，封装 SSE 传输、JSON-RPC、Lifecycle 管理
McpSyncClientSpring AI 同步客户端接口，一个实例 = 一个 MCP Server 的已初始化连接
McpClientTemplate本模块模板类，封装多 Server 遍历，业务代码通过它间接使用 MCP
Tool（工具）MCP 原语之一：Client 调用 Server 暴露的函数，有参数有返回，语义类似 RPC
Resource（资源）MCP 原语之一：Client 读取 Server 暴露的数据，语义类似 REST GET
CallToolRequest / CallToolResultSpring AI 工具调用请求(toolName+arguments) / 响应(content+isError+structuredContent)
ReadResourceRequest / ReadResourceResultSpring AI 资源读取请求(uri) / 响应(List<ResourceContents>)
@ConditionalOnBean仅当容器中存在 McpSyncClient Bean 时才创建 McpClientTemplate

常用代码

配置文件（application.yml）

spring:
  ai:
    mcp:
      client:
        connections:
          weather-server:           # Server 名称（自定义）
            type: SYNC
            transport: SSE
            sse-endpoint: http://localhost:8081/sse
          file-server:
            type: SYNC
            transport: SSE
            sse-endpoint: http://localhost:8082/sse

列出所有工具

// 列出所有 Server 的工具
List<Tool> allTools = mcpClientTemplate.listTools();

// 列出指定 Server 的工具
List<Tool> weatherTools = mcpClientTemplate.listTools("weather-server");

调用工具（指定 Server）

// 精确调用指定 Server 的工具
Map<String, Object> args = Map.of("city", "北京");
McpToolCallResult result = mcpClientTemplate.callTool(
    "weather-server",              // Server 名称
    "getWeather",                  // 工具名称
    args                           // 参数 Map
);

if (result.error()) {
    // 处理错误
} else {
    // 获取文本内容列表
    List<String> texts = result.content();
    // 获取结构化数据
    Object data = result.structuredContent();
}

调用工具（广播到所有 Server）

// 不指定 serverName，广播到所有已连接的 Server
List<McpToolCallResult> results = mcpClientTemplate.callTool(
    "ping",                        // 工具名称
    Map.of()                       // 空参数
);
// 遍历每个 Server 的返回结果
results.forEach(r ->
    System.out.println(r.serverName() + ": " + r.content()));

读取资源

// 读取指定 Server 的资源
McpResourceReadResult result = mcpClientTemplate.readResource(
    "file-server",                 // Server 名称
    "file:///data/report.txt"      // 资源 URI
);
// contents() 返回 List<ResourceContents>，可能是文本或二进制
result.contents().forEach(content -> {
    if (content instanceof TextResourceContents text) {
        System.out.println(text.text());
    }
});

查找客户端

// 按 Server 名称查找 McpSyncClient
McpSyncClient client = mcpClientTemplate.findClient("weather-server");
// 获取 Server 信息
ServerInfo info = client.getServerInfo();
System.out.println(info.name() + " v" + info.version());
