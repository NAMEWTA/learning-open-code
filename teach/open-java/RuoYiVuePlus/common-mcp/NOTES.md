# NOTES — 教学偏好暂存

> 记录用户在课程中表达的教学偏好，供后续课程设计参考。

- 学习者：全栈背景（Java 后端 + Vue/React 前端）。
- 本主题（ruoyi-common-mcp）为纯后端 Java + MCP 协议，讲解聚焦后端 Java 调用 MCP Server。
- 目标：全面读懂模块整体设计、MCP 协议的 Client-Server 模型、McpClientTemplate 的封装思路与使用方式，不要求改 Spring AI 框架。
- 课程风格：以追踪真实代码 + 解释设计动机为主；练习偏「读代码答问题 / 复述调用链路 / 设计判断」。
- 语言：简体中文。
- 偏好「鸟瞰图先行」风格（先全景图再拆解细节 + 提取练习）。
- MCP 协议本身较新，课程需兼顾「协议是什么」的简介与「Java 代码怎么用」的实操。
