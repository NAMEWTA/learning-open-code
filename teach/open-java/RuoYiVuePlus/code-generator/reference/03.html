isBooleanColumn() 优先于数值类型判断：tinyint(1) + 字段名匹配开关类 → javaType=Boolean, htmlType=switch。匹配规则：endsWith(status/flag/enabled/disabled/available/visible) 或 startsWith(is_/has_/enable_/disable_)。

查询方式自动推断

条件queryType
endsWith(columnName, "name")LIKE（模糊查询）
htmlType == "datetime" && isQuery()BETWEEN（日期范围）
其他所有情况EQ（默认，精确匹配）

参与场景黑名单（默认全部参与，以下字段排除）

场景排除字段额外排除
isInsert（新增）create_dept, create_by, create_time, del_flag, update_by, update_time, version+ 主键字段
isEdit（编辑）create_dept, create_by, create_time, del_flag, update_by, update_time, version—
isList（列表）create_dept, create_by, create_time, del_flag, update_by, update_time, version—
isQuery（查询）id, create_dept, create_by, create_time, del_flag, update_by, update_time, remark, version—

GenTableColumn 辅助方法速查

方法作用模板用法
isPk()是否主键<#if column.pk>
isInsert()是否参与新增控制 BO/Domain/VO 字段可见性
isEdit()是否参与编辑同上
isList()是否参与列表控制 VO 和表格列
isQuery()是否参与查询控制 BO 查询条件

isDictColumn()是否字典控件列决定是否生成字典翻译代码
isUsableColumn()是否页面白名单parentId/orderNum/remark 即使 isSuperColumn 也在页面保留
