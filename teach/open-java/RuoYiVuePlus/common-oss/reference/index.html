定位

基于 AWS SDK S3 v2 的 S3 兼容对象存储模块（18 个 Java 文件）。用模板方法模式封装异步上传下载、工厂模式管理多客户端实例、预签名 URL 生成临时凭证，一套代码同时支持阿里云 OSS / 腾讯云 COS / 七牛云 / 华为云 OBS / MinIO。

术语速查

术语解释
S3 兼容协议Amazon S3 定义的 RESTful 对象存储协议，阿里云/腾讯云/七牛/华为云/MinIO 均兼容
OssClient本模块核心接口，抽象了所有对象存储操作，继承 AutoCloseable
模板方法模式AbstractOssClientImpl 实现全部上传/下载逻辑，子类只需实现 doInitialize() 创建 SDK 组件
OssFactory静态工厂+缓存管理器，从 Redis 读取配置，双重检查锁创建/复用客户端
Path-Style / Site-StyleS3 两种 Bucket URL：/{bucket}/key vs {bucket}.endpoint/key，根据厂商关键词自动推断
预签名 URL临时授权访问 URL（presignGetUrl/presignPutUrl），前端可直接上传/下载，无需经过后端
S3TransferManagerAWS SDK 高级传输管理器，封装分段上传/下载/重试逻辑
AccessPolicyBucket 预制访问策略，每个枚举值携带对应的 BucketCannedACL 和 ObjectCannedACL

常用代码

上传文件：

// 获取默认客户端
OssClient client = OssFactory.instance();

// 上传 File
HandleAsyncResult<PutObjectResult> result = client.upload("images/avatar.png", new File("/tmp/avatar.png"));

// 上传 InputStream
HandleAsyncResult<PutObjectResult> result2 = client.upload("data/report.pdf", inputStream);

获取预签名 URL（前端直传）：

// 生成上传预签名 URL（前端可直接 PUT 文件）
String uploadUrl = client.presignPutUrl("uploads/photo.jpg", Duration.ofMinutes(10));

// 生成下载预签名 URL
String downloadUrl = client.presignGetUrl("reports/2024.pdf", Duration.ofHours(1));
