三类故障排查清单

❌ 权限不生效（应过滤未过滤）

  1. Mapper 方法无 @DataPermission 注解
  2. 当前用户是超级管理员（SUPER_ADMIN_USER_ID）
  3. 角色 dataScope=1（全部数据）
  4. MyBatis-Plus ignore 标记
  5. 方法走了缓存绕过了 MyBatis 拦截器

⚠️ 权限越权（能看不该看的数据）

  1. 角色 dataScope 配错（如 1 而非 3）
  2. 多角色权限叠加（一个角色 ALL=短路）
  3. 接口无 @SaCheckPermission 约束（全部角色参与）
  4. deptIds 未清理（改了 dataScope 但残留旧 deptIds）

💥 SQL 执行报错

  1. 多表 join 列名冲突（忘了表别名如 d.dept_id）
  2. key/value 数组长度不匹配 → ServiceException
  3. SpEL 表达式语法错误
  4. sdss 方法返回 null 而非 "-1"
  5. 部门树为空时生成 IN () 空参数
