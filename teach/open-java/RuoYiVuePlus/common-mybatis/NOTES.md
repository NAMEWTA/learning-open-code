# 教学偏好 NOTES

- 学习者：全栈背景（Java 后端 + Vue/React 前端）。
- 本主题（ruoyi-common-mybatis）为纯后端 Java 持久层模块。
- 目标：全面读懂模块整体设计、各组件职责与设计动机，并能在业务中正确选用，不要求改框架。
- 课程风格：以追踪真实代码 + 解释设计动机为主；练习偏「读代码答问题 / 选型判断 / 复述链路」。
- 语言：简体中文。
- 特别注意：本模块是全项目最核心、Java 文件最多（34 个）的模块。8 节课的重点分配：
  - 第 1-2 课：建立全景认知 + Entity/Mapper 基类基础
  - 第 3-4 课：深入数据权限体系（最复杂的一块，占比最重）
  - 第 5-6 课：查询构建器 + 分页聚合（开发者最常用的 API）
  - 第 7 课：多数据源 + SQL 日志（运维支撑）
  - 第 8 课：实战整合，场景驱动
