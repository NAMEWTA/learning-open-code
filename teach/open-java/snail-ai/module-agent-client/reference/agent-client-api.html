Agent Client 与 OpenAPI SDK API

L3 微观 API 参考：按子模块列出 snail-ai-agent-chat、snail-ai-agent-executor、snail-ai-agent-openapi、snail-ai-agent-common 与示例配置的公开类型、方法签名、参数、返回值与异常。架构地图见 执行层速查。

0. 启用注解与 Maven 模块

注解模块导入配置用途
@EnableSnailAiAgent snail-ai-agent-executor-starter @Import(SnailAiAgentAutoConfiguration.class) 启用 Agent Client 执行层：gRPC server、心跳、ChatClient 工厂、Advisor、Tool resolver。
@EnableSnailAiOpenApi snail-ai-openapi-starter @Import(SnailAiOpenApiAutoConfiguration.class) 启用 OpenAPI HTTP 客户端动态代理（OpenApi*Client Bean）。

嵌入式 Chat UI 依赖 OpenAPI：SnailAiChatAutoConfiguration 在 snail-ai.openapi.enabled=true 时生效，并 @AutoConfigureAfter(SnailAiOpenApiAutoConfiguration.class)。

Maven 子模块职责
snail-ai-agent-chat-api嵌入式 Chat 会话模型、配置属性、认证注解。
snail-ai-agent-chat-starterChat BFF 网关、JWT 拦截器、静态页转发。
snail-ai-agent-executor-coreChatClient 构建、Interceptor、内置 Tool、执行请求模型。
snail-ai-agent-executor-modelModelConfig → ChatModel 工厂。
snail-ai-agent-executor-starter@EnableSnailAiAgent 与自动配置。
snail-ai-openapi-coreOpenAPI 客户端接口与参数注解。
snail-ai-openapi-starter@EnableSnailAiOpenApi 与 HTTP 客户端 Bean。
snail-ai-agent-common聊天上下文、滑动窗口、回调异常、RPC 注解。

4.3 回调异常层次

异常构造器语义
CallbackException(String message)、(String, Throwable)Client 回调 Server 的基础异常，继承 SnailAiException。
CallbackChannelUnavailableException(String message)gRPC Channel 不可用；可重试。
CallbackTimeoutException(String message, Throwable cause)回调超时；可重试。
CallbackServerErrorException(String message)Server 业务错误；不可重试。

4.4 RPC 回调注解

7. 边界条件

  • Chat 网关依赖 OpenAPI 已启用；未配置 snail-ai.openapi.enabled=true 时 Chat starter 不会装配。
  • SnailAiChatSessionContext.current() 与 AgentChatContextHolder 分别服务 HTTP 会话与 gRPC Chat 执行，不要混用。
  • HttpTool 拒绝内网地址；ShellTool 受工作目录与超时约束。
  • 回调异常中仅 CallbackChannelUnavailableException 与 CallbackTimeoutException 语义上可重试。
  • 优先以源码与 snail-ai-agent-example 为准；官方文档与自动配置差异见 执行层速查 第 4 节。
