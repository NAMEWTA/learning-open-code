持久化 PO/Mapper 与存储 API 速查

一句话：本页按领域列出业务库 PO 字段、Mapper 方法签名，以及向量/搜索模板的 DTO 与异常类型；模块边界与 Factory 路由见 persistence-overview.html。

模块职责运行时依赖

snail-ai-mysql-storageMySQL 方言 XMLmysql-connector-j
snail-ai-postgres-storagePostgreSQL 方言 XMLpostgresql

枚举值MyBatis DbType检测方式

POSTGRESQLPOSTGRE_SQL
DMDM

Mapper 通用约定：除下文标注的自定义方法外，所有 *Mapper extends BaseMapper<*PO> 均继承 MyBatis-Plus 标准 CRUD（insert、deleteById、updateById、selectById、selectList、selectPage 等）。复杂查询在 Service 层用 LambdaQueryWrapper 组装。
