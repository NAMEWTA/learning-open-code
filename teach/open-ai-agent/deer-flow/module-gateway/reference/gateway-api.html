调用示例

1. 健康检查（无需认证）

curl -s http://localhost:8001/health

2. 登录并列出模型

curl -i -c cookies.txt -X POST http://localhost:8001/api/v1/auth/login/local \
  -H "Content-Type: application/x-www-form-urlencoded" \
  --data "username=admin@example.com&password=your-password"

curl -b cookies.txt http://localhost:8001/api/models

3. 创建 thread 并流式 run（需 CSRF）

CSRF=$(grep csrf_token cookies.txt | awk '{print $7}')
THREAD_ID=$(curl -s -b cookies.txt -X POST http://localhost:8001/api/threads \
  -H "Content-Type: application/json" \
  -H "X-CSRF-Token: $CSRF" \
  -d '{}' | python3 -c "import sys,json; print(json.load(sys.stdin)['thread_id'])")

curl -N -b cookies.txt -X POST "http://localhost:8001/api/threads/${THREAD_ID}/runs/stream" \
  -H "Content-Type: application/json" \
  -H "X-CSRF-Token: $CSRF" \
  -d '{"input":{"messages":[{"type":"human","content":"Hello"}]}}'

4. Internal channel 调用（IM worker）

curl -X POST http://localhost:8001/api/threads \
  -H "X-DeerFlow-Internal-Token: $DEER_FLOW_INTERNAL_TOKEN" \
  -H "X-DeerFlow-Owner-User-Id: user-uuid" \
  -H "Content-Type: application/json" \
  -d '{"thread_id":"preferred-id"}'
