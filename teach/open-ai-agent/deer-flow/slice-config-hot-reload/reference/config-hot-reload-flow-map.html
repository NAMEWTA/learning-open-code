决策流程

编辑 config.yaml │ ├─ 顶层段 ∈ STARTUP_ONLY_FIELDS ? │ ├─ 是 → 重启 Gateway（make dev 重启或 docker restart gateway） │ └─ 否 → 保存即生效（下一请求 get_app_config 重载） │ └─ 仍异常？ ├─ 代码是否误用 lifespan 的 startup_config 局部变量？ └─ extensions_config.json（MCP/skills）→ 另走 mtime 失效，本切片不展开

常见误报

现象原因处理
改了 sandbox 仍走 localprovider 单例已缓存重启；确认 sandbox.use 类路径正确
改了 log_level 无新日志apply_logging_level 未重跑重启 Gateway
改了 memory 但下条消息仍旧值非边界问题：异步 updater 未完成等 debounce + LLM；见 memory 切片
HTTP 503 Configuration not availableget_config() 加载失败检查 YAML 语法与必填 sandbox
