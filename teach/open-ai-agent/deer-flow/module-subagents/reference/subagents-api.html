task_tool

@tool("task") async def task_tool(runtime: Runtime, description: str, prompt: str, subagent_type: str, tool_call_id: Annotated[str, InjectedToolCallId]) -> str | Command

backend/packages/harness/deerflow/tools/builtins/task_tool.py — lead agent 的 task tool。解析 subagent 配置、继承 runtime context、以 tool_call_id 作为 task_id 启动后台执行、每 5s 轮询 SubagentResult、写出 task_* stream chunk，终态返回带结构化 metadata 的 Command(update={"messages": [ToolMessage(...)]})。

参数

参数含义与约束
description3–5 词短描述；写入 task_started，供前端 subtask card 标题。
prompt传给 subagent 的完整任务说明（HumanMessage 内容）。
subagent_typeregistry 名称；未知类型返回 status="failed" 并列出 get_available_subagent_names()。
tool_call_idLangChain 注入；同时作为后台 task_id，保证 trace 与前端卡片一致。
runtime提供 state（sandbox/thread_data）、context（身份、run_id、channel_user_id）、config.metadata（model、tool_groups、available_skills）。

编排要点

config = get_subagent_config(subagent_type, app_config=runtime_app_config)
tools = get_available_tools(groups=parent_tool_groups, model_name=effective_model, subagent_enabled=False)
executor = SubagentExecutor(config=config, tools=tools, thread_id=..., trace_id=..., ...)
task_id = executor.execute_async(prompt, task_id=tool_call_id)
max_poll_count = (config.timeout_seconds + 60) // 5   # 每 5s 一次

终态与 stream 事件

SubagentStatusstream chunksubagent_status
COMPLETEDtask_completedcompleted
FAILEDtask_failedfailed
CANCELLEDtask_cancelledcancelled
TIMED_OUTtask_timed_outtimed_out
MAX_TURNS_REACHEDtask_failed（卡片退出 running）max_turns_reached
轮询安全网超时task_timed_outpolling_timed_out

精确终态原因以 ToolMessage.additional_kwargs.subagent_status 为准，不以 stream event type 推断。max_turns_reached 与 polling_timed_out 都可能在 UI 上表现为 failed/timed_out 样式，但 ledger 与复用逻辑读结构化字段。

边界与异常

场景行为
未知 subagent_type立即 _task_result_command(status="failed")，不启动 executor。
subagent_type=="bash" 且 host bash 禁止返回 LOCAL_BASH_SUBAGENT_DISABLED_MESSAGE。
后台任务从表中消失task_failed + cleanup_background_task + status="failed"。
轮询超过 max_poll_countrequest_cancel_background_task + 延迟清理 + polling_timed_out。
父 run CancelledError协作取消 subagent；shield 等待终态后上报 token usage；可能调度 deferred cleanup。
skills allowlist父 metadata.available_skills 与 subagent 配置取交集后写入 config override。

Registry 与配置

SubagentConfig

@dataclass class SubagentConfig: name, description, system_prompt=None, tools=None, disallowed_tools=["task"], skills=None, model="inherit", max_turns=50, timeout_seconds=900

backend/packages/harness/deerflow/subagents/config.py

字段语义边界
toolsallowlist；None 继承父 agent 过滤后的全集。在 executor 构造时与 disallowed_tools 叠加过滤。
skillsNone 加载全部 enabled；[] 不加载；列表为白名单。与父 available_skills 取交集（task_tool 层）。
model"inherit" 继承父模型；否则用指定模型名。见 resolve_subagent_model_name。
max_turns写入 run_config["recursion_limit"]。内置 general-purpose=150、bash=60；可被全局/ per-agent 覆盖。
timeout_seconds后台线程 Future.result(timeout=...) 上限。内置默认 900；全局默认 1800 覆盖 built-in。

resolve_subagent_model_name

def resolve_subagent_model_name(config: SubagentConfig, parent_model: str | None, *, app_config: AppConfig | None = None) -> str

解析顺序：config.model（非 inherit）→ parent_model → app_config.models[0].name。无配置模型时 ValueError。

SubagentExecutor

class SubagentExecutor(config, tools, app_config=None, parent_model=None, sandbox_state=None, thread_data=None, thread_id=None, trace_id=None, user_id=None, user_role=None, oauth_provider=None, oauth_id=None, run_id=None, channel_user_id=None, deerflow_trace_id=None)

backend/packages/harness/deerflow/subagents/executor.py — 后台执行引擎。构造时按 config 过滤 tools；checkpointer=False 编译一次性 LangGraph agent。

公开方法

方法签名行为
execute_async(task: str, task_id: str | None = None) -> str登记 SubagentResult(PENDING) 到全局表，提交 _scheduler_pool，在隔离事件循环运行 _aexecute；返回 task_id。
execute(task: str, result_holder=None) -> SubagentResult同步包装；有运行中 loop 时走隔离 loop，否则 asyncio.run。
_aexecuteasync (task, result_holder=None) -> SubagentResult加载 skills → 组装 state → agent.astream(..., stream_mode="values") → 捕获 step → 写终态。

make_subagent_additional_kwargs

def make_subagent_additional_kwargs(status: SubagentStatusValue, *, result: str | None = None, error: str | None = None) -> dict[str, str]

规则细节
非法 statusValueError；不接受任意字符串。
result-bearingcompleted、max_turns_reached 且 result 非空 → brief + sha256。
error非 completed 且 error 非空 → subagent_error；空白 error 省略键。
文本截断_bound_metadata_text 首尾保留式截断至 2000 字符。

format_subagent_result_message

def format_subagent_result_message(status, *, result=None, error=None) -> tuple[str, str | None]

返回 (ToolMessage.content, normalized_error)。模型可见文本与 metadata 分离：max_turns_reached 的 content 含 partial result 前缀。

read_subagent_result_metadata

def read_subagent_result_metadata(additional_kwargs: Mapping[str, object] | None) -> StructuredSubagentResult | None

读取并校验 subagent_status；sha256 须匹配 [0-9a-f]{64}。供 delegation_ledger 与前端 subtask-result.ts 消费。

SubagentLimitMiddleware

class SubagentLimitMiddleware(AgentMiddleware): __init__(max_concurrent: int = MAX_CONCURRENT_SUBAGENTS)

backend/packages/harness/deerflow/agents/middlewares/subagent_limit_middleware.py

行为细节
钩子after_model / aafter_model 检查最后一条 AIMessage。
截断规则统计 tool_calls 中 name=="task" 的索引；保留前 N 个，丢弃其余。
限制范围max_concurrent 钳制到 [2, 4]（MIN_SUBAGENT_LIMIT / MAX_SUBAGENT_LIMIT）。
状态更新clone_ai_message_with_tool_calls 替换 AIMessage（同 id 触发 reducer 替换）。

仅在 lead agent subagent_enabled=True 时装配（build_middlewares）。
