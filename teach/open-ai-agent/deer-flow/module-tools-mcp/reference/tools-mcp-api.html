get_available_tools

def get_available_tools(groups: list[str] | None = None, include_mcp: bool = True, model_name: str | None = None, subagent_enabled: bool = False, *, app_config: AppConfig | None = None) -> list[BaseTool]

backend/packages/harness/deerflow/tools/tools.py — 把配置工具、内置工具、MCP、ACP 合并为去重后的 BaseTool 列表。不负责 skill policy 或 deferred assembly；那一步在 lead agent 装配点完成。

参数

参数含义与约束
groups为 None 时加载全部配置工具；否则只保留 tool.group in groups 的项。
include_mcp默认 True。为 False 时跳过 MCP 分支（subagent 内常用）。
model_name决定是否为 vision 模型追加 view_image。为 None 且 config.models 非空时取第一个模型名。
subagent_enabled为 True 时把 task 加入内置工具；默认 False 防止嵌套委托。
app_config显式传入时覆盖 get_app_config()；ACP agents 解析也走该实例。

装配顺序与去重

loaded_tools          ← config.tools[*].use 经 resolve_variable 解析
builtin_tools         ← present_files, ask_clarification, [skill_manage], [task], [view_image]
mcp_tools             ← get_cached_mcp_tools() + tag_mcp_tool()
acp_tools             ← build_invoke_acp_agent_tool()
all_tools = loaded + builtin + mcp + acp
unique by tool.name   ← first occurrence wins

边界与异常

场景行为
host bash 被安全策略禁止过滤 group=="bash" 或 use=="deerflow.sandbox.tools:bash_tool" 的配置工具。
配置 name 与 tool 对象 .name 不一致记录 warning，实际绑定以 tool 对象名为准（#1803）。
async-only 工具_ensure_sync_invocable_tool 附加 sync wrapper，供同步 client 调用。
MCP 模块缺失ImportError 捕获后 warning，MCP 列表为空，不阻断其余工具。
MCP 缓存加载失败记录 error，MCP 列表为空。
ACP 构造失败记录 warning，不追加 ACP 工具。
重复工具名后者跳过并 warning；优先级：配置 > 内置 > MCP > ACP。

MCP 缓存与加载

initialize_mcp_tools

async def initialize_mcp_tools() -> list[BaseTool]

backend/packages/harness/deerflow/mcp/cache.py — 应用启动时调用一次；持 asyncio.Lock，已初始化则直接返回缓存。记录 extensions 配置 mtime。

get_cached_mcp_tools

def get_cached_mcp_tools() -> list[BaseTool]

行为细节
mtime stale 检测extensions_config.json mtime 变大时调用 reset_mcp_tools_cache() 并 lazy 重载。
lazy init未初始化时在当前/新线程 event loop 中运行 initialize_mcp_tools()；失败返回 []。
配置来源始终 ExtensionsConfig.from_file()，不读内存缓存，以便 Gateway API 跨进程写入后可见。

reset_mcp_tools_cache

def reset_mcp_tools_cache() -> None

清空进程内 MCP 工具缓存、重置初始化标志，并调用 session_pool.close_all_sync() + reset_session_pool()。Gateway PUT /api/mcp/config 与显式 reset 端点会触发。

get_mcp_tools

async def get_mcp_tools() -> list[BaseTool]

backend/packages/harness/deerflow/mcp/tools.py

阶段行为
发现MultiServerMCPClient 按 server 并行 get_tools；单 server 失败不阻断其余 server。
OAuthHTTP/SSE 在发现阶段注入初始 Authorization；运行时经 interceptor 刷新。
自定义 interceptor读取 mcpInterceptors 列表，经 resolve_variable 加载 builder；异常仅 warning。
stdio 包装仅 transport=="stdio" 且工具名带 {server}_ 前缀时走 session pool；HTTP/SSE 不 pool（#3203）。
sync 兼容async-only 工具附加 make_sync_tool_wrapper。
失败langchain-mcp-adapters 未安装或顶层异常 → 返回 [] 并记录日志。
from deerflow.mcp import get_cached_mcp_tools, reset_mcp_tools_cache

mcp_tools = get_cached_mcp_tools()
# 配置热更新后
reset_mcp_tools_cache()
mcp_tools = get_cached_mcp_tools()

tool_search 延迟暴露 API

以下符号位于 backend/packages/harness/deerflow/tools/builtins/tool_search.py。必须在 skill/agent policy 过滤之后 调用。

DeferredToolCatalog

@dataclass(frozen=True) class DeferredToolCatalog: tools: tuple[BaseTool, ...]

成员说明
.names冻结工具名集合（cached_property）。
.hash基于排序后 name+OpenAI function schema 的 SHA256 前 16 位；用于 ThreadState.promoted 作用域。
.search(query)返回最多 MAX_RESULTS=5 个 BaseTool。

查询语法：select:a,b 精确选取；+slack send 名称必须含 slack 再按余下词排序；否则大小写不敏感正则/字面量搜索（无效正则降级为字面量）。

DeferredToolSetup

@dataclass(frozen=True) class DeferredToolSetup: tool_search_tool: BaseTool | None; deferred_names: frozenset[str]; catalog_hash: str | None

不变量：tool_search_tool is None ⟺ deferred_names 为空 ⟺ catalog_hash is None。

build_deferred_tool_setup

def build_deferred_tool_setup(filtered_tools: list[BaseTool], *, enabled: bool) -> DeferredToolSetup

条件返回
enabled=False空 setup；所有工具直接绑定 schema。
enabled=True 且无 MCP survivor空 setup；不追加 tool_search。
enabled=True 且有 MCP survivor含 tool_search 工具、deferred 名称集与 catalog hash。

assemble_deferred_tools

def assemble_deferred_tools(filtered_tools: list[BaseTool], *, enabled: bool) -> tuple[list[BaseTool], DeferredToolSetup]

在 build_deferred_tool_setup 之上追加 tool_search 到 final 列表。fail-closed：若 enabled=True 且 filtered 列表中仍有 MCP 工具，但 deferred set 为空 → 抛出 RuntimeError，拒绝静默绑定完整 MCP schema。

build_tool_search_tool

def build_tool_search_tool(catalog: DeferredToolCatalog) -> BaseTool

返回的 tool_search 工具执行时写 Command(update={"promoted": {"catalog_hash", "names"}, "messages": [...]})。匹配结果以 OpenAI function JSON 返回；无匹配时 content 为提示文本且 names=[]。

get_deferred_tools_prompt_section

def get_deferred_tools_prompt_section(*, deferred_names: frozenset[str] = frozenset()) -> str

生成 <available-deferred-tools> 块，仅含排序后的工具名；空集返回空字符串。

from deerflow.tools.builtins.tool_search import assemble_deferred_tools, get_deferred_tools_prompt_section

final_tools, setup = assemble_deferred_tools(filtered_tools, enabled=app_config.tool_search.enabled)
prompt_extra = get_deferred_tools_prompt_section(deferred_names=setup.deferred_names)
