3. 注入格式化（format_memory_for_injection）

backend/packages/harness/deerflow/agents/memory/prompt.py

format_memory_for_injection(
    memory_data: dict[str, Any],
    max_tokens: int = 2000,
    *,
    use_tiktoken: bool = True,
    guaranteed_categories: list[str] | None = None,
    guaranteed_token_budget: int = 500,
) -> str
行为说明
输出结构User Context → History → Facts 三段，\n\n 分隔
fact 排序按 confidence 降序贪心填充 token 预算
guaranteed_categories默认配置 ["correction"]；guaranteed fact 单独预算，溢出时保护 Facts 后缀不被前缀截断
use_tiktoken=FalseCJK 感知字符估算，无网络依赖（memory.token_counting: char）
空输入memory_data 空或无内容 → ""
异常回退guaranteed 路径异常 → confidence-only 单遍排序；主路径异常不阻断注入

辅助函数：warm_tiktoken_cache() 可在 Gateway 启动时预热 BPE，避免首请求阻塞（见 DynamicContextMiddleware 5s 超时降级）。

4. 更新队列（MemoryUpdateQueue）

backend/packages/harness/deerflow/agents/memory/queue.py

方法签名行为边界
add add(thread_id, messages, agent_name=None, user_id=None, deerflow_trace_id=None, correction_detected=False, reinforcement_detected=False) 入队并重置 debounce 计时器 memory.enabled=False → no-op；同 (thread_id, user_id, agent_name) 去重替换
add_nowait 同上 入队并 delay=0 立即调度 summarization flush hook 使用
flush flush() 取消 timer，同步处理队列 测试 / 优雅关闭
flush_nowait flush_nowait() 后台 daemon timer 立即处理 进程退出可能丢队列
clear clear() 清空队列不处理 测试用
pending_count / is_processing 属性 队列长度 / 处理中标志 线程安全

线程边界：user_id 与 deerflow_trace_id 必须在入队时捕获——threading.Timer 线程不传播 ContextVar。批处理间隔默认 0.5s（多 context 时防 rate limit）。

8. Middleware 公共 API

8.1 MemoryMiddleware

backend/packages/harness/deerflow/agents/middlewares/memory_middleware.py

MemoryMiddleware(agent_name: str | None = None, *, memory_config: MemoryConfig | None = None)
Hook方法行为跳过条件

8.2 DynamicContextMiddleware

backend/packages/harness/deerflow/agents/middlewares/dynamic_context_middleware.py

DynamicContextMiddleware(agent_name: str | None = None, *, app_config: AppConfig | None = None)
Hook方法行为
before_agent / abefore_agent _inject(state) → {"messages": [...]} | None 首 turn：ID-swap 注入 date SystemMessage + 可选 memory HumanMessage + user HumanMessage；跨午夜：仅 date 更新
辅助符号用途
is_dynamic_context_reminder(msg)判断 hidden reminder（SystemMessage 或 legacy HumanMessage）
_INJECT_TIMEOUT_SECONDS = 5.0abefore_agent 超时 → 跳过注入（降级无 memory/date）
memory.injection_enabled关闭时不读 memory.json，仅注入日期

角色分离：日期走 SystemMessage（框架权威）；memory 走 HumanMessage（用户数据，防 OWASP LLM01 提权）。

Hook方法写入 state投影（不写回 state）

8.4 DeerFlowSummarizationMiddleware

backend/packages/harness/deerflow/agents/middlewares/summarization_middleware.py

    *args,
    before_summarization: list[BeforeSummarizationHook] | None = None,

类型 / 符号说明
