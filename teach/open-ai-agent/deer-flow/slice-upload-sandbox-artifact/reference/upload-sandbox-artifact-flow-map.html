端到端分层

层职责关键文件
L0 入口 UI输入框选文件、消息内附件展示frontend/src/components/workspace/input-box.tsx
L1 提交编排先上传再 submit；乐观 UI 与 files 元数据frontend/src/core/threads/hooks.ts
L2 上传 API 客户端FormData、limits、list/deletefrontend/src/core/uploads/api.ts、hooks.ts
L3 Gateway 上传路由鉴权、分块写、chmod、sandbox 同步backend/app/gateway/routers/uploads.py
L4 上传业务逻辑路径安全、staging、artifact_url 生成backend/packages/harness/deerflow/uploads/manager.py
L5 Sandbox虚拟路径映射或 update_filebackend/packages/harness/deerflow/sandbox/
L6 Artifact 服务虚拟路径解析、MIME、安全下载backend/app/gateway/routers/artifacts.py
L7 预览 UI拉取 artifact 内容、侧栏组件frontend/src/core/artifacts/、components/workspace/artifacts/
用户选文件 + Enter → hooks.sendMessage → uploadFiles (POST .../uploads) → uploads.upload_files → manager.ensure/staged write → chmod + [可选] sandbox.update_file → 返回 virtual_path + artifact_url → thread.submit（messages 带 files 元数据） → agent 读 /mnt/user-data/uploads/… → UI 预览：GET .../artifacts/mnt/user-data/uploads/… → artifacts.get_artifact → FileResponse / PlainText

存储与路径对照

概念示例说明
Host 物理路径.deer-flow/users/{uid}/threads/{tid}/user-data/uploads/foo.pdfGateway 权威写入位置
virtual_path/mnt/user-data/uploads/foo.pdfAgent / sandbox 工具使用的路径
artifact_url/api/threads/{tid}/artifacts/mnt/user-data/uploads/foo.pdf浏览器 HTTP 访问（无 leading slash 于 mnt）
Staging 文件.upload-*.part上传中途崩溃残留；启动时清扫，列表/API 均隐藏

主路径 HTTP

步骤方法说明
上传POST /api/threads/{thread_id}/uploadsmultipart files；返回 UploadResponse
读限制GET .../uploads/limits前端 useUploadLimits 预检
列表GET .../uploads/list含 virtual_path / artifact_url
删除DELETE .../uploads/{filename}可连带删除转换出的 .md
预览/下载GET /api/threads/{thread_id}/artifacts/{path:path}path 以 mnt/user-data/uploads/... 开头

Sandbox 同步分支

条件行为
uses_thread_data_mounts=true（典型 Local / 部分 Docker bind-mount）Host 写入即 sandbox 可见；不调用 update_file
uses_thread_data_mounts=false（远程/AIO 无挂载）acquire_async + 对每个文件 update_file(virtual_path, bytes)
两种模式共通上传后对 host 文件 chmod 增加 group/other 读位

常见错误与边界

场景HTTP处理要点
单文件或总量超限413_write_upload_file_with_limits；已写路径 rollback
文件数超过 max_files413请求入口即拒绝
不安全文件名 / 非 regular file 目标跳过或 400进入 skipped_files 或 UnsafeUploadPathError
删除路径穿越400PathTraversalError
artifact 不存在404resolve_thread_virtual_path 后文件缺失
HTML / SVG artifact 预览200 attachmentACTIVE_CONTENT_MIME_TYPES 强制下载防 XSS
sandbox acquire 失败500非 mount 模式且 get_sandbox 返回空
