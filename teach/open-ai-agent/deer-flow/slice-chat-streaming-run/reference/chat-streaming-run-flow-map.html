步骤方法 / 事件说明

时序图（含断开取消）

sequenceDiagram
    participant C as Browser
    participant G as Gateway stream_run
    participant B as StreamBridge
    participant W as run_agent
    participant L as lead_agent graph

    C->>G: POST /runs/stream
    G->>G: start_run → Task(W)
    G->>B: subscribe(run_id)
    W->>B: publish(metadata)
    W->>L: astream(values, messages, custom)
    L-->>W: chunks
    W->>B: publish(events)
    B-->>G: entries
    G-->>C: SSE frames

    alt 客户端断开且 on_disconnect=cancel
        C-xG: TCP close
        G->>G: sse_consumer finally
        G->>W: RunManager.cancel
        W->>B: publish_end
    else 同线程已有活跃 run (reject)
        C->>G: POST /runs/stream
        G-->>C: 409 Conflict
    end
