文件职责

fail-closed 触发条件

场景行为
enabled=false 不 defer；MCP 全量 bind
enabled 但 policy 后无 MCP 空 setup；不追加 tool_search；不 fail-closed
enabled 且有 MCP，setup 却为空（ wiring 回归） RuntimeError fail-closed
模型直接调用未 promote 的 deferred 工具 error ToolMessage，不进入 ToolNode 成功路径
policy 剔除的 MCP 不进 catalog；tool_search 不可搜到（Finding 1）

catalog hash 与 merge_promoted

# DeferredToolCatalog.hash — 排序 name + OpenAI schema JSON
canon = [{"name": t.name, "schema": convert_to_openai_function(t)} ...]
hash = sha256(json.dumps(canon, sort_keys=True))[:16]

# merge_promoted（thread_state.py）
# new 为空 → 保留 existing
# catalog_hash 变化 → 整包替换 names（丢弃 stale）
# 相同 hash → names 并集去重，保序

Middleware _promoted(state) 仅在 state["promoted"]["catalog_hash"] == self._catalog_hash 时返回 names； _hidden = deferred_names - promoted 决定 bind 过滤集合。
