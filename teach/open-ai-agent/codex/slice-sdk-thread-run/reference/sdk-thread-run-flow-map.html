SDK Thread/Run 调用链速查表

用途：快速定位 TypeScript SDK 和 Python SDK 中 thread/run 链路的每一层入口、关键函数、输入输出和异常处理。基于当前本地源码快照。

层级概览 — TypeScript SDK

层入口文件关键函数/方法输入输出
1. Codex 构造 sdk/typescript/src/codex.ts new Codex(options) CodexOptions Codex 实例（含 CodexExec）
2. 创建 Thread sdk/typescript/src/codex.ts、thread.ts codex.startThread(options) ThreadOptions Thread 实例（id=null）
3. 发起 run sdk/typescript/src/thread.ts thread.run(input) → runStreamedInternal() Input + TurnOptions AsyncGenerator<ThreadEvent>
4. 子进程通信 sdk/typescript/src/exec.ts CodexExec.run(args) CodexExecArgs AsyncGenerator<string> (JSONL 行)
5. 事件解析 sdk/typescript/src/thread.ts JSON.parse(line) as ThreadEvent JSONL 字符串 ThreadEvent 联合类型
6. 结果聚合 sdk/typescript/src/thread.ts thread.run() 内 for await 循环 事件流 Turn {items, finalResponse, usage}

层级概览 — Python SDK

层入口文件关键函数/方法输入输出
1. Codex 构造 sdk/python/src/openai_codex/api.py Codex.__init__() → CodexClient.start() CodexConfig 已初始化的长连接客户端
2. 创建 Thread sdk/python/src/openai_codex/api.py codex.thread_start() → client.thread_start() sandbox, model, cwd 等参数 Thread 实例（含 id）
3. 发起 run sdk/python/src/openai_codex/api.py thread.run(input) → thread.turn(input) RunInput TurnHandle
4. turn/start RPC sdk/python/src/openai_codex/client.py client.turn_start(thread_id, input) thread_id + wire_input TurnStartResponse
5. 事件路由 sdk/python/src/openai_codex/client.py _reader_loop() → _router.route_notification() stdout JSON 行 按 turn_id 路由到专用队列
6. 结果收集 sdk/python/src/openai_codex/_run.py _collect_turn_result(stream, turn_id) 通知流 TurnResult

异常路径一览

异常触发层错误信息示例排查方向
二进制缺失 findCodexPath() (exec.ts) Unable to locate Codex CLI binaries 检查 @openai/codex 及其 optional dependencies 是否完整安装
平台不支持 findCodexPath() (exec.ts) Unsupported platform: freebsd (x64) 确认 platform/arch 是否在 PLATFORM_PACKAGE_BY_TARGET 映射中
spawn 失败 spawn() error 事件 (exec.ts) ENOENT / EACCES 等系统错误 检查 binaryPath 是否存在且可执行
JSON 解析失败 runStreamedInternal() (thread.ts) Failed to parse item: ... 检查 Rust 侧输出的 JSONL 是否合法
子进程异常退出 CodexExec.run() (exec.ts) Codex Exec exited with code 1: ... 检查 stderr 输出；验证参数合法性
turn 执行失败 Thread.run() (thread.ts) Error(event.error.message) 检查 turn.failed 事件中携带的错误信息
进程 stdin 不可用 CodexExec.run() (exec.ts) Child process has no stdin 检查 spawn 配置是否有误
进程 stdout 不可用 CodexExec.run() (exec.ts) Child process has no stdout 检查 spawn 配置是否有误
