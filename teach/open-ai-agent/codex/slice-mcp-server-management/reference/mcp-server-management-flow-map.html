决策点速查

决策点位置决策依据
选择 transport 类型 mcp_cmd.rs — AddMcpTransportArgs 的 clap group 用户提供 --url 或 -- command（互斥）
Transport 字段互斥校验 mcp_types.rs — RawMcpServerConfig::try_into() command 存在 → 禁止所有 HTTP 字段；url 存在 → 禁止 stdio 字段
OAuth 登录触发 mcp_cmd.rs — run_add() 末尾 oauth_login_support(&transport) 探测服务器是否暴露 OAuth metadata
Scope 发现与降级 mcp_cmd.rs — perform_oauth_login_retry_without_scopes() 发现 scopes 失败 → 重试一次不带 scopes 的登录（兼容旧服务器）
连接启动超时 codex-mcp rmcp_client.rs — ManagedClientStartup::start() startup_timeout_sec 配置值或默认 30s
Codex Apps 重连 codex-mcp rmcp_client.rs — CodexAppsStartupReconnect 仅 Codex Apps MCP server；指数退避 1s→30s
