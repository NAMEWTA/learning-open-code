Skill 激活链路速查表

用途：快速定位 Skill 从文件系统被发现到注入模型上下文的每一层入口、关键函数和异常处理。基于当前本地源码快照。

层级概览

层 入口文件 关键函数 输入 输出
1. Root 收集 codex-rs/core-skills/src/loader.rs skill_roots() ConfigLayerStack、cwd、plugin_skill_roots Vec<SkillRoot>（5 种来源、去重）
2. 目录扫描 codex-rs/core-skills/src/loader.rs discover_skills_under_root() SkillRoot.path SkillFileDiscovery { skill_files, warnings }
3. 文件解析 codex-rs/core-skills/src/loader.rs parse_skill_file() SKILL.md 路径 SkillMetadata（或 SkillError）
4. 服务快照 codex-rs/core-skills/src/service.rs SkillsService::snapshot_for_cwd() SkillsLoadInput HostSkillsSnapshot（含缓存）
5. 显式提及 codex-rs/core-skills/src/injection.rs collect_explicit_skill_mentions() UserInput[]、SkillMetadata[] Vec<SkillMetadata>（已排序去重）
6. 全文注入 codex-rs/core-skills/src/injection.rs build_skill_injections() mentioned_skills[] SkillInjections { items, warnings }
7. 列表渲染 codex-rs/core-skills/src/render.rs build_available_skills() SkillLoadOutcome、预算 AvailableSkills（可能含截断）
8. 上下文注入 codex-rs/core/src/context/available_skills_instructions.rs AvailableSkillsInstructions::from_available_skills() AvailableSkills developer 角色上下文片段
9. 文件监听 codex-rs/app-server/src/skills_watcher.rs SkillsWatcher::spawn_event_loop() 文件变化事件 缓存清空 + SkillsChanged 通知
