4. 审批系统安全分层

层级机制YOLO 模式下？触发条件
L1 硬线阻断 无条件拒绝（rm -rf /、shutdown、裸块设备覆写） 仍然生效 命令匹配 HARDLINE 正则，且后端为 local/ssh/容器宿主机
L2 危险模式 正则匹配 50+ 危险模式（sudo、系统配置写入、凭据文件访问） 跳过 命令匹配 DANGEROUS_PATTERNS 正则
L3 智能审批 LLM 辅助判断低风险命令自动放行，高风险弹出确认 跳过 CLI 交互模式 / gateway 异步队列
L4 永久允许列表 用户手动添加到 config.yaml 的命令模式 N/A 命令匹配允许列表模式

5. 委托与异步委托架构

5.1 子代理隔离模型

隔离维度父代理子代理
对话历史完整上下文全新对话，仅含委托目标摘要
task_id父 task_id独立 task_id（独立终端、文件缓存）
工具集完整工具集减去 DELEGATE_BLOCKED_TOOLS 的受限集
审批回调交互式（CLI）自动拒绝（安全默认）/ 自动通过（opt-in）
结果可见性仅见调用和摘要中间推理不可见
