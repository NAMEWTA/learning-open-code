2. 内部四层架构

agent/
│
├── 【初始化层】agent_init.py (103 KB)
│   └── init_agent() — AIAgent 构造体（60+ 参数，~1400 行初始化）
│
├── 【提示词层】prompt_builder.py (94 KB)
│   └── _build_system_prompt() — 系统提示词拼装
│
├── 【对话循环层 — 主控引擎】
│   ├── conversation_loop.py (295 KB)
│   │   └── run_conversation() — 一次完整对话回合
│   ├── chat_completion_helpers.py (147 KB)
│   │   └── LLM API 调用 / 流式处理 / 降级激活
│   ├── auxiliary_client.py (315 KB)
│   │   └── call_llm() — 子任务 LLM 路由（压缩/搜索/视觉）
│   └── agent_runtime_helpers.py (143 KB)
│       └── 消息修复 / 轨迹转换 / 凭证轮换
│
├── 【工具执行层】
│   └── tool_executor.py (79 KB)
│       ├── _execute_tool_calls_sequential()  — 串行执行
│       └── _execute_tool_calls_concurrent() — 并发执行（≤8 线程）
│
├── 【记忆管理层】
│   └── memory_manager.py (43 KB)
│       └── MemoryManager — 编排外部 MemoryProvider 插件
│
├── 【上下文压缩层】
│   └── context_compressor.py (149 KB)
│       └── ContextCompressor — 自动上下文压缩与摘要
│
└── 【学习策展层】
    └── curator.py (83 KB)
        └── maybe_run_curator() — 后台技能自动创建/维护
