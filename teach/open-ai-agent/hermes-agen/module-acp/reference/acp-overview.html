1. 模块标识

目录acp_adapter/ + acp_registry/
文件数11 个 .py + 2 个注册文件
总大小~252 KB
最大文件tools.py (~45 KB) — 工具映射与内容格式化
核心类HermesACPAgent (server.py)、SessionManager (session.py)

2. 文件清单与职责

文件大小核心职责
entry.py8.9 KBCLI 入口：加载环境变量、配置日志（stderr）、发现 MCP 工具、启动 ACP 服务器
server.py90.7 KBHermesACPAgent：实现 acp.Agent 接口的 8 个核心方法 + 9 个斜杠命令 + 模型切换 + 模式管理
session.py24.2 KBSessionManager：会话生命周期管理、SessionDB 持久化、WSL 路径转换、AIAgent 工厂
events.py9.0 KB回调工厂：tool_progress / thinking / step / message 事件 → ACP 通知桥接
tools.py45.4 KB工具映射：TOOL_KIND_MAP、build_tool_start、build_tool_complete、30+ 格式化器
auth.py2.4 KB认证：提供商检测、build_auth_methods 广告可用认证方式
permissions.py5.4 KB权限桥接：危险命令审批 → ACP PermissionOption，超时默认拒绝
edit_approval.py10.2 KB编辑审批：ContextVar 隔离、build_edit_proposal、三种自动审批策略
provenance.py4.5 KB来源追踪：压缩驱动的会话轮转时构建 sessionProvenance 元数据
__init__.py62 B包标识
__main__.py90 B支持 python -m acp_adapter

3. 注册中心（acp_registry/）

agent.jsonAgent 注册元数据（id: "hermes-agent", name, version, repository, distribution 方式）
icon.svgHermes Agent 图标，ACP 客户端在 Agent 选择界面展示

agent.json 中的 distribution.uvx 定义了 ACP 客户端如何自动安装和启动 Hermes Agent：

{
  "distribution": {
    "uvx": {
      "package": "hermes-agent[acp]==0.18.0",
      "args": ["hermes-acp"]
    }
  }
}

4. HermesACPAgent 核心接口

4.1 连接生命周期

方法触发时机返回
on_connect(conn)客户端首次连接存储连接引用
initialize(...)协议握手InitializeResponse（能力集 + 认证方法）
authenticate(method_id)客户端请求认证AuthenticateResponse 或 None

4.2 会话管理

方法说明
new_session(cwd, mcp_servers)创建新会话，返回 session_id + 模型列表 + 模式状态
load_session(cwd, session_id)加载已有会话，从 DB 恢复历史并重放到客户端
resume_session(cwd, session_id)恢复会话，不存在则创建新会话
fork_session(cwd, session_id)深拷贝会话历史到新会话
list_sessions(cursor, cwd)分页列出会话，支持 cwd 过滤
cancel(session_id)取消正在运行的任务

4.3 核心交互

方法说明
prompt(prompt, session_id)处理用户消息：斜杠命令分发 / /steer 中断恢复 / 队列积压 / 线程池执行 AIAgent
set_session_model(model_id, session_id)切换会话模型，重建 AIAgent 实例
set_session_mode(mode_id, session_id)切换编辑审批策略模式
set_config_option(config_id, session_id, value)更新会话配置选项

5. Hermes 工具 → ACP ToolKind 完整映射

ToolKind含义Hermes 工具
read读取操作read_file, skill_view, skills_list, browser_snapshot, browser_vision, browser_get_images, vision_analyze
edit编辑操作write_file, patch, skill_manage
execute执行操作terminal, process, execute_code, browser_click, browser_type, browser_scroll, browser_press, browser_back, delegate_task, image_generate, text_to_speech
search搜索操作search_files
fetch网络获取web_search, web_extract, browser_navigate
think思考操作_thinking
other其他（默认）todo 及未映射的工具

6. 支持的斜杠命令

命令说明
/help列出所有可用命令
/model [name]查看或切换当前模型
/tools列出可用工具及描述
/context显示对话上下文信息（消息数、token 用量、压缩阈值）
/reset清除对话历史
/compact压缩对话上下文（不触发 SQLite 会话分叉）
/steer向运行中的 agent 注入指导，或排队到下一回合
/queue将提示词加入队列，当前回合结束后运行
/version显示 Hermes Agent 版本

7. 编辑审批策略

策略值模式名称行为
askDefault每次编辑都询问用户确认
workspace_sessionAccept Edits自动批准工作区和 /tmp 内的编辑，敏感路径（.git/.ssh/.env）仍询问
sessionDon't Ask自动批准所有文件编辑（敏感路径除外）
