2. 双进程通信架构

┌── JSON-RPC Request ──→ stdin Node.js (React+Ink) ──┤ Python (Gateway) └── JSON-RPC Response ── stdout ← ┌── Gateway Event ── stdout ← stderr → 内存日志环 (不直接写入终端) Signal: SIGTERM → 优雅关闭 / SIGPIPE → 忽略

2.1 通信协议: JSON-RPC 2.0 (换行分隔)

帧类型方向格式
请求 (Request)Node → Python{"jsonrpc":"2.0","id":"r1","method":"session.create","params":{...}}
响应 (Response)Python → Node{"jsonrpc":"2.0","id":"r1","result":{...}}
事件 (Event)Python → Node{"jsonrpc":"2.0","method":"event","params":{"type":"message.delta","payload":{...}}}

2.2 运行模式

模式触发条件传输方式
spawn (默认)无 HERMES_TUI_GATEWAY_URLNode spawn Python 子进程，stdio JSON-RPC
attach设置 HERMES_TUI_GATEWAY_URLWebSocket 连接到已有 Gateway
sidecar设置 HERMES_TUI_SIDECAR_URLTeeTransport 镜像事件到 Dashboard

3. 核心 Gateway 事件表

事件类型触发时机前端处理
gateway.readyGateway 启动完成初始化 UI 状态、skin 主题合并、恢复会话
message.start助手开始生成标记 busy=true，新增流式消息占位
message.delta流式文本增量追加到当前消息文本缓冲
message.complete助手响应完成固化消息、追加 token 用量、标记 busy=false
thinking.delta推理过程文本追加到推理面板
reasoning.delta推理总结文本追加 reasoning 到详情面板
tool.start工具开始执行在工具面板中新增条目
tool.complete工具执行完成更新工具状态、耗时、摘要
tool.progress工具执行进度更新工具条目的预览文本
approval.request需要权限确认弹出审批弹窗 (允许/拒绝/总是)
clarify.request需要澄清弹出选项弹窗 (选择或自由输入)
sudo.request需要 sudo 密码弹出密码掩码输入
secret.request需要密钥值弹出掩码输入
session.info会话元数据更新 banner、工具面板、技能面板
notification.show系统通知在状态栏显示通知
status.update状态变更更新状态栏文本
subagent.*子 Agent 生命周期管理子 Agent 面板条目
gateway.stderrPython stderr 输出追加到活动日志
gateway.start_timeoutGateway 启动超时显示错误信息含 stderr 尾部

4. 关键 RPC 方法速查

方法分类说明
session.create会话创建新会话，返回 sid
session.resume会话恢复已有会话，加载历史消息
session.close会话关闭会话并持久化
session.active_list会话获取活跃会话列表 (含标题/状态)
session.info会话获取当前会话的完整元数据
message.send消息发送用户消息并启动 Agent 推理
message.cancel消息取消当前运行的 Agent Turn
slash.exec命令执行斜杠命令 (Gateway 端注册的命令)
command.dispatch命令通用命令 dispatch (别名/插件/技能)
complete.slash补全斜杠命令 Tab 补全
complete.path补全文件路径 Tab 补全
config.get配置读取配置值
config.set配置写入配置值 (如切换模型)
approval.respond交互回复审批请求
clarify.respond交互回复澄清请求
sudo.respond交互回复 sudo 密码
secret.respond交互回复密钥值
terminal.resize终端通知终端尺寸变化
clipboard.paste剪贴板粘贴剪贴板图片

5. 关键热键速查

热键行为
Enter提交当前输入
Shift+Enter / Alt+Enter插入换行 (多行编辑)
\ + Enter追加到多行缓冲区 (无修饰键备选)
Ctrl+C中断当前运行 / 清空草稿 / 退出
Ctrl+D退出 TUI
Ctrl+G / Alt+G在 $EDITOR 中打开当前草稿
Tab应用当前补全项
Up/Down队列编辑优先 → 输入历史导航
Ctrl+L新会话 (等价于 /clear)
Ctrl+V / Alt+V粘贴 (文本优先，然后图片/路径)
Ctrl+U / Ctrl+K删除到行首 / 行尾
Ctrl+W删除前一个单词
