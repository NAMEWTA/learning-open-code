类/函数文件行数说明

3.3 投递与中继层

类/函数文件行数说明

SessionResetPolicysession.pyEnum会话重置策略：never、daily、on_command、per_message

CapabilityDescriptorrelay/descriptor.pydataclass能力描述：max_message_length、markdown_dialect、supports_threads

4. 模块间依赖关系

gateway/
│
├── 依赖（上游）
│   ├── hermes_cli/          ← config 读取（get_hermes_home）
│   ├── agent/               ← 向核心注入 SessionContext
│   ├── agent.secret_scope   ← 密钥读取（API token 等）
│   ├── utils/               ← 工具函数（atomic_json_write、normalize_proxy_url）
│   └── cron/                ← 定时任务通过 DeliveryRouter 投递结果
│
└── 被依赖（下游消费者）
    ├── agent/conversation_loop.py  ← 读取 SessionSource 确定消息上下文
    ├── agent/system_prompt.py      ← 注入 build_session_context_prompt()
    ├── cron/scheduler.py           ← 调用 DeliveryRouter.route() 投递定时输出
    └── tools/send_message_tool.py  ← 读取 channel_directory 解析目标

5. 配置示例（config.yaml 节选）

# gateway 配置块（config.yaml）
gateway:
  platforms:                    # 启用的平台列表
    telegram:
      token: "${TELEGRAM_TOKEN}"
      home_channel: "-1001234567890"
      session_reset: "daily"
    discord:
      token: "${DISCORD_TOKEN}"
      home_channel: "9876543210"
      session_reset: "never"
    whatsapp:
      enabled: false           # 禁用但保留配置
  relay_url: "wss://connector.example.com/relay"  # 可选：中继连接器
  require_mention: true        # 全局：仅响应 @提及
