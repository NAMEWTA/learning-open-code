2. 核心接口清单

2.1 技能列表与查看

函数/命令所属文件说明
skills_list()tools/skills_tool.py返回所有可见技能的 [{name, description, category}]
skill_view(name, path?)tools/skills_tool.py查看完整技能内容或指定子文件
skill_manage(action, ...)tools/skills_tool.pycreate/patch/edit/delete/write_file/remove_file 操作
hermes skills listCLI命令行列出已安装技能
hermes skills browseCLI浏览所有可用技能（含可选和 hub）
hermes skills install <id>CLI安装可选或 hub 技能到 ~/.hermes/skills/
hermes skills audit --deepCLI对技能脚本执行 AST 级安全审计

5. 策展器生命周期状态机

        创建/安装
           │
           ▼
       ┌───────┐    bump_use()     ┌────────┐
       │ active │ ───────────────→ │ active │ (重置计数器)
       └───┬───┘                  └────────┘
           │ 长时间不活跃
           ▼
       ┌───────┐    继续不活跃    ┌─────────┐
       │ stale │ ───────────────→ │ archive │ (.archive/ 子目录)
       └───┬───┘                  └────┬────┘
           │ bump_use()               │ restore_skill()
           ▼                           ▼
       ┌────────┐                 ┌────────┐
       │ active │                 │ active │
       └────────┘                 └────────┘

  内置技能(skills/) 和 hub 安装技能 → 默认跳过策展器
  仅 agent 自创建的技能 → 受策展器管理
  PROTECTED_BUILTIN_SKILLS = {"plan"} → 永不归档
