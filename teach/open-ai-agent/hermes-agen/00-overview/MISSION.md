# 使命：Hermes Agent 项目总览

## 为什么
掌握 Hermes Agent 的全局架构与核心设计理念，建立对"自进化 AI Agent 框架"的系统认知。这是后续深入学习 agent 运行循环、gateway 消息网关、技能系统、工具注册等所有子模块的前提——没有全局坐标系，每个模块的学习都是盲人摸象。

## 成功的样子
- 能用一句话向他人准确描述 Hermes Agent 是什么
- 能画出项目的分层架构图（CLI/TUI/Web → Gateway → Agent Core → Provider → Tools）
- 能说出所有顶层目录（30+）的职责归属，无需翻看源码
- 能解释"自进化学习闭环"的含义，并指出源码中的对应入口文件

## 约束条件
- 学习方式：阅读教学课程 + 对照源码验证，非视频学习
- 先修要求：具备 Python 基础和 AI Agent 基本概念认知

## 不在范围内
- 各子模块的深入源码分析（将在 L1/L2 课程中逐一展开）
- CLI 命令的逐个使用教程（属于用户手册范畴）
- 消息网关各平台的对接细节（属于 gateway 模块课程）
