single foreground run 参考图谱

本页承载 L2 长清单：完整流程、函数索引、类型产物、环境变量与测试佐证。短课 0001-flow-map 只保留八跳记忆技能。

1. 总流程（ASCII）

Pi 父会话调用 subagent tool
  ↓
extension/index.ts
  createSubagentExecutor → executor.execute (executeWithSingleDispatchGuard)
  ↓
subagent-executor.ts :: execute()
  ├─ action? → management / status / resume / …
  ├─ effectiveAsync? → runAsyncPath → background/async-execution.ts
  ├─ chain? → runChainPath
  ├─ tasks? → runParallelPath
  └─ agent+task → runSinglePath
        ↓
      runSinglePath()
        查 AgentConfig · model/skill/output · sessionRoot
        foregroundControl 注册（interrupt 用）
        ↓
      execution.ts :: runSync()
        拼 prompt · artifact 输入 · model 候选列表
        ↓
      runSingleAttempt()
        buildPiArgs → getPiSpawnCommand (pi-spawn.ts)
        spawn(child) · stdout JSONL · onUpdate(progress)
        ↓
      child exit → SingleResult
        ↓
      AgentToolResult { details.mode: "single", results: [SingleResult] }

2. 进入条件（门卫表）

条件必须满足否则去向
无 action参数不是管理/观测动作management 分支
effectiveAsync === falseasync 未设或为 false，且 clarify !== true 时未转后台runAsyncPath
无 chain / tasks只有单 agent+taskchain / parallel 路径
spawn 限额reserveSubagentSpawns 通过错误 tool result
递归深度checkSubagentDepth 通过错误 tool result
重复调度subagentInProgress 为 false（非 action 调用）duplicate guard 错误
