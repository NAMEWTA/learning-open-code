2. 事件总线时序

事件常量方向载荷要点
subagent:slash:requestcommands → bridgerequestId、params、可选 ctx
subagent:slash:startedbridge → commandsrequestId；须在 request 处理中同步发出
subagent:slash:updatebridge → commandsprogress、currentTool、toolCount
subagent:slash:responsebridge → commandsresult、isError、errorText
subagent:slash:cancelcommands → bridgeEsc 或显式取消；abort 正在执行的 signal
