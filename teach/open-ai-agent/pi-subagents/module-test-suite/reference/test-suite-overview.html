5. e2e 全链路要点

项说明
文件open-ai-agent/pi-subagents/test/e2e/real-session-subagent.test.ts
跳过条件tryImport("@earendil-works/pi-coding-agent") 或 pi-ai 返回 null 时整组 skip。
环境隔离测试前写入 bogus 的 PI_SUBAGENT_* 环境变量，防止父进程污染子 spawn。
断言链faux parent 调 subagent → 真实 child CLI → JSONL marker → extension 解析 → tool result 含 CHILD_REAL_SESSION_OK。
与 integration 差异integration 劫持 PATH 用 mock CLI；e2e 走真实 AgentSession 与 extension 注册，验证「在 Pi runtime 里仍成立」。

6. 常用过滤命令

npm run test:unit -- recursion-guard agent-selection
npm run test:integration -- single-execution chain-execution async-execution
npm run test:e2e

Node 原生 test runner 支持在命令末尾追加文件名子串过滤；适合 L2 垂直切片时只跑与当前源码文件相关的子集。
