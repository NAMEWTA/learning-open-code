3. Supervisor request 生命周期

状态含义父端可见清理
pending等待 reply 或 progress 展示是（session 匹配）—
resolvedreply 文件已存在否删 request
expired超过 expiresAt / ask timeout否删 request
inactive子 run 已 complete/failed/paused否删 request
wrong-sessionorchestratorSessionId 不匹配否poll 跳过

modeactive 条件inactive 原因
