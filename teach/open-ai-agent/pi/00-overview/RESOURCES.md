# Pi Agent Harness 项目总览 资源

## 知识

- [Pi 官方网站 (pi.dev)](https://pi.dev) — 项目官网，包含 Demo 和文档。适用于：了解 Pi 的功能定位与使用方式。
- [Pi 官方文档](https://pi.dev/docs/latest) — 完整使用文档。适用于：查阅各命令、配置项和功能说明。
- [README.md](https://github.com/earendil-works/pi) — 项目自述文件，含包列表、开发命令、供应链安全策略。适用于：快速了解项目入口和开发流程。
- [AGENTS.md](https://github.com/earendil-works/pi/blob/main/AGENTS.md) — AI Agent 协作规范，含代码质量规则、Git 操作约定、发布流程。适用于：理解项目的工程纪律和 AI 协作边界。
- [CONTRIBUTING.md](https://github.com/earendil-works/pi/blob/main/CONTRIBUTING.md) — 贡献指南，含贡献门槛、质量标准和 PR 流程。适用于：了解项目的社区治理模型。
- [SECURITY.md](https://github.com/earendil-works/pi/blob/main/SECURITY.md) — 安全策略，定义安全边界和漏洞报告流程。适用于：理解 Pi 的安全模型和信任边界。
- [RFCs (rfc.earendil.com)](https://rfc.earendil.com/keyword/pi/) — Pi 相关的 RFC 文档，讨论较大范围的架构变更。适用于：追踪项目的长期演进方向。
