SessionStorage API 速查

本文档是 SessionStorage<TMetadata> 接口和两个实现的完整方法清单，供课程快速查阅。

SessionStorage 接口

方法签名说明
getMetadata () => Promise<TMetadata> 返回会话元数据（id, createdAt + 后端特定字段）
getLeafId () => Promise<string | null> 当前活跃的会话树叶子节点 ID
setLeafId (leafId: string | null) => Promise<void> 将叶子指针移动到指定节点（追加 LeafEntry）
createEntryId () => Promise<string> 生成唯一的条目 ID（UUIDv7 尾 8 位，冲突时重试最长 UUID）
appendEntry (entry: SessionTreeEntry) => Promise<void> 追加一个条目到存储，自动更新 leaf 指针
getEntry (id: string) => Promise<SessionTreeEntry | undefined> 按 ID 查询单个条目
findEntries (type: TType) => Promise<Array> 按类型过滤条目（泛型推导返回类型）
getLabel (id: string) => Promise<string | undefined> 查询条目的标签（从 LabelEntry 解析）
getPathToRoot (leafId: string | null) => Promise<SessionTreeEntry[]> 沿 parentId 回溯到根，返回有序路径
getEntries () => Promise<SessionTreeEntry[]> 返回所有条目（用于 Fork 全量拷贝）

SessionTreeEntry 类型速查（11 种）

类型type 值关键字段用途
MessageEntry"message"message: AgentMessage对话消息（user/assistant/toolResult/bashExec 等）
CompactionEntry"compaction"summary, firstKeptEntryId, tokensBefore上下文压缩摘要
BranchSummaryEntry"branch_summary"summary, fromId分支切换时的放弃分支摘要
LeafEntry"leaf"targetId会话树当前位置指针
LabelEntry"label"targetId, label给节点打标签
ThinkingLevelChangeEntry"thinking_level_change"thinkingLevel推理级别变更记录
ModelChangeEntry"model_change"provider, modelId模型切换记录
ActiveToolsChangeEntry"active_tools_change"activeToolNames活跃工具集变更记录
CustomEntry"custom"customType, data通用自定义条目
CustomMessageEntry"custom_message"customType, content, display自定义消息条目
SessionInfoEntry"session_info"name会话名称（legacy name）

错误码速查

错误类错误码触发场景
SessionErrornot_found查找的条目/会话不存在
SessionErrorinvalid_sessionJSONL 格式损坏、版本不兼容
SessionErrorinvalid_entryJSONL 中某行不是合法条目
SessionErrorstorage文件系统操作失败
CompactionErroraborted摘要请求被取消
CompactionErrorsummarization_failedLLM 摘要生成失败
BranchSummaryErroraborted分支摘要请求被取消
BranchSummaryErrorsummarization_failed分支摘要生成失败
