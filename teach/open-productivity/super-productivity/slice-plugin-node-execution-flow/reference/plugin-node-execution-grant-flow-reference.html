2. Grant 申请决策树

条件main 行为用户看到
非 Electron / 无 nodeExecution 权限 renderer 直接拒绝，不发起 IPC web 提示仅桌面可用；或 API 返回 permission denied
renderer 已有 token `_ensureNodeExecutionGrant()` 短路返回 true 通常无新对话框
本会话已 Deny 不再弹窗，直到用户重新启用 插件 tile 显示 permission denied
同 webContents 已有 session grant 复用 token 无对话框
uploaded + persisted consent 命中 跳过对话框，直接 mintGrant 无对话框（仍新建 session token）
built-in manifest 磁盘校验通过 verified built-in dialog（每会话） 显示磁盘读取的 name/version
uploaded 或无 verified manifest unverified uploaded dialog；Allow 后写 consent 强调第三方不可验证身份
对话框 Deny / 导航 / destroyed sender 返回 null，清理 grant cleanup onReady 失败；Deny 记入 `_nodeExecutionDeniedThisSession`

3. IPC 与安全不变量

IPC 通道

事件方向作用
PLUGIN_REQUEST_NODE_EXECUTION_GRANT renderer → main 申请 session grant；可能弹 native dialog
PLUGIN_EXEC_NODE_SCRIPT renderer → main 携带 grantToken 执行脚本
PLUGIN_REVOKE_NODE_EXECUTION_GRANT renderer → main 按 pluginId + webContents 撤销 session grant
PLUGIN_CLEAR_NODE_EXECUTION_CONSENT renderer → main 清 session grant + persisted consent

安全不变量

  • renderer 没有 `setConsent` IPC；只有 main Allow 可写持久 consent。
  • grant 绑定 issuing `webContentsId`；跨窗口 token 无效。
  • `assertSafePluginId()` 消毒 pluginId，防对话框注入与路径逃逸。
  • `consumePluginNodeExecutionApi()` one-shot；PluginBridgeService 必须早于插件代码实例化。
  • navigation / destroyed 会清 session grant；generic teardown 不清 persisted consent。
  • main 从不信任 renderer 提供的 manifest 来升级 built-in 信任级别。

4. 脚本执行边界

路径触发条件限制
direct vm 脚本不含 child_process / eval / 非白名单 require 仅允许 fs、path、os；5s vm timeout
spawn 检测到危险模式 `process.execPath` + `ELECTRON_RUN_AS_NODE=1`；默认 30s，上限 5min；脚本 max 100KB
ping `onReady` 前健康检查 脚本 `return true`；1.5s timeout；最多 3 次重试

spawn 路径仍在本机以 Node 权限运行，consent 对话框已明确警告“full access to files and system”。不要把 vm 路径误解为完整沙箱。

5. Consent 生命周期

事件session grantpersisted consent源码

6. 失败症状表

现象先判断下一步源码
web 版无法启用 node 插件 平台限制，非 grant 问题 `IS_ELECTRON` 检查；管理页提示
首次启用无对话框且立即失败 IPC/bridge 不可用或本会话已 Deny `PluginBridgeService._consumeNodeExecutionApi()`
Allow 后 onReady 仍报 bridge unavailable ping 失败，grant 已有但 exec 不通 `pingNodeBridge()` 与 `PLUGIN_EXEC_NODE_SCRIPT`
脚本报 unauthorized token 丢失、webContents 不匹配或导航清 grant main `PLUGIN_EXEC_NODE_SCRIPT` 校验分支
re-upload 后仍像旧插件 consent/grant 未清干净 `clearNodeExecutionConsent()` 返回值检查
iframe 插件 node 调用失败 宿主 grant 未就绪或 postMessage 代理失败 plugin-iframe.util.ts executeNodeScript 代理
