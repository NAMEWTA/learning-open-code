职责边界

层能做什么不能假设什么
ngsw assetGroups缓存 index.html、JS/CSS bundle、icons、en.json、字体与 lazy assets。不持久化用户任务、op-log 或 sync 状态。
ngsw api-freshness对列出的 Issue REST API 做 network-first + 短期缓存。不覆盖 SuperSync/WebDAV/自定义 sync provider。
InitialPwaUpdateCheckService检测 ngsw 新版本并提示 reload。不下载 Electron 安装包；不在 native 上运行（SwUpdate.isEnabled 为 false）。
UpdateCheckServiceElectron 上对比 GitHub release tag。Web/PWA 完全不参与。
main.ts unregister清理 Electron/Capacitor 中误注册的 SW。不提供 native 离线壳层；native 离线依赖打包进 WebView 的 dist/browser。

// initial-pwa-update-check.service.ts
afterInitialUpdateCheck$ =
  !IS_ELECTRON && this._swUpdate.isEnabled && isOnline()
    ? defer(() => from(this._swUpdate.checkForUpdate())).pipe(
        timeout(8000),
        concatMap((isUpdateAvailable) => {
          if (isUpdateAvailable && confirmDialog(...)) {
            window.location.reload();
            return EMPTY;
          }
          return of(undefined);
        }),
        shareReplay(1),
      )
    : of(undefined);

// sync.effects.ts — initial sync 等待更新检测
