阶段速查

阶段入口输出失败时表现
插件注册 CustomViewController.capacitorDidLoad WebDavHttp 插件实例挂到 Capacitor bridge JS 调用 WebDavHttp.request 报插件未实现
Provider 工厂 createWebdavProvider() 注入 APP_WEBDAV_NATIVE_HTTP、平台信息、credential store 配置缺失时 isReady() 为 false，同步被跳过
HTTP 路由 WebDavHttpAdapter.request isNativePlatform 走插件；否则 fetch cache:'no-store' Web 端 CORS 失败抛 PotentialCorsError；native 抛 WebDavNativeRequestError
Native 执行 APP_WEBDAV_NATIVE_HTTP → WebDavHttp.request 文本 body、status、headers 原样返回 网络/SSL/超时映射为 NETWORK_ERROR、SSL_ERROR、TIMEOUT_ERROR
URLSession WebDavHttpPlugin.request reloadIgnoringLocalCacheData + urlCache=nil 若误用默认 session，GET 可能返回缓存副本（#7144）
下载 WebdavApi.download GET content-hash 作为 rev + dataStr 空 body 抛 EmptyRemoteBodySPError
上传冲突检测 WebdavApi.upload 上传前 GET hash 不等抛 RemoteFileChangedUnexpectedly GET 陈旧则静默覆盖；日志 cacheHeaders 可辅助诊断
同步编排 SyncWrapperService 先 downloadRemoteOps 再 uploadPendingOps transient 网络错误可重试；rev 不匹配弹冲突 UI
iOS 恢复（邻接） ios-interface.onResume$ 补时 + flushPendingWrites，间接影响同步时机 与 WebDAV HTTP 桥接无直接调用关系

错误码速查（iOS 插件）

code含义典型场景
NETWORK_ERROR离线、DNS、无法连接飞行模式、错误 baseUrl
TIMEOUT_ERROR请求超时（30s）慢速 WebDAV 服务器
SSL_ERROR证书不受信或过期自签证书 NAS
DECODE_ERROR响应非 UTF-8二进制误当文本下载
INVALID_RESPONSE非 HTTPURLResponse协议层异常

排障检查清单

  1. 确认 CustomViewController 已注册 WebDavHttpPlugin，且 iOS 工程包含 Swift 源文件。
  2. 在共享日志中搜索 WebDavHttpAdapter.response()，查看 cacheHeaders 是否含 age 或 x-cache=HIT。
  3. 对比上传前 GET 与独立下载的 body hash；不一致则怀疑缓存或代理。
  4. 确认同步走的是 WrappedProviderService 包装后的 WebDAV，而非裸 provider。
  5. 恢复前台未同步时，沿 SyncTriggerService 查触发条件，勿只看 ios-interface。
