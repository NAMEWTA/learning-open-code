## 🔧 管理

本项目使用 `git submodule` 管理所有子项目。通过 [submodule-manager](.agents/skills/submodule-manager/SKILL.md) skill 进行自动化管理：

- **添加项目**: `python3 .agents/skills/submodule-manager/scripts/add_submodule.py --url <url> --category <dir> --branch <branch>`
- **同步所有项目**: `python3 .agents/skills/submodule-manager/scripts/sync_submodules.py`
- **更新此文档**: `python3 .agents/skills/submodule-manager/scripts/update_readme.py`

*最后更新: 由 submodule-manager 自动生成*
