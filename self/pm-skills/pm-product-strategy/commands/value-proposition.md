# /value-proposition -- 价值主张设计

使用 6 要素 JTBD 模板为产品或功能设计清晰、有说服力的价值主张。是 Strategyzer 价值主张画布的替代方案，从客户出发并聚焦实际成果。

## 工作流

### 第 1 步：理解产品和市场
### 第 2 步：构建价值主张

应用 **value-proposition** skill：
- 谁 → 为什么 → 之前做什么 → 如何做 → 之后做什么 → 替代方案
- 价值主张陈述（一句话）
- 可复用的价值主张陈述（营销、销售、引导）

### 第 3 步：保存并提供后续建议

## 注意事项

- 本模板从客户（谁/为什么）开始，向解决方案推进
- 每个价值主张是细分市场特定的
- 使用价值曲线与竞争对手进行可视化对比
