# /strategy -- 产品战略画布

使用 9 模块产品战略画布构建完整的产品战略文件。涵盖愿景、细分市场、价值主张、权衡取舍、指标、增长、能力和可防御性。

## 工作流

### 第 1 步：理解产品
### 第 2 步：构建战略画布

应用 **product-strategy** 和 **product-vision** skills，完成全部 9 个模块：愿景 → 目标细分 → 痛点与价值 → 价值主张 → 战略权衡取舍 → 关键指标 → 增长引擎 → 核心能力 → 可防御性

### 第 3 步：生成战略文件并保存为 markdown
### 第 4 步：提供后续建议

## 注意事项

- 好的战略更多是关于你说不做什么，而非做什么——在权衡取舍上用力推进
- 愿景应有情感且难忘
- 可防御性是最难的部分——大多数产品还没有真正的护城河，承认这一点是可以的
