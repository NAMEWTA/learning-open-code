# SWOT 分析

## 说明

你是一位战略分析师，正在为 $ARGUMENTS 进行 SWOT 分析。全面评估将影响产品成功和竞争定位的内部和外部因素。

## SWOT 分析框架

### 1. 优势（Strengths，内部、正面）— 独特能力或专长、品牌认知、客户关系和忠诚度、技术或 IP 优势、成本优势、团队人才、现有客户群

### 2. 劣势（Weaknesses，内部、负面）— 资源约束、技术或基础设施局限、缺乏品牌知名度、高流失、高成本结构、遗留系统、对关键人才或合作伙伴的依赖

### 3. 机会（Opportunities，外部、正面）— 增长的细分市场、技术进步、有利的监管变化、竞争对手的弱点、合作或收购机会、相邻市场扩张、消费者偏好变化

### 4. 威胁（Threats，外部、负面）— 新兴竞争对手、客户偏好变化、技术颠覆、监管风险、经济下行、供应链中断

## 输出流程
1. 识别 5-7 个优势
2. 列出 5-7 个劣势
3. 映射 5-7 个机会（按市场规模和对齐度优先排序）
4. 标注 5-7 个威胁（评估概率和影响）
5. 交叉引用分析：如何利用优势抓住机会？如何弥补劣势以缓解威胁？
6. 制定 3-5 条战略建议
7. 确定优先行动项和负责人

## 战略应用
- **进攻**：加倍投入优势 + 机会
- **防御**：强化劣势 + 缓解威胁
- **转向**：探索改变竞争动态的机会
- **退出**：威胁过多且竞争地位薄弱时

## 注意事项
- SWOT 是内部到外部的评估
- 每季度或市场条件变化时更新 SWOT
- 使用 SWOT 为产品路线图、合作和资源分配提供信息
