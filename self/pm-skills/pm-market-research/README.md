# pm-market-research

面向 PM 的市场研究 skills：用户画像、市场细分、情感分析和竞争分析。

## Skills（7 个）

- **competitor-analysis** — 分析竞争对手，识别优势、劣势和差异化机会。
- **customer-journey-map** — 创建端到端的客户旅程地图，包含阶段、触点、情绪、痛点和机会。
- **market-segments** — 识别 3-5 个潜在客户细分群体，包含人口统计特征、JTBD 和产品匹配度分析。
- **market-sizing** — 使用自上而下和自下而上的方法，估算 TAM、SAM 和 SOM 市场规模。
- **sentiment-analysis** — 分析用户反馈数据，识别细分市场及其情感得分、JTBD 和产品满意度洞察。
- **user-personas** — 从研究数据创建精细的用户画像。
- **user-segmentation** — 基于行为、JTBD 和需求从反馈数据中对用户进行细分。

## Commands（3 个）

- `/pm-market-research:analyze-feedback` — 大规模分析用户反馈——情感分析、主题提取和细分层面洞察。
- `/pm-market-research:competitive-analysis` — 分析竞争格局——识别竞争对手、比较优势和劣势、发现差异化机会。
- `/pm-market-research:research-users` — 全面的用户研究——构建用户画像、细分用户，并从研究数据中绘制客户旅程地图。
