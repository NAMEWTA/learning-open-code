# pm-data-analytics

面向 PM 的数据分析 skills：SQL 查询生成和同期群分析。分析用户数据、生成查询并识别留存模式。

## Skills（3 个）

- **ab-test-analysis** — 分析 A/B 测试结果，包含统计显著性、样本量验证、置信区间和可操作的建议。
- **cohort-analysis** — 对用户行为数据进行同期群分析。
- **sql-queries** — 从自然语言描述生成 SQL 查询。

## Commands（3 个）

- `/pm-data-analytics:analyze-cohorts` — 对用户数据进行同期群分析——留存曲线、功能采用和参与趋势。
- `/pm-data-analytics:analyze-test` — 分析 A/B 测试结果——统计显著性、样本量验证，以及发布/继续/停止建议。
- `/pm-data-analytics:write-query` — 从自然语言生成 SQL 查询——支持 BigQuery、PostgreSQL、MySQL 等。
