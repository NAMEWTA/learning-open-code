# pm-marketing-growth

产品营销与增长 skills：营销创意、价值主张陈述、北极星指标、产品命名和定位。

## Skills（5 个）

- **marketing-ideas** — 生成 5 个创意性、低成本高效益的营销方案，包含渠道、信息传递和参与理由。
- **north-star-metric** — 识别北极星指标和 3-5 个输入指标。
- **positioning-ideas** — 构思与竞争对手差异化的产品定位创意。
- **product-name** — 构思 5 个独特、易记的产品名称，附带与品牌价值观和目标受众对齐的理由。
- **value-prop-statements** — 从已有的价值主张生成适用于营销、销售和引导场景的价值主张陈述。

## Commands（2 个）

- `/pm-marketing-growth:market-product` — 头脑风暴营销创意、定位、价值主张陈述和产品名称——创意营销工具箱。
- `/pm-marketing-growth:north-star` — 定义你的北极星指标及支撑性输入指标——对业务类型进行分类，并根据最佳实践进行验证。
