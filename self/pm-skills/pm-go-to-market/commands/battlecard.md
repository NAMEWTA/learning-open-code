# /battlecard -- 竞争对比卡

创建简洁、面向销售的对比卡，帮助团队在与特定竞争对手的竞争中赢得交易。

## 调用方式

```
/battlecard 我们的 CRM vs Salesforce
/battlecard ProjectFlow vs Monday.com 面向中端市场团队
/battlecard [上传竞争对手资料或赢/输数据]
```

## 工作流

### 第 1 步：明确对决双方

### 第 2 步：研究竞争对手

应用 **competitive-battlecard** skill 配合网络搜索

### 第 3 步：生成对比卡

```
## 竞争对比卡：[你的产品] vs [竞争对手]
### 快速摘要
### 定位 | 功能对比 | 定价对比 | 异议处理 | 布设地雷 | 预期陷阱问题 | 赢/输模式
### 对话开场白 | 资源
```

保存为 markdown。

### 第 4 步：提供后续建议

## 注意事项

- 对比卡应每季度更新
- 「地雷」是对销售最有价值的部分
- 绝不在潜在客户面前贬低竞争对手
- 保持一页篇幅
