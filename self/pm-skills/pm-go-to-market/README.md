# pm-go-to-market

面向 PM 的产品上市 skills：GTM 策略、增长循环、上市模式、滩头阵地细分和理想客户画像。

## Skills（6 个）

- **beachhead-segment** — 为产品发布识别首个滩头阵地细分市场。
- **competitive-battlecard** — 创建面向销售的竞争对比卡，将你的产品与特定竞争对手进行比较。
- **growth-loops** — 识别增长循环（飞轮），实现可持续的牵引力。
- **gtm-motions** — 识别最佳的 GTM 上市模式和工具。
- **gtm-strategy** — 为产品发布创建产品上市策略，涵盖营销渠道、信息传递、成功指标和发布计划。
- **ideal-customer-profile** — 从研究数据中识别理想客户画像（ICP），包含人口统计特征、行为、JTBD 和需求。

## Commands（3 个）

- `/pm-go-to-market:battlecard` — 创建面向销售的竞争对比卡——定位、功能对比、异议处理和制胜策略。
- `/pm-go-to-market:growth-strategy` — 设计可持续的增长机制——面向产品驱动和销售驱动战略的增长循环和 GTM 模式。
- `/pm-go-to-market:plan-launch` — 创建完整的 GTM 策略——滩头阵地细分、ICP、信息传递、渠道和发布计划。
