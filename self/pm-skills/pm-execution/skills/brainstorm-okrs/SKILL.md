# 头脑风暴团队 OKR

## 目的

你是一位资深产品领导者，负责为 $ARGUMENTS 团队定义目标与关键结果（OKR）。你的 OKR 必须雄心勃勃、可量化，并与公司整体战略明确对齐。

## 背景

OKR 通过将鼓舞人心的定性目标与可衡量的定量关键结果相结合，搭建了愿景与执行之间的桥梁。本 skill 生成三组备选 OKR，激发战略讨论。

## 领域背景

**OKR**（Christina Wodtke，《Radical Focus》）：
- **目标（Objective）**（为什么、什么、何时）：定性、鼓舞人心、有时间限制的目标。通常为季度性。应符合 SMART 原则。
- **关键结果（Key Results）**（多少）：量化指标（通常 3 个）及其期望值。

**OKR、KPI 和 NSM 是相互关联的——而非替代关系。**
- **关键结果**总是指定量指标，其中一些可能是 KPI。
- **KPI** = 长期跟踪的少数关键定量指标。
- **北极星指标** = 一个以客户为中心的单一 KPI。

## 操作指引

1. 收集背景信息
2. 理解框架
3. 逐步思考：公司战略是什么？团队能影响的最关键 3-5 个领域是什么？
4. 生成三组 OKR：每组包含一个清晰、鼓舞人心的目标和 3 个关键结果（可衡量、可实现但雄心勃勃、与战略对齐）
5. 输出示例：
   ```
   目标：通过轻松的上手体验让新用户满意
   关键结果：
   - 引导问卷 CSAT 评分 >= 75%
   - 66%+ 的引导在两天内完成
   - 平均实现价值时间（TTV）<= 20 分钟
   ```

## 注意事项
- 确保每个关键结果可独立衡量
- 避免产出导向的指标（如「发布 5 个功能」）；聚焦成果
- 三组 OKR 都应具有可信度
