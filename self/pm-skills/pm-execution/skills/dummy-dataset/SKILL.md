# 虚拟数据集生成

生成用于测试的真实感虚拟数据集，支持自定义列、约束条件和输出格式（CSV、JSON、SQL、Python 脚本）。创建可执行脚本或直接可用的数据文件。

**使用场景：** 创建测试数据、生成样本数据集、构建用于开发的真实感模拟数据，或填充测试环境。

**参数：**
- `$PRODUCT`：产品或系统名称
- `$DATASET_TYPE`：数据类型（如客户反馈、交易记录、用户档案）
- `$ROWS`：生成的行数（默认：100）
- `$COLUMNS`：包含的具体列或字段
- `$FORMAT`：输出格式（CSV、JSON、SQL、Python 脚本）
- `$CONSTRAINTS`：附加约束条件或业务规则

## 分步流程

1. 识别数据集类型
2. 定义列规格——名称、数据类型和值范围
3. 确定行数
4. 选择输出格式
5. 应用真实感模式
6. 添加业务约束
7. 生成或脚本化数据
8. 验证输出

## 示例数据集规格

**数据集类型：** 客户反馈

**列：** feedback_id、customer_name、email、feedback_date、rating（1-5 星）、category（Bug、功能请求、投诉、好评）、text、product

**约束：** 评分分布偏斜（40% 5 星、30% 4 星、20% 3 星、10% 1-2 星）；Bug 类别仅限评分 1-3；功能请求仅限评分 3-5

## 输出产物
- 可直接执行的 Python 脚本或直接可用的数据文件
- CSV/JSON/SQL INSERT 语句
- 数据验证和约束合规
- 数据生成逻辑的文档
