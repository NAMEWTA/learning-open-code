# /meeting-notes -- 会议摘要

将原始会议记录或粗略笔记转化为清晰、结构化的会议纪要，捕获决策并分配行动项。

## 工作流

### 第 1 步：接收会议记录——任何格式
### 第 2 步：提取和结构化

应用 **summarize-meeting** skill：参与人、讨论主题、所做决策、行动项（负责人和截止日期）、待解决问题、关键引述

### 第 3 步：生成会议摘要并保存为 markdown
### 第 4 步：提供后续选项

## 注意事项

- 决策是最有价值的输出——确保每个决策被清晰捕获
- 没有负责人的行动项毫无用处
- 区分「讨论了」和「决定了」
