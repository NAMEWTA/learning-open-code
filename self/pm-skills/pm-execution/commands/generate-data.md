# /generate-data -- 测试数据生成器

为开发、测试、演示或原型创建真实感的虚拟数据集。按你偏好的格式输出即用文件。

## 工作流

### 第 1 步：定义数据集——实体、列、行数、关系、分布、约束
### 第 2 步：生成数据

应用 **dummy-dataset** skill：创建生成数据集的 Python 脚本、使用真实感数据、遵守约束

### 第 3 步：交付——CSV / JSON / SQL INSERT / Python 脚本
### 第 4 步：提供后续选项

## 注意事项

- 始终提供生成器脚本以便用户用不同参数重新生成
- 绝不要包含真实个人数据
