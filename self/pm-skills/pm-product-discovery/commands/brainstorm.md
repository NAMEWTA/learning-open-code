# /brainstorm -- 多角度创意构思

从三个视角（PM、设计师、工程师）生成创意产品想法或实验设计，根据你在改进已有产品还是从零开始构建进行定制。

## 调用方式

```
/brainstorm ideas existing 移动银行应用参与度
/brainstorm ideas new 面向忙碌父母的 AI 驱动的餐食规划
/brainstorm experiments existing 引导流程重新设计
/brainstorm experiments new 面向自由职业设计师的市场平台
```

## 工作流

### 第 1 步：确定模式

解析参数识别两个维度：**创意什么**（ideas/experiments）和**产品阶段**（existing/new）

### 第 2 步：收集背景

对话式询问关键问题。

### 第 3 步：生成输出

- **如头脑风暴创意**——应用 **brainstorm-ideas-existing** 或 **brainstorm-ideas-new** skill
- **如头脑风暴实验**——应用 **brainstorm-experiments-existing** 或 **brainstorm-experiments-new** skill

### 第 4 步：深入和迭代

呈现初步结果后，提供：详细展开、识别假设、设计实验、排定优先级等后续选项。

## 注意事项

- 已有产品的基础创意应基于当前用户行为和已验证的问题
- 新产品应首先聚焦需求性和可行性风险
- 鼓励先广度后深度——评估前生成大量创意
