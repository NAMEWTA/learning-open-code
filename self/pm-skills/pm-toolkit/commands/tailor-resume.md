# /tailor-resume -- 简历与 JD 优化

获取你的简历和目标职位描述，然后策略性地对齐你的经验以最大化面试机会。

## 调用方式

```
/tailor-resume [上传简历] 这是 JD：[粘贴职位描述]
/tailor-resume [上传简历和 JD 作为文件]
```

## 工作流

### 第 1 步：接收两份文档

需要简历和目标职位描述两份输入。

### 第 2 步：分析职位描述

提取：必需资格和技能、优先资格、关键职责、行业和领域信号、级别指标、文化和团队信号。

### 第 3 步：定制简历

应用 **review-resume** skill：关键词对齐、要点重写、章节重排序、摘要/目标重写、技能部分对齐。

### 第 4 步：生成定制后简历 + 分析

```
## 简历定制：[职位] at [公司]
### 对齐评分
### 关键词差距分析
### 所做修改
### 定制后的简历
### 差距分析
### 求职信要点
```

## 注意事项

- 绝不要编造经验——真实地重构而非凭空捏造
- 摘要/目标是每次申请中定制 ROI 最高的部分
- 尽可能精确匹配 JD 的语言
