# /privacy-policy -- 隐私政策生成器

为你的产品起草全面的隐私政策。涵盖数据类型、管辖区、合规（GDPR、CCPA），并标注需法律审核的条款。

## 调用方式

```
/privacy-policy 收集用户行为数据的 SaaS 分析工具——服务美国和欧盟客户
/privacy-policy 具有位置数据和第三方集成的移动应用
```

## 工作流

### 第 1 步：收集背景

询问：什么产品或服务？收集什么数据？用户在哪里？任何第三方数据共享？数据存储位置和时长？年龄限制？

### 第 2 步：起草政策

应用 **privacy-policy** skill：生成涵盖数据收集、使用目的、处理的法律依据、数据共享与第三方、数据保留与删除、用户权利、Cookie 政策、安全措施、儿童隐私、国际传输、联系信息和政策更新流程的完整政策。

### 第 3 步：交付

保存为 markdown：完整政策文本 + 合规检查清单 + 需法律审核的条款 + 实施检查清单。

## 注意事项

- 法律顾问应在发布前审核
- 隐私政策应使用简明语言而非法律术语
- 当数据实践变化时更新政策
