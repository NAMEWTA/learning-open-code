# /proofread -- 语法与流畅度检查

识别文本中的语法、逻辑和流畅度问题。提供具体、有针对性的修改建议，无需重写整篇文档。

## 调用方式

```
/proofread [粘贴文本]
/proofread [上传文档]
```

## 工作流

### 第 1 步：接收文本

接受任何形式的文本：粘贴内容、上传文档（DOCX、PDF、markdown）或邮件草稿。

### 第 2 步：分析

应用 **grammar-check** skill：扫描三类问题——**语法**、**逻辑**、**流畅度**。

### 第 3 步：报告问题

```
## 校对报告
文本长度 | 发现的问题 | 按类别列出问题
```

### 第 4 步：提供选项

- "需要我**应用所有修改**并返回清理后的文本吗？"
- "需要我**聚焦于某个特定部分**更详细地检查吗？"

## 注意事项

- 修改建议应尽可能小——仅修改必要内容，保留作者风格
- 区分错误与不同风格
