# /review-resume -- PM 简历审核

获得对照产品管理最佳实践的全面简历审核。评估结构、影响力指标、关键词优化，并提供附带示例的具体改进建议。

## 调用方式

```
/review-resume [粘贴简历文本]
/review-resume [上传简历 PDF 或 DOCX]
```

## 工作流

### 第 1 步：接收简历

### 第 2 步：对照 10 项最佳实践评估

应用 **review-resume** skill：影响力指标、XYZ+S 公式、PM 特定语言、结构与可读性、关键词优化、故事线、简洁性、相关性、技术可信度、领导力信号。

### 第 3 步：生成审核

```
## 简历审核
总体评分 | 最强领域 | 最大改进机会 | 评分卡 | Top 3 改进 | 逐节反馈 | 缺失元素 | 需添加的关键词
```

### 第 4 步：提供后续建议

- "需要我**针对特定职位描述定制**这份简历吗？"

## 注意事项

- 要具体且有建设性——「加指标」没有帮助，给出具体改写是可行的
- PM 简历应强调成果而非产出
- ATS 优化很重要
