# /draft-nda -- NDA 起草

起草一份针对你的情况定制的专业保密协议。涵盖信息类型、管辖区、期限，并明确标注需要法律审核的条款。

## 调用方式

```
/draft-nda 我们的初创公司与潜在企业客户之间的双向 NDA
/draft-nda 面向访问我们代码库的自由职业承包商的双向 NDA
```

## 工作流

### 第 1 步：收集背景

询问：双方是谁？双向还是单向 NDA？保护什么信息？管辖区域？期限？任何特殊顾虑？

### 第 2 步：起草 NDA

应用 **draft-nda** skill：生成覆盖双方和鉴于条款、保密信息定义、接收方义务、例外情形、期限和存续、材料返还/销毁、救济措施、管辖法律和司法管辖、标准模板条款的完整 NDA。

### 第 3 步：交付

```
## 保密协议
[完整 NDA 文本]
### 需法律审核的条款
### 简明摘要
```

保存为 markdown。

## 注意事项

- 这是一个起点——始终建议由合格法律顾问审核
- 用 ⚠️ 标记涉及重大法律风险的条款
- 包含简明注释，使非法律人士理解他们同意的内容
- 双向 NDA 通常更受欢迎
