# pm-toolkit

PM 实用工具 skills：简历审核、NDA 起草、隐私政策生成和语法/流畅度检查。产品经理核心产品工作之外的基本工具。

## Skills（4 个）

- **draft-nda** — 起草双方之间的详细保密协议（NDA）。
- **grammar-check** — 识别文本中的语法、逻辑和流畅度问题，并提出有针对性的修改建议，无需重写整段文本。
- **privacy-policy** — 为产品起草详细的隐私政策，涵盖数据类型、管辖区域、合规考量以及需法律审核的条款。
- **review-resume** — 全面的 PM 简历审核与优化，对照 10 项最佳实践，包括 XYZ+S 公式、关键词优化、职位针对性定制和结构规范。

## Commands（5 个）

- `/pm-toolkit:draft-nda` — 起草双方之间的保密协议，包含适合管辖区域的条款。
- `/pm-toolkit:privacy-policy` — 起草涵盖数据收集、使用、存储和合规要求的隐私政策。
- `/pm-toolkit:proofread` — 检查任意文本中的语法、逻辑和流畅度——有针对性的修改，无需重写。
- `/pm-toolkit:review-resume` — 对照 10 项最佳实践全面审核 PM 简历——结构、影响力指标、关键词和可操作的反馈。
- `/pm-toolkit:tailor-resume` — 针对特定职位描述定制 PM 简历——关键词对齐、经验重构和策略性优化。
