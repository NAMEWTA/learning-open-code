# 分类标签

Skills 使用五个规范分类角色进行交流。此文件将这些角色映射到此仓库 issue 跟踪器中实际使用的标签字符串。

| Label in mattpocock/skills | Label in our tracker | Meaning                                  |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此 issue                   |
| `needs-info`               | `needs-info`         | 等待报告者提供更多信息                   |
| `ready-for-agent`          | `ready-for-agent`    | 已完整描述，可供 AFK agent 处理          |
| `ready-for-human`          | `ready-for-human`    | 需要人工实现                             |
| `wontfix`                  | `wontfix`            | 不会执行                                 |

当 skill 提及某个角色时（例如"应用 AFK-ready 分类标签"），使用此表中对应的标签字符串。

编辑右列以匹配你实际使用的词汇。
