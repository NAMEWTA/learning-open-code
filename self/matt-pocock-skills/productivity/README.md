# 效率

通用工作流工具，不限于编码场景。

## 用户调用

只能通过手动输入调用（`disable-model-invocation: true`）。

- **[grill-me](./grill-me/SKILL.md)** — 就一个计划或设计接受无情的访谈，直到决策树的每个分支都被解决。
- **[handoff](./handoff/SKILL.md)** — 将当前对话压缩为交接文档，以便另一个代理可以继续工作。
- **[teach](./teach/SKILL.md)** — 通过多次会话向用户教授新技能或概念，使用当前目录作为有状态的教学工作区。
- **[writing-great-skills](./writing-great-skills/SKILL.md)** — 编写和编辑高质量技能的参考指南：使技能可预测的词汇和原则。

## 模型调用

模型或用户均可触达（通过丰富的触发短语让模型能主动调用）。

- **[grilling](./grilling/SKILL.md)** — 就一个计划或设计无情地访谈用户，直到决策树的每个分支都被解决。
