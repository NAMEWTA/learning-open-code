# 开发中

仍在开发中的技能。尚未准备好发布——可能会有粗糙边缘、破坏性变更和被放弃的实验。在毕业到稳定分类之前，它们不会出现在插件和顶层 README 中。

- **[decision-mapping](./decision-mapping/SKILL.md)** — 将一个模糊想法转化为有序的调研工单地图，然后逐一推动解决。用户调用。
- **[review](./review/SKILL.md)** — 从两个并行维度审查自某个固定点以来的变更：**标准**（diff 是否遵循仓库的编码标准？）和 **Spec**（diff 是否忠实地实现了原始 issue/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 以节奏旅程、自选冒险的方式塑造一篇文章。选择一个起始节拍，只写那个节拍，然后转向下一个，直到文章自然收尾。
- **[writing-fragments](./writing-fragments/SKILL.md)** — 质询会话，挖掘你的碎片——异质的写作素材——并将其追加到单个文档中，作为未来文章的原始素材。
- **[writing-shape](./writing-shape/SKILL.md)** — 将一个包含原始素材的 markdown 文件逐段塑造成一篇文章，并在每一步论证格式选择。
